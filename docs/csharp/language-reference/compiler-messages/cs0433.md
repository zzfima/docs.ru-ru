## <a name="example"></a><span data-ttu-id="f15b6-114">Пример</span><span class="sxs-lookup"><span data-stu-id="f15b6-114">Example</span></span>  
 <span data-ttu-id="f15b6-115">В следующем примере возникает ошибка CS0433.</span><span class="sxs-lookup"><span data-stu-id="f15b6-115">The following example generates CS0433.</span></span>  
  
```csharp

using TypeBindConflicts;  
public class Test
{  
   public static void Main()
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="f15b6-116">Пример</span><span class="sxs-lookup"><span data-stu-id="f15b6-116">Example</span></span>  
 <span data-ttu-id="f15b6-117">В следующем примере показано, как можно использовать функцию псевдонима параметра компилятора **/reference** для устранения возникшей ошибки CS0433.</span><span class="sxs-lookup"><span data-stu-id="f15b6-117">The following example shows how you can use the alias feature of the **/reference** compiler option to resolve this CS0433 error.</span></span>

```csharp  
   }

```
