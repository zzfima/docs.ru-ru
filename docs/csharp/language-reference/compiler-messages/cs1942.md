# <a name="compiler-error-cs1942"></a><span data-ttu-id="23c39-102">Ошибка компилятора CS1942</span><span class="sxs-lookup"><span data-stu-id="23c39-102">Compiler Error CS1942</span></span>

<span data-ttu-id="23c39-103">Тип выражения в предложении "предложение" неверен.</span><span class="sxs-lookup"><span data-stu-id="23c39-103">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="23c39-104">Ошибка определения типа при вызове в "метод".</span><span class="sxs-lookup"><span data-stu-id="23c39-104">Type inference failed in the call to 'method'.</span></span>  
  
<span data-ttu-id="23c39-105">Эта ошибка обычно возникает в том случае, если переменной диапазона назначается неверный явный тип.</span><span class="sxs-lookup"><span data-stu-id="23c39-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="23c39-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="23c39-106">To correct this error</span></span>

1. <span data-ttu-id="23c39-107">Если переменной диапазона явно назначается тип, убедитесь, что этот тип совпадает с типом элементов в коллекции, для которой будут выполняться итерации, или может быть неявно преобразован из него.</span><span class="sxs-lookup"><span data-stu-id="23c39-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="23c39-108">Если переменной диапазона предшествует ключевое слово `var`, удалите `var`.</span><span class="sxs-lookup"><span data-stu-id="23c39-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="23c39-109">Пример</span><span class="sxs-lookup"><span data-stu-id="23c39-109">Example</span></span>

<span data-ttu-id="23c39-110">В следующем коде возникает ошибка CS1942:</span><span class="sxs-lookup"><span data-stu-id="23c39-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="23c39-111">Ошибка CS1942 связана с ошибкой CS1949, поскольку использование `var` с переменными диапазона приводит к ошибке базовой операции `Cast<T>`, так как `var` не является типом.</span><span class="sxs-lookup"><span data-stu-id="23c39-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
