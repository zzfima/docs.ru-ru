# <a name="compiler-error-cs0038"></a><span data-ttu-id="309cd-102">Ошибка компилятора CS0038</span><span class="sxs-lookup"><span data-stu-id="309cd-102">Compiler Error CS0038</span></span>

<span data-ttu-id="309cd-103">Невозможно получить доступ к нестатическому члену внешнего типа "тип1" через вложенный тип "тип2"</span><span class="sxs-lookup"><span data-stu-id="309cd-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>

 <span data-ttu-id="309cd-104">Поле класса не является автоматически доступным вложенному классу.</span><span class="sxs-lookup"><span data-stu-id="309cd-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="309cd-105">Чтобы обеспечить доступ из вложенного класса к полю, этому полю необходимо присвоить модификатор [static](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="309cd-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="309cd-106">В противном случае необходимо создать экземпляр внешнего класса.</span><span class="sxs-lookup"><span data-stu-id="309cd-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="309cd-107">Дополнительные сведения см. в разделе [Вложенные типы](../../programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="309cd-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>

 <span data-ttu-id="309cd-108">В следующем примере возникает ошибка CS0038:</span><span class="sxs-lookup"><span data-stu-id="309cd-108">The following sample generates CS0038:</span></span>

```csharp
// CS0038.cs
class OuterClass
{
    public int count;
    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();
            // int count2 = class_inst.count;
            int count2 = count;   // CS0038
        }
    }

    public static void Main()
    {
    }
}
```
