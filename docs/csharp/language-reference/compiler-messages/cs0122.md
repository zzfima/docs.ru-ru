# <a name="compiler-error-cs0122"></a>Ошибка компилятора CS0122
"член" недоступен из-за его уровня защиты  
  
 [Модификатор доступа](../../../csharp/language-reference/keywords/modifiers.md) для члена класса запрещает доступ к этому члену. Дополнительные сведения см. в разделе [Модификаторы доступа](../../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 Одна из причин этого (не показана в следующем примере) может быть связана с отсутствием флага компилятора **/out** для целевого объекта дружественной сборки. Дополнительные сведения см. в разделах [Дружественные сборки](http://msdn.microsoft.com/library/df0c70ea-2c2a-4bdc-9526-df951ad2d055) и [/out (параметры компилятора C#)](../../../csharp/language-reference/compiler-options/out-compiler-option.md)  
  
## <a name="example"></a>Пример  
 В следующем примере возникает ошибка CS0122:  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
