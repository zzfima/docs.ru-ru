# <a name="compiler-error-cs1674"></a><span data-ttu-id="0f02a-102">Ошибка компилятора CS1674</span><span class="sxs-lookup"><span data-stu-id="0f02a-102">Compiler Error CS1674</span></span>
<span data-ttu-id="0f02a-103">"T": тип, использованный в операторе using, должен иметь неявное преобразование в System.IDisposable</span><span class="sxs-lookup"><span data-stu-id="0f02a-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="0f02a-104">С помощью [оператора using](../keywords/using-statement.md) можно гарантировать высвобождение объекта в конце блока `using`, поэтому используемые в данном операторе типы должны быть высвобождаемыми.</span><span class="sxs-lookup"><span data-stu-id="0f02a-104">The [using Statement](../keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="0f02a-105">Например, типы значений не являются высвобождаемыми, а параметры типа, которые не имеют ограничений классов, не могут гарантированно считаться высвобождаемыми.</span><span class="sxs-lookup"><span data-stu-id="0f02a-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>

## <a name="example"></a><span data-ttu-id="0f02a-108">Пример</span><span class="sxs-lookup"><span data-stu-id="0f02a-108">Example</span></span>  
 <span data-ttu-id="0f02a-109">В следующем примере возникает ошибка CS1674.</span><span class="sxs-lookup"><span data-stu-id="0f02a-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
