 <span data-ttu-id="1d173-104">При компиляции использовался параметр компилятора [/win32icon](../../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) или не использовался параметр [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md).  <span data-ttu-id="1d173-105">Если вам не удается решить проблему с созданием файла, попробуйте использовать параметр [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), при выборе которого файл со сведениями о ресурсах не создается.</span><span class="sxs-lookup"><span data-stu-id="1d173-105">If you are unable to resolve the file-generation problem, you could use [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), which does not generate a file that contains resource information.</span></span>
