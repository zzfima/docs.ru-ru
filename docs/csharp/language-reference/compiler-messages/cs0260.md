# <a name="compiler-error-cs0260"></a><span data-ttu-id="e0150-102">Ошибка компилятора CS0260</span><span class="sxs-lookup"><span data-stu-id="e0150-102">Compiler Error CS0260</span></span>

<span data-ttu-id="e0150-103">Отсутствует частичный модификатор в объявлении типа "type"; существует другое частичное объявление этого типа</span><span class="sxs-lookup"><span data-stu-id="e0150-103">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="e0150-104">Эта ошибка означает, что объявлено несколько классов с одинаковыми именами.</span><span class="sxs-lookup"><span data-stu-id="e0150-104">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="e0150-105">Кроме того, по крайней мере одно объявление (но не все) содержит модификатор `partial`.</span><span class="sxs-lookup"><span data-stu-id="e0150-105">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="e0150-106">Если требуется определить класс в нескольких частях, необходимо объявить каждую из них с помощью ключевого слова `partial`.</span><span class="sxs-lookup"><span data-stu-id="e0150-106">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="e0150-107">Эта ошибка также возникает, если объявить новый класс и случайно дать ему имя, совпадающее с именем разделяемого класса, уже объявленного в этом же пространстве имен.</span><span class="sxs-lookup"><span data-stu-id="e0150-107">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="e0150-108">Следующий пример приводит к возникновению ошибки CS0260:</span><span class="sxs-lookup"><span data-stu-id="e0150-108">The following sample generates CS0260:</span></span>  

```csharp
// CS0260.cs  
// You must mark both parts of the definition of class C
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C
{  
}  
  
partial class C  
{  
}  
```
