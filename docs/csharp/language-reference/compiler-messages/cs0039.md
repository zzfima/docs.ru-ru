# <a name="compiler-error-cs0039"></a>Ошибка компилятора CS0039
Не удается преобразовать тип "тип1" в "тип2" с помощью преобразования ссылок, упаковки-преобразования, распаковки-преобразования, преобразования в оболочку или преобразования типа Null  
  
 Преобразование с использованием оператора [as](../../../csharp/language-reference/keywords/as.md) допускается посредством наследования, преобразования ссылок и упаковки-преобразования. Дополнительные сведения см. в разделе [Операторы преобразования](../../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).  
  
## <a name="example"></a>Пример  
 Следующий пример приводит к возникновению ошибки CS0039.  
  
```csharp  
// CS0039.cs  
using System;  
class A  
{  
}  
class B: A  
{  
}  
class C: A  
{  
}  
class M  
{  
    static void Main()  
    {  
        A a = new C();  
        B b = new B();  
        C c;  
  
        // This is valid; there is a built-in reference  
        // conversion from A to C.  
        c = a as C;    
  
        //The following generates CS0039; there is no  
        // built-in reference conversion from B to C.  
        c = b as C;  // CS0039  
    }  
}  
```
