# Compiler Error CS0563
Один из параметров двоичного оператора должен вмещать тип  
  
 Объявление метода для [перегрузки оператора](../../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md) должно следовать определенным правилам.  
  
## Пример  
 Следующий пример приводит к возникновению ошибки CS0563:  
  
```  
// CS0563.cs  
public class iii  
{  
    public static implicit operator int(iii x)  
    {  
        return 0;  
    }  
    public static implicit operator iii(int x)  
    {  
        return null;  
    }  
    public static int operator +(int aa, int bb)   // CS0563   
    // Use the following line instead:  
    // public static int operator +(int aa, iii bb)      
    {  
        return 0;  
    }  
    public static void Main()  
    {  
    }  
}  
```
