# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="83ee5-102">Предупреждение компилятора (уровень 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="83ee5-102">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="83ee5-103">"член" устарел: "текст"</span><span class="sxs-lookup"><span data-stu-id="83ee5-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="83ee5-104">Член класса помечен атрибутом `Obsolete`, чтобы при обращении к этому члену возникало предупреждение.</span><span class="sxs-lookup"><span data-stu-id="83ee5-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="83ee5-105">Дополнительные сведения см. в разделе [Общие атрибуты](../../programming-guide/concepts/attributes/common-attributes.md).</span><span class="sxs-lookup"><span data-stu-id="83ee5-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="83ee5-106">В следующем примере возникает ошибка CS0618:</span><span class="sxs-lookup"><span data-stu-id="83ee5-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
