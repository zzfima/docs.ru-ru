# <a name="endregion-c-reference"></a>#endregion (справочник по C#)
`#endregion` отмечает конец блока [#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md). Например:  
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
