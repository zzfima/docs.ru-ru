## <a name="finally-blocks"></a><span data-ttu-id="4ed18-124">Блоки "Finally"</span><span class="sxs-lookup"><span data-stu-id="4ed18-124">Finally Blocks</span></span>  
 <span data-ttu-id="4ed18-125">Блок `finally` позволяет удалить действия, выполненные в блоке `try`.</span><span class="sxs-lookup"><span data-stu-id="4ed18-125">A `finally` block enables you to clean up actions that are performed in a `try` block.</span></span> <span data-ttu-id="4ed18-126">При наличии блока `finally` он выполняется последним, после блока `try` и всех выполняемых блоков `catch`.</span><span class="sxs-lookup"><span data-stu-id="4ed18-126">If present, the `finally` block executes last, after the `try` block and any matched `catch` block.</span></span> <span data-ttu-id="4ed18-127">Блок `finally` выполняется всегда вне зависимости от возникновения исключения или обнаружения блока `catch`, соответствующего типу исключения.</span><span class="sxs-lookup"><span data-stu-id="4ed18-127">A `finally` block always runs, regardless of whether an exception is thrown or a `catch` block matching the exception type is found.</span></span>
