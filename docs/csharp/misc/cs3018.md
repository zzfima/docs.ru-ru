# <a name="compiler-warning-level-1-cs3018"></a><span data-ttu-id="fea71-102">Предупреждение компилятора (уровень 1) CS3018</span><span class="sxs-lookup"><span data-stu-id="fea71-102">Compiler Warning (level 1) CS3018</span></span>
<span data-ttu-id="fea71-103">"тип" не может быть помечен как совместимый с CLS, поскольку является членом типа "тип", не совместимого с CLS</span><span class="sxs-lookup"><span data-stu-id="fea71-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>  
  
 <span data-ttu-id="fea71-104">Это предупреждение возникает, когда вложенный класс, атрибуту CLSCompliant которого присвоено значение `true` , объявляется как член класса, который объявлен с атрибутом CLSCompliant, равным `false`.</span><span class="sxs-lookup"><span data-stu-id="fea71-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="fea71-105">Это недопустимо, так как вложенный класс не может быть совместимым с CLS, если он является членом внешнего класса, несовместимого с CLS.</span><span class="sxs-lookup"><span data-stu-id="fea71-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="fea71-106">Для устранения этого предупреждения удалите атрибут CLSCompliant из вложенного класса либо измените его значение с `true` на `false`.</span><span class="sxs-lookup"><span data-stu-id="fea71-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="fea71-107">Дополнительные сведения о соответствии спецификации CLS см. в разделе [независимость от языка и независимые от языка компоненты](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="fea71-107">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="fea71-108">Пример</span><span class="sxs-lookup"><span data-stu-id="fea71-108">Example</span></span>  
 <span data-ttu-id="fea71-109">Следующий пример приводит к возникновению предупреждения CS3018:</span><span class="sxs-lookup"><span data-stu-id="fea71-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
