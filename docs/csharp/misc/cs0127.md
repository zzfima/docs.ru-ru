# <a name="compiler-error-cs0127"></a><span data-ttu-id="f8684-102">Ошибка компилятора CS0127</span><span class="sxs-lookup"><span data-stu-id="f8684-102">Compiler Error CS0127</span></span>
<span data-ttu-id="f8684-103">Поскольку "функция" возвращает значение void, после ключевого слова return не должно задаваться выражение объекта</span><span class="sxs-lookup"><span data-stu-id="f8684-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="f8684-104">Метод с типом возврата [void](../../csharp/language-reference/keywords/void.md) не может возвращать значение.</span><span class="sxs-lookup"><span data-stu-id="f8684-104">A method with a [void](../../csharp/language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="f8684-105">Дополнительные сведения см. в статье [Методы](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="f8684-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="f8684-106">Следующий пример приводит к возникновению ошибки CS0127:</span><span class="sxs-lookup"><span data-stu-id="f8684-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
