# <a name="compiler-error-cs0211"></a><span data-ttu-id="c5580-102">Ошибка компилятора CS0211</span><span class="sxs-lookup"><span data-stu-id="c5580-102">Compiler Error CS0211</span></span>
<span data-ttu-id="c5580-103">Невозможно получить адрес указанного выражения.</span><span class="sxs-lookup"><span data-stu-id="c5580-103">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="c5580-104">Можно получить адреса полей, локальных переменных и косвенных обращений указателей, однако нельзя получить, например, адрес суммы двух локальных переменных.</span><span class="sxs-lookup"><span data-stu-id="c5580-104">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="c5580-105">Дополнительные сведения см. в разделе [Небезопасный код и указатели](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="c5580-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="c5580-106">Следующий пример приводит к возникновению ошибки CS0211:</span><span class="sxs-lookup"><span data-stu-id="c5580-106">The following sample generates CS0211:</span></span>  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
