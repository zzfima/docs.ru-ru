# <a name="compiler-error-cs0551"></a><span data-ttu-id="daca6-102">Ошибка компилятора CS0551</span><span class="sxs-lookup"><span data-stu-id="daca6-102">Compiler Error CS0551</span></span>
<span data-ttu-id="daca6-103">В явной реализации интерфейса "реализация" отсутствует метод доступа "метод_доступа"</span><span class="sxs-lookup"><span data-stu-id="daca6-103">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="daca6-104">Класс, который явно реализует свойство интерфейса, должен реализовывать все методы доступа, которые определяет этот интерфейс.</span><span class="sxs-lookup"><span data-stu-id="daca6-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>  
  
 <span data-ttu-id="daca6-105">Для получения дополнительной информации см. [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="daca6-105">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="daca6-106">Пример</span><span class="sxs-lookup"><span data-stu-id="daca6-106">Example</span></span>  
 <span data-ttu-id="daca6-107">В следующем примере возникает ошибка CS0551:</span><span class="sxs-lookup"><span data-stu-id="daca6-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
