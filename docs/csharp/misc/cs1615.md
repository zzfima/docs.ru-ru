# <a name="compiler-error-cs1615"></a><span data-ttu-id="67d18-102">Ошибка компилятора CS1615</span><span class="sxs-lookup"><span data-stu-id="67d18-102">Compiler Error CS1615</span></span>
<span data-ttu-id="67d18-103">Аргумент "число" не должен передаваться с ключевым словом "ключевое слово".</span><span class="sxs-lookup"><span data-stu-id="67d18-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="67d18-104">Одно из ключевых слов `ref` или **out** было использовано, когда функция не принимает параметр `ref` или **out** для этого аргумента.</span><span class="sxs-lookup"><span data-stu-id="67d18-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="67d18-105">Чтобы устранить эту ошибку, удалите неверное ключевое слово и используйте подходящее ключевое слово, соответствующее объявлению функции, если оно существует.</span><span class="sxs-lookup"><span data-stu-id="67d18-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="67d18-106">Следующий пример приводит к возникновению ошибки CS1615:</span><span class="sxs-lookup"><span data-stu-id="67d18-106">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
