# <a name="compiler-error-cs0643"></a>Ошибка компилятора CS0643
arg повторяющийся именованный аргумент атрибута  
  
 Параметр `arg`в пользовательском атрибуте был указан дважды. Дополнительные сведения см. в разделе [Атрибуты](../../csharp/programming-guide/concepts/attributes/index.md).  
  
## <a name="example"></a>Пример  
 В следующем примере возникает ошибка CS0643.  
  
```csharp  
// CS0643.cs  
using System;  
using System.Runtime.InteropServices;  
  
[AttributeUsage(AttributeTargets.Class)]  
public class MyAttribute : Attribute  
{  
    public MyAttribute()  
    {  
    }  
  
    public int x;  
}  
  
[MyAttribute(x = 5, x = 6)]   // CS0643, error setting x twice  
// try the following line instead  
// [MyAttribute(x = 5)]  
class MyClass  
{  
}  
  
public class MainClass  
{  
    public static void Main ()  
    {  
    }  
}  
```
