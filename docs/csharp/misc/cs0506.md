# <a name="compiler-error-cs0506"></a><span data-ttu-id="d9f84-102">Ошибка компилятора CS0506</span><span class="sxs-lookup"><span data-stu-id="d9f84-102">Compiler Error CS0506</span></span>
<span data-ttu-id="d9f84-103">"функция1": невозможно переопределить наследуемый член "функция2", так как он не помечен как "virtual", "abstract" или "override"</span><span class="sxs-lookup"><span data-stu-id="d9f84-103">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="d9f84-104">Был переопределен метод, не помеченный явно как **virtual**, **abstract**или `override`.</span><span class="sxs-lookup"><span data-stu-id="d9f84-104">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="d9f84-105">Следующий пример приводит к возникновению ошибки CS0506:</span><span class="sxs-lookup"><span data-stu-id="d9f84-105">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
