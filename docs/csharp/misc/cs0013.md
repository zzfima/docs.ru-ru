# <a name="compiler-error-cs0013"></a>Ошибка компилятора CS0013
Непредвиденная ошибка при записи метаданных в файл "ModelStore.dll" — "Не осталось логического пространства для создания дополнительных пользовательских строк".  
  
 Среде CLR платформы .NET Framework не удалось создать метаданные. Убедитесь в том, что задан правильный путь и диск не полон. Если проблема сохранится, возможно, восстановите или переустановите Visual Studio или .NET Framework.
