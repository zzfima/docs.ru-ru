# <a name="compiler-error-cs0520"></a>Ошибка компилятора CS0520
Предопределенный тип "имя" объявлен неправильно  
  
 Не удалось найти один или несколько файлов, необходимых для компилятора. Переустановите среду CLR платформы .NET Framework.
