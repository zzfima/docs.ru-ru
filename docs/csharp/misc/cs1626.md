# <a name="compiler-error-cs1626"></a><span data-ttu-id="188bd-102">Ошибка компилятора CS1626</span><span class="sxs-lookup"><span data-stu-id="188bd-102">Compiler Error CS1626</span></span>
<span data-ttu-id="188bd-103">Нельзя использовать оператор yield в теле блока try, имеющего предложение catch.</span><span class="sxs-lookup"><span data-stu-id="188bd-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="188bd-104">Оператор yield нельзя использовать в блоке try, если имеется связанное с этим блоком предложение catch.</span><span class="sxs-lookup"><span data-stu-id="188bd-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="188bd-105">Чтобы избежать этой ошибки, переместите оператор yield за пределы предложения catch.</span><span class="sxs-lookup"><span data-stu-id="188bd-105">To avoid this error, move the yield statement out of the catch clause.</span></span>  
  
 <span data-ttu-id="188bd-106">При компиляции следующего примера возникнет ошибка CS1626:</span><span class="sxs-lookup"><span data-stu-id="188bd-106">The following sample generates CS1626:</span></span>  
  
```  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
