# <a name="compiler-error-cs0104"></a><span data-ttu-id="6db35-102">Ошибка компилятора CS0104</span><span class="sxs-lookup"><span data-stu-id="6db35-102">Compiler Error CS0104</span></span>
<span data-ttu-id="6db35-103">"ссылка" является неоднозначной ссылкой между "идентификатор" и "идентификатор"</span><span class="sxs-lookup"><span data-stu-id="6db35-103">'reference' is an ambiguous reference between 'identifier' and 'identifier'</span></span>  
  
 <span data-ttu-id="6db35-104">Программа содержит директивы [using](../../csharp/language-reference/keywords/using.md) для двух пространств имен, и код ссылается на имя, которое отображается в обоих пространствах имен.</span><span class="sxs-lookup"><span data-stu-id="6db35-104">Your program contains [using](../../csharp/language-reference/keywords/using.md) directives for two namespaces and your code references a name that appears in both namespaces.</span></span>  
  
 <span data-ttu-id="6db35-105">В следующем примере возникает ошибка CS0104:</span><span class="sxs-lookup"><span data-stu-id="6db35-105">The following sample generates CS0104:</span></span>  
  
```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }  
}  
  
namespace y  
{  
   public class Test  
   {  
   }  
}  
  
public class a  
{  
   public static void Main()  
   {  
      Test test = new Test();   // CS0104, is Test in x or y namespace?  
      // try the following line instead  
      // y.Test test = new y.Test();  
   }  
}  
```
