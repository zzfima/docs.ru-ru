# <a name="compiler-error-cs0674"></a>Ошибка компилятора CS0674
Не используйте System.ParamArrayAttribute. Используйте ключевое слово "params".  
  
 Компилятор C# не допускает использование <xref:System.ParamArrayAttribute?displayProperty=nameWithType>. Используйте вместо этого ключевое слово [params](../../csharp/language-reference/keywords/params.md) .  
  
 Следующий пример приводит к возникновению ошибки CS0674:  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
