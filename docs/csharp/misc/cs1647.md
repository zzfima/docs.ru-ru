# <a name="compiler-error-cs1647"></a>Ошибка компилятора CS1647
Выражение слишком длинное или сложное для компиляции рядом с "код"  
  
 Произошло переполнение стека в компиляторе, обрабатывающем ваш код. Чтобы устранить эту ошибку, упростите код. Если код является допустимым, обратитесь в службу [технической поддержки](http://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4).
