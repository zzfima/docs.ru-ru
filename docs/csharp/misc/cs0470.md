## <a name="example"></a><span data-ttu-id="3e186-107">Пример</span><span class="sxs-lookup"><span data-stu-id="3e186-107">Example</span></span>

```  
// CS0470.cs

interface I  
{  
   int P { get; }  
}

{  
   public int get_P() { return 0; }   // CS0470

}  
```
