# <a name="compiler-error-cs1055"></a><span data-ttu-id="812c9-102">Ошибка компилятора CS1055</span><span class="sxs-lookup"><span data-stu-id="812c9-102">Compiler Error CS1055</span></span>
<span data-ttu-id="812c9-103">Требуется функция доступа add или remove.</span><span class="sxs-lookup"><span data-stu-id="812c9-103">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="812c9-104">Если [событие](../../csharp/language-reference/keywords/event.md) не объявлено как поле, оно должно определять функции доступа **add** и **remove** .</span><span class="sxs-lookup"><span data-stu-id="812c9-104">If your [event](../../csharp/language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="812c9-105">Следующий пример приводит к возникновению ошибки CS1055:</span><span class="sxs-lookup"><span data-stu-id="812c9-105">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
