 <span data-ttu-id="6beb5-106">Следующий пример приводит к возникновению ошибки CS0526:</span><span class="sxs-lookup"><span data-stu-id="6beb5-106">The following sample generates CS0526:</span></span>  
  
```csharp

   public interface clx  
   {  
      public clx()   // CS0526  
      {  
      }  
   }

   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
