# Предупреждение компилятора (уровень 1) CS0602
Функция "старая\_функция" устарела. Используйте вместо нее функцию "новая\_функция"  
  
 Функция языка, используемая в коде \(*старая\_функция*\) по\-прежнему поддерживается, но в будущих выпусках эта поддержка может быть прекращена. Вместо этого следует использовать рекомендованный синтаксис \(*новая\_функция*\).
