 <span data-ttu-id="2be89-104">[Статический](../../csharp/language-reference/keywords/static.md) конструктор не может объявляться с одним или несколькими параметрами.</span><span class="sxs-lookup"><span data-stu-id="2be89-104">A [static](../../csharp/language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="2be89-105">Дополнительные сведения см. в разделе [Конструкторы](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="2be89-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>

```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
