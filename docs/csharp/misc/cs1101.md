# <a name="compiler-error-cs1101"></a><span data-ttu-id="19bd8-102">Ошибка компилятора CS1101</span><span class="sxs-lookup"><span data-stu-id="19bd8-102">Compiler Error CS1101</span></span>
<span data-ttu-id="19bd8-103">Модификатор параметра "ref" не может использоваться с "this"</span><span class="sxs-lookup"><span data-stu-id="19bd8-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="19bd8-104">Изменение ключевым словом `this` первого параметра статического метода сообщает компилятору, что метод является методом расширения.</span><span class="sxs-lookup"><span data-stu-id="19bd8-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="19bd8-105">Никакие другие модификаторы не требуются и не допускаются в первом параметре метода расширения.</span><span class="sxs-lookup"><span data-stu-id="19bd8-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="19bd8-106">Пример</span><span class="sxs-lookup"><span data-stu-id="19bd8-106">Example</span></span>  
 <span data-ttu-id="19bd8-107">Следующий пример приводит к возникновению ошибки CS1101:</span><span class="sxs-lookup"><span data-stu-id="19bd8-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
