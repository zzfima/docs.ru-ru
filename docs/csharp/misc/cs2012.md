# <a name="compiler-error-cs2012"></a>Ошибка компилятора CS2012
Не удается открыть файл "файл" для записи  
  
 При использовании параметра компилятора [/bugreport:](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md)`file` не удалось открыть файл для записи. Убедитесь в том, что указано допустимое имя файла и что файл доступен не только для чтения.
