# <a name="compiler-error-cs0629"></a><span data-ttu-id="9b47a-102">Ошибка компилятора CS0629</span><span class="sxs-lookup"><span data-stu-id="9b47a-102">Compiler Error CS0629</span></span>
<span data-ttu-id="9b47a-103">Условный член "член" не может реализовывать член интерфейса "член_базового_класса" в типе "имя_типа"</span><span class="sxs-lookup"><span data-stu-id="9b47a-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="9b47a-104">[Условного](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) атрибут не может использоваться для реализации интерфейса.</span><span class="sxs-lookup"><span data-stu-id="9b47a-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="9b47a-105">В следующем примере возникает ошибка CS0629:</span><span class="sxs-lookup"><span data-stu-id="9b47a-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
