# <a name="compiler-warning-level-1-cs1592"></a>Предупреждение компилятора (уровень 1) CS1592
Некорректный XML во включенном файле комментариев — "причина"  
  
 Сообщило о проблеме, ***Причина***, был найден в файл, указанный параметром [ \<включают >](../../csharp/programming-guide/xmldoc/include.md) тега.
