# <a name="compiler-error-cs1525"></a><span data-ttu-id="a01df-102">Ошибка компилятора CS1525</span><span class="sxs-lookup"><span data-stu-id="a01df-102">Compiler Error CS1525</span></span>
<span data-ttu-id="a01df-103">Недопустимый термин "символ" в выражении</span><span class="sxs-lookup"><span data-stu-id="a01df-103">Invalid expression term 'character'</span></span>  
  
 <span data-ttu-id="a01df-104">Компилятор обнаружил недопустимый символ в выражении.</span><span class="sxs-lookup"><span data-stu-id="a01df-104">The compiler detected an invalid character in an expression.</span></span>  
  
 <span data-ttu-id="a01df-105">Следующий пример приводит к возникновению ошибки CS1525:</span><span class="sxs-lookup"><span data-stu-id="a01df-105">The following sample generates CS1525:</span></span>  
  
```csharp  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 <span data-ttu-id="a01df-106">Пустая метка также может приводить к возникновению ошибки CS1525, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="a01df-106">An empty label can also generate CS1525, as in the following sample:</span></span>  
  
```csharp  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```
