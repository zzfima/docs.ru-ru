# <a name="compiler-error-cs0077"></a>Ошибка компилятора CS0077
Оператор as необходимо использовать с типом ссылки или типом, допускающим значение null (int является типом, не допускающим значение null).  
  
 В оператор [as](../language-reference/operators/type-testing-and-cast.md#as-operator) был передан [тип значения](../language-reference/keywords/value-types.md). Поскольку `as` может возвращать [значение NULL](../language-reference/keywords/null.md), ему может быть передан только [ссылочный тип](../language-reference/keywords/reference-types.md) или [тип значения, допускающий значение NULL](../language-reference/builtin-types/nullable-value-types.md).
  
 Следующий пример приводит к возникновению ошибки CS0077:  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
