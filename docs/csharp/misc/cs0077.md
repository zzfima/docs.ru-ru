# <a name="compiler-error-cs0077"></a><span data-ttu-id="c9c84-102">Ошибка компилятора CS0077</span><span class="sxs-lookup"><span data-stu-id="c9c84-102">Compiler Error CS0077</span></span>
<span data-ttu-id="c9c84-103">Оператор as необходимо использовать с типом ссылки или типом, допускающим значение null (int является типом, не допускающим значение null).</span><span class="sxs-lookup"><span data-stu-id="c9c84-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="c9c84-104">В оператор [as](../language-reference/operators/type-testing-and-cast.md#as-operator) был передан [тип значения](../language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="c9c84-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="c9c84-105">Поскольку `as` может возвращать [значение NULL](../language-reference/keywords/null.md), он может передавать только [Ссылочные типы](../language-reference/keywords/reference-types.md) или типы значений, допускающие значение null.</span><span class="sxs-lookup"><span data-stu-id="c9c84-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed [reference types](../language-reference/keywords/reference-types.md) or nullable value types.</span></span> <span data-ttu-id="c9c84-106">Дополнительные сведения о типах значений, допускающих значения NULL, см. в разделе [типы значений Nullable](../programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="c9c84-106">For more information about nullable value types, see [Nullable value types](../programming-guide/nullable-types/index.md).</span></span>
  
 <span data-ttu-id="c9c84-107">Следующий пример приводит к возникновению ошибки CS0077:</span><span class="sxs-lookup"><span data-stu-id="c9c84-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
