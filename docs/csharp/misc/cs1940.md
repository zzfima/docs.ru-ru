# <a name="compiler-error-cs1940"></a><span data-ttu-id="1f14d-102">Ошибка компилятора CS1940</span><span class="sxs-lookup"><span data-stu-id="1f14d-102">Compiler Error CS1940</span></span>
<span data-ttu-id="1f14d-103">Обнаружено несколько реализаций шаблона запроса для типа источника "тип".</span><span class="sxs-lookup"><span data-stu-id="1f14d-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="1f14d-104">Неоднозначный вызов метода "метод".</span><span class="sxs-lookup"><span data-stu-id="1f14d-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="1f14d-105">Эта ошибка возникает в том случае, если определено несколько реализаций метода запроса, и компилятор не может выбрать, какой из них лучше всего использовать для запроса.</span><span class="sxs-lookup"><span data-stu-id="1f14d-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="1f14d-106">В следующем примере обе версии `Select` имеют одинаковые сигнатуры, так как они принимают одно значение `int` в качестве входного параметра и имеют `int` в качестве возвращаемого значения.</span><span class="sxs-lookup"><span data-stu-id="1f14d-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

1.  <span data-ttu-id="1f14d-108">Предоставьте только одну реализацию для каждого метода.</span><span class="sxs-lookup"><span data-stu-id="1f14d-108">Provide only one implementation for each method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1f14d-109">Пример</span><span class="sxs-lookup"><span data-stu-id="1f14d-109">Example</span></span>  
 <span data-ttu-id="1f14d-110">Следующий код приводит к возникновению ошибки CS1940:</span><span class="sxs-lookup"><span data-stu-id="1f14d-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
