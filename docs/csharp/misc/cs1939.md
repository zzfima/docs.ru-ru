# <a name="compiler-error-cs1939"></a><span data-ttu-id="05499-102">Ошибка компилятора CS1939</span><span class="sxs-lookup"><span data-stu-id="05499-102">Compiler Error CS1939</span></span>

 <span data-ttu-id="05499-104">Переменная диапазона доступна только для чтения, она используется в выражении запроса и служит идентификатором каждого следующего элемента в исходной последовательности.</span><span class="sxs-lookup"><span data-stu-id="05499-104">A range variable is a read-only variable that is introduced in a query expression and serves as an identifier for each successive element in a source sequence.</span></span> <span data-ttu-id="05499-105">Так как ее невозможно изменить каким-либо образом, ее не нужно передавать с модификатором `ref` или `out`.</span><span class="sxs-lookup"><span data-stu-id="05499-105">Because it cannot be modified in any way, there is no point in passing it by `ref` or `out`.</span></span> <span data-ttu-id="05499-106">Следовательно, обе операции недопустимы.</span><span class="sxs-lookup"><span data-stu-id="05499-106">Therefore, both operations are not valid.</span></span>

## <a name="example"></a><span data-ttu-id="05499-109">Пример</span><span class="sxs-lookup"><span data-stu-id="05499-109">Example</span></span>  
 <span data-ttu-id="05499-110">При компиляции следующего примера возникнет ошибка CS1939:</span><span class="sxs-lookup"><span data-stu-id="05499-110">The following example generates CS1939:</span></span>  
  
```csharp  
// cs1939.cs  
using System.Linq;  
class Test  
{  
    public static void F(ref int i)  
    {  
    }  
    public static void Main()  
    {  
        var list = new int[] { 0, 1, 2, 3, 4, 5 };  
        var q = from x in list  
                let k = x  
                select Test.F(ref x); // CS1939  
    }  
}  
```
