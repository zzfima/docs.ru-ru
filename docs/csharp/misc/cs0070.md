# <a name="compiler-error-cs0070"></a><span data-ttu-id="bb0bb-102">Ошибка компилятора CS0070</span><span class="sxs-lookup"><span data-stu-id="bb0bb-102">Compiler Error CS0070</span></span>
<span data-ttu-id="bb0bb-103">Событие "событие" может присутствовать только в левой части операций += и -= (кроме случая использования в типе "тип")</span><span class="sxs-lookup"><span data-stu-id="bb0bb-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="bb0bb-104">За пределами класса, в котором оно определено, [событие](../../csharp/language-reference/keywords/event.md) может только добавлять или убирать ссылки.</span><span class="sxs-lookup"><span data-stu-id="bb0bb-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="bb0bb-105">Дополнительные сведения см. в статье [Events (Visual Basic)](../../csharp/programming-guide/events/index.md) (События в Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="bb0bb-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="bb0bb-106">В следующем примере возникает ошибка CS0070:</span><span class="sxs-lookup"><span data-stu-id="bb0bb-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
