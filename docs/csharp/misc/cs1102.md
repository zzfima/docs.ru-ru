# <a name="compiler-error-cs1102"></a><span data-ttu-id="03b8d-102">Ошибка компилятора CS1102</span><span class="sxs-lookup"><span data-stu-id="03b8d-102">Compiler Error CS1102</span></span>
<span data-ttu-id="03b8d-103">Модификатор параметра "out" не может использоваться с "this"</span><span class="sxs-lookup"><span data-stu-id="03b8d-103">The parameter modifier 'out' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="03b8d-104">Изменение ключевым словом `this` первого параметра статического метода сообщает компилятору, что метод является методом расширения.</span><span class="sxs-lookup"><span data-stu-id="03b8d-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="03b8d-105">Никакие другие модификаторы не требуются и не допускаются в первом параметре метода расширения.</span><span class="sxs-lookup"><span data-stu-id="03b8d-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="03b8d-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="03b8d-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="03b8d-107">Удалите неразрешенные модификаторы из первого параметра.</span><span class="sxs-lookup"><span data-stu-id="03b8d-107">Remove the unauthorized modifiers from the first parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="03b8d-108">Пример</span><span class="sxs-lookup"><span data-stu-id="03b8d-108">Example</span></span>  
 <span data-ttu-id="03b8d-109">Следующий пример приводит к возникновению ошибки CS1102:</span><span class="sxs-lookup"><span data-stu-id="03b8d-109">The following example generates CS1102:</span></span>  
  
```  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
