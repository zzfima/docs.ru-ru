# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="0163d-102">Предупреждение компилятора (уровень 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="0163d-102">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="0163d-103">Возможно непреднамеренное сравнение ссылок; для получения сравнения значений приведите левую часть к типу "тип".</span><span class="sxs-lookup"><span data-stu-id="0163d-103">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="0163d-104">Компилятор выполняет сравнение ссылок.</span><span class="sxs-lookup"><span data-stu-id="0163d-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="0163d-105">Если нужно сравнить значения строк, приведите левую часть выражения к `type`.</span><span class="sxs-lookup"><span data-stu-id="0163d-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="0163d-106">Следующий пример приводит к возникновению ошибки CS0252:</span><span class="sxs-lookup"><span data-stu-id="0163d-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
