# <a name="compiler-error-cs0765"></a><span data-ttu-id="852d3-102">Ошибка компилятора CS0765</span><span class="sxs-lookup"><span data-stu-id="852d3-102">Compiler Error CS0765</span></span>
<span data-ttu-id="852d3-103">В деревьях выражений не могут использоваться разделяемые методы, имеющие только определяющее объявление или только удаленные условные методы.</span><span class="sxs-lookup"><span data-stu-id="852d3-103">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="852d3-104">Хотя вызов удаленного разделяемого метода является выражением, это недопустимое выражение в дереве выражения.</span><span class="sxs-lookup"><span data-stu-id="852d3-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="852d3-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="852d3-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="852d3-106">Добавьте реализующее объявление разделяемого метода или удалите код, который вызывает исключение условного метода из компиляции.</span><span class="sxs-lookup"><span data-stu-id="852d3-106">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="852d3-107">Пример</span><span class="sxs-lookup"><span data-stu-id="852d3-107">Example</span></span>  
 <span data-ttu-id="852d3-108">В следующем коде ошибка CS0765 возникает в двух местах:</span><span class="sxs-lookup"><span data-stu-id="852d3-108">The following code generates CS0765 in two locations:</span></span>  
  
```  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
