<span data-ttu-id="1e255-103">Член "член" не скрывает унаследованный член.</span><span class="sxs-lookup"><span data-stu-id="1e255-103">The member 'member' does not hide an inherited member.</span></span> <span data-ttu-id="1e255-104">Новое ключевое слово не требуется</span><span class="sxs-lookup"><span data-stu-id="1e255-104">The new keyword is not required</span></span>  
  
 <span data-ttu-id="1e255-105">Объявление класса включает ключевое слово [new](../language-reference/keywords/new-modifier.md) , хотя это объявление не переопределяет существующее определение в базовом классе.</span><span class="sxs-lookup"><span data-stu-id="1e255-105">A class declaration included the [new](../language-reference/keywords/new-modifier.md) keyword even though the declaration does not override an existing declaration in a base class.</span></span> <span data-ttu-id="1e255-106">Вы можете удалить ключевое слово **new** .</span><span class="sxs-lookup"><span data-stu-id="1e255-106">You can delete the **new** keyword.</span></span>  
  
 <span data-ttu-id="1e255-107">Следующий пример приводит к возникновению предупреждения CS0109:</span><span class="sxs-lookup"><span data-stu-id="1e255-107">The following sample generates CS0109:</span></span>  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
