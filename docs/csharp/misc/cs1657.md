 <span data-ttu-id="0eee5-104">Эта ошибка происходит при передаче переменной как аргумента [ref](../../csharp/language-reference/keywords/ref.md) или [out](../../csharp/language-reference/keywords/out.md) в контексте, когда переменная доступна только для чтения.</span><span class="sxs-lookup"><span data-stu-id="0eee5-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="0eee5-105">Контексты только для чтения включают переменные итерации [foreach](../../csharp/language-reference/keywords/foreach-in.md) , переменные [using](../../csharp/language-reference/keywords/using-statement.md) и переменные `fixed` .</span><span class="sxs-lookup"><span data-stu-id="0eee5-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="0eee5-106">Чтобы устранить эту ошибку, не вызывайте функции, принимающие переменную `foreach`, `using` или `fixed` в качестве параметра `ref` или `out` в блоках `using` , операторах `foreach` и `fixed` .</span><span class="sxs-lookup"><span data-stu-id="0eee5-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0eee5-107">Пример</span><span class="sxs-lookup"><span data-stu-id="0eee5-107">Example</span></span>  
 <span data-ttu-id="0eee5-108">Следующий пример приводит к возникновению ошибки CS1657:</span><span class="sxs-lookup"><span data-stu-id="0eee5-108">The following sample generates CS1657:</span></span>  
  
```  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
