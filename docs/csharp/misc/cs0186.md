 <span data-ttu-id="a5639-104">В следующем примере возникает ошибка CS0186:</span><span class="sxs-lookup"><span data-stu-id="a5639-104">The following sample generates CS0186:</span></span>  
  
```csharp  
// CS0186.cs  
using System;  
using System.Collections;  
  
class MyClass   
{  
   static void Main()   
   {  
      // Each of the following lines generates CS0186:  
      foreach (int i in null) {}   // CS0186  
      foreach (int i in (IEnumerable) null) { };   // CS0186  
   }  
}  
```
