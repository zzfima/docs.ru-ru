# <a name="compiler-error-cs1575"></a><span data-ttu-id="4053f-102">Ошибка компилятора CS1575</span><span class="sxs-lookup"><span data-stu-id="4053f-102">Compiler Error CS1575</span></span>
<span data-ttu-id="4053f-103">В выражении stackalloc после типа требуется []</span><span class="sxs-lookup"><span data-stu-id="4053f-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="4053f-104">Размер выделенной памяти, запрошенной с помощью оператора [stackalloc](../language-reference/operators/stackalloc.md), необходимо указывать в квадратных скобках.</span><span class="sxs-lookup"><span data-stu-id="4053f-104">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="4053f-105">Следующий пример приводит к возникновению ошибки CS1575:</span><span class="sxs-lookup"><span data-stu-id="4053f-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
