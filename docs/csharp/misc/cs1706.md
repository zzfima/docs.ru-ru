## <a name="example"></a><span data-ttu-id="85811-107">Пример</span><span class="sxs-lookup"><span data-stu-id="85811-107">Example</span></span>  
 <span data-ttu-id="85811-108">Следующий пример приводит к возникновению ошибки CS1706:</span><span class="sxs-lookup"><span data-stu-id="85811-108">The following example generates CS1706.</span></span>

```csharp  
class MyAttribute : Attribute  
{  
    public MyAttribute(MyDelegate d) { }  
}  
  
// Anonymous Method in Attribute declaration is not allowed.  
[MyAttribute(delegate{/* anonymous Method in Attribute declaration */})]  // CS1706  
class Program  
{  
}  
```
