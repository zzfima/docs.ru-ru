# <a name="compiler-error-cs1608"></a>Ошибка компилятора CS1608
Атрибут Required не разрешен для типов C#  
  
 <xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> не разрешен в типах, определенных в коде C#.
