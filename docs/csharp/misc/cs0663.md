# <a name="compiler-error-cs0663"></a><span data-ttu-id="c932e-102">Ошибка компилятора CS0663</span><span class="sxs-lookup"><span data-stu-id="c932e-102">Compiler Error CS0663</span></span>
<span data-ttu-id="c932e-103">Не может определять перегруженные методы, которые различаются только параметрами ref и out</span><span class="sxs-lookup"><span data-stu-id="c932e-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="c932e-104">Методы, которые отличаются только использованием модификаторов [в](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) и [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) в параметре не допускаются.</span><span class="sxs-lookup"><span data-stu-id="c932e-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="c932e-105">Следующий пример приводит к возникновению ошибки CS0663:</span><span class="sxs-lookup"><span data-stu-id="c932e-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
