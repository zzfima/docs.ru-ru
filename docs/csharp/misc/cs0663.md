# <a name="compiler-error-cs0663"></a><span data-ttu-id="f556d-102">Ошибка компилятора CS0663</span><span class="sxs-lookup"><span data-stu-id="f556d-102">Compiler Error CS0663</span></span>
<span data-ttu-id="f556d-103">Не может определять перегруженные методы, которые различаются только параметрами ref и out</span><span class="sxs-lookup"><span data-stu-id="f556d-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="f556d-104">Не допускается применять методы, которые отличаются только использованием модификаторов [ref](../../csharp/language-reference/keywords/ref.md) и [out](../../csharp/language-reference/keywords/out.md) в параметре.</span><span class="sxs-lookup"><span data-stu-id="f556d-104">Methods that differ only on their use of [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="f556d-105">Следующий пример приводит к возникновению ошибки CS0663:</span><span class="sxs-lookup"><span data-stu-id="f556d-105">The following sample generates CS0663:</span></span>  
  
```  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
