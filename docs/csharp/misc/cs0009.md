# <a name="compiler-error-cs0009"></a>Ошибка компилятора CS0009
Не удалось открыть файл метаданных "файл" — "описание"  
  
 Файл, заданный параметром компилятора [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) , не содержит допустимые метаданные.
