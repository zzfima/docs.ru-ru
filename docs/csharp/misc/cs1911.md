# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="0d2db-102">Предупреждение компилятора (уровень 1) CS1911</span><span class="sxs-lookup"><span data-stu-id="0d2db-102">Compiler Warning (level 1) CS1911</span></span>
<span data-ttu-id="0d2db-103">Доступ к члену "имя" через ключевое слово "base" из анонимного метода, лямбда-выражения, выражения запроса или итератора приводит к появлению непроверяемого кода.</span><span class="sxs-lookup"><span data-stu-id="0d2db-103">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="0d2db-104">Рекомендуется переместить доступ во вспомогательный метод во вмещающем типе.</span><span class="sxs-lookup"><span data-stu-id="0d2db-104">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="0d2db-105">Вызов виртуальных функций с ключевым словом `base` в теле метода итератора или в анонимных методах приведет к непроверяемому коду.</span><span class="sxs-lookup"><span data-stu-id="0d2db-105">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="0d2db-106">Непроверяемый код не сможет запуститься в среде с частичным доверием.</span><span class="sxs-lookup"><span data-stu-id="0d2db-106">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="0d2db-107">Один из способов устранения ошибки CS1911 состоит в том, чтобы переместить вызов виртуальной функции во вспомогательную функцию.</span><span class="sxs-lookup"><span data-stu-id="0d2db-107">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0d2db-108">Пример</span><span class="sxs-lookup"><span data-stu-id="0d2db-108">Example</span></span>  
 <span data-ttu-id="0d2db-109">Следующий пример приводит к возникновению ошибки CS1911:</span><span class="sxs-lookup"><span data-stu-id="0d2db-109">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
