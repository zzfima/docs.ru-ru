# <a name="compiler-error-cs1951"></a><span data-ttu-id="38cff-102">Ошибка компилятора CS1951</span><span class="sxs-lookup"><span data-stu-id="38cff-102">Compiler Error CS1951</span></span>
<span data-ttu-id="38cff-103">Дерево лямбда-выражения не может содержать in, out или ref параметра.</span><span class="sxs-lookup"><span data-stu-id="38cff-103">An expression tree lambda may not contain an in, out, or ref parameter.</span></span>  
  
 <span data-ttu-id="38cff-104">Дерево выражений лишь представляет выражения в виде структур данных.</span><span class="sxs-lookup"><span data-stu-id="38cff-104">An expression tree just represents expressions as data structures.</span></span> <span data-ttu-id="38cff-105">В нем не предусмотрен способ представления определенных мест в памяти, что необходимо при передаче параметра по ссылке.</span><span class="sxs-lookup"><span data-stu-id="38cff-105">There is no way to represent specific memory locations as is required when you pass a parameter by reference.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="38cff-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="38cff-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="38cff-107">Единственный параметр — удалите модификатор в определении делегата и передаче параметра по значению.</span><span class="sxs-lookup"><span data-stu-id="38cff-107">The only option is to remove the modifier in the delegate definition and pass the parameter by value.</span></span>  
  
## <a name="example"></a><span data-ttu-id="38cff-108">Пример</span><span class="sxs-lookup"><span data-stu-id="38cff-108">Example</span></span>  
 <span data-ttu-id="38cff-109">Следующий пример приводит к возникновению ошибки CS1951:</span><span class="sxs-lookup"><span data-stu-id="38cff-109">The following example generates CS1951:</span></span>

```csharp  
// cs1951.cs  
using System.Linq;  
public delegate int TestDelegate(ref int i);  
class Test  
{  
    static void Main()

    }

```
