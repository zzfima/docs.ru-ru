# Ошибка компилятора CS1514
Требуется "{"  
  
 Компилятор ожидал закрывающую фигурную скобку \(`{`\), которая не была найдена.  
  
 В следующем примере возникает ошибка CS1514:  
  
```  
// CS1514.cs namespace x // CS1514, no open curly brace  
```
