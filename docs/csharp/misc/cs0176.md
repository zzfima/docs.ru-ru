# <a name="compiler-error-cs0176"></a><span data-ttu-id="e2f1f-102">Ошибка компилятора CS0176</span><span class="sxs-lookup"><span data-stu-id="e2f1f-102">Compiler Error CS0176</span></span>
<span data-ttu-id="e2f1f-103">Доступ к статическому члену "член" по ссылке на экземпляр невозможен. Уточните имя, указав тип.</span><span class="sxs-lookup"><span data-stu-id="e2f1f-103">Static member 'member' cannot be accessed with an instance reference; qualify it with a type name instead</span></span>  
  
 <span data-ttu-id="e2f1f-104">Для уточнения [статической](../language-reference/keywords/static.md) переменной можно использовать только имя класса; имя экземпляра не может быть квалификатором.</span><span class="sxs-lookup"><span data-stu-id="e2f1f-104">Only a class name can be used to qualify a [static](../language-reference/keywords/static.md) variable; an instance name cannot be a qualifier.</span></span> <span data-ttu-id="e2f1f-105">Дополнительные сведения см. в статье [Статические классы и члены статических классов](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="e2f1f-105">For more information, see [Static Classes and Static Class Members](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span>  
  
 <span data-ttu-id="e2f1f-106">При компиляции следующего примера возникнет ошибка CS0176:</span><span class="sxs-lookup"><span data-stu-id="e2f1f-106">The following sample generates CS0176:</span></span>  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
