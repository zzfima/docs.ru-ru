# <a name="compiler-error-cs1920"></a><span data-ttu-id="f6c66-102">Ошибка компилятора CS1920</span><span class="sxs-lookup"><span data-stu-id="f6c66-102">Compiler Error CS1920</span></span>
<span data-ttu-id="f6c66-103">Инициализатор элемента не может быть пустым.</span><span class="sxs-lookup"><span data-stu-id="f6c66-103">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="f6c66-104">Инициализатор коллекции состоит из последовательности инициализаторов элементов.</span><span class="sxs-lookup"><span data-stu-id="f6c66-104">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="f6c66-105">Инициализаторы элементов не требуется заключать в фигурные скобки, если они не содержат выражение назначения.</span><span class="sxs-lookup"><span data-stu-id="f6c66-105">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="f6c66-106">Однако если вы указали фигурные скобки, они не могут быть пустыми.</span><span class="sxs-lookup"><span data-stu-id="f6c66-106">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="f6c66-107">Если инициализатор элемента является инициализатором объекта, фигурные скобки могут быть пустыми, пока инициализатор содержит выражение создания нового объекта.</span><span class="sxs-lookup"><span data-stu-id="f6c66-107">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f6c66-108">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="f6c66-108">To correct this error</span></span>

- <span data-ttu-id="f6c66-110">Если выражение предназначено быть инициализатором объектов, добавьте выражение создания нового объекта перед фигурными скобками.</span><span class="sxs-lookup"><span data-stu-id="f6c66-110">If the expression is intended to be an object initializer, add the new object creation expression in front of the braces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f6c66-111">Пример</span><span class="sxs-lookup"><span data-stu-id="f6c66-111">Example</span></span>  
 <span data-ttu-id="f6c66-112">Следующий пример приводит к возникновению ошибки CS1920:</span><span class="sxs-lookup"><span data-stu-id="f6c66-112">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```
