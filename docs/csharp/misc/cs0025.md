# Ошибка компилятора CS0025
Не удалось найти файл стандартной библиотеки "файл"  
  
 Отсутствует файл, необходимый для компилятора. Проверьте правильность пути и существование файла.  
  
 Если файл является системным файлом [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)], возможно, потребуется восстановить установку среды [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] или переустановить ее полностью.
