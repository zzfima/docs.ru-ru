# <a name="compiler-error-cs1649"></a><span data-ttu-id="92d34-102">Ошибка компилятора CS1649</span><span class="sxs-lookup"><span data-stu-id="92d34-102">Compiler Error CS1649</span></span>
<span data-ttu-id="92d34-103">Члены доступного только для чтения поля "идентификатор" могут передаваться как параметры с ключевым словом ref или out только в конструкторе</span><span class="sxs-lookup"><span data-stu-id="92d34-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="92d34-104">Эта ошибка возникает при передаче в функцию переменной, которая является членом поля `readonly` , в качестве аргумента `ref` или `out` .</span><span class="sxs-lookup"><span data-stu-id="92d34-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="92d34-105">Так как параметры `ref` и `out` могут быть изменены функцией, такая передача не разрешается.</span><span class="sxs-lookup"><span data-stu-id="92d34-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="92d34-106">Чтобы устранить эту ошибку, удалите ключевое слово `readonly` в поле или не передавайте члены поля `readonly` в функцию.</span><span class="sxs-lookup"><span data-stu-id="92d34-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="92d34-107">Например, вы можете создать временную переменную, которая может быть изменена, и передать ее как аргумент `ref` , как показано в следующем примере.</span><span class="sxs-lookup"><span data-stu-id="92d34-107">For example, you might try creating a temporary variable which can be modifed and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="92d34-108">Пример</span><span class="sxs-lookup"><span data-stu-id="92d34-108">Example</span></span>  
 <span data-ttu-id="92d34-109">Следующий пример приводит к возникновению ошибки CS1649:</span><span class="sxs-lookup"><span data-stu-id="92d34-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();   
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
