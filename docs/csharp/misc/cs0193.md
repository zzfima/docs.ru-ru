# <a name="compiler-error-cs0193"></a><span data-ttu-id="d1be6-102">Ошибка компилятора CS0193</span><span class="sxs-lookup"><span data-stu-id="d1be6-102">Compiler Error CS0193</span></span>
<span data-ttu-id="d1be6-103">К указателю должен быть применен оператор * или ->.</span><span class="sxs-lookup"><span data-stu-id="d1be6-103">The * or -> operator must be applied to a pointer</span></span>  
  
 <span data-ttu-id="d1be6-104">Оператор **\*** или **->** используется с типом, не являющимся указателем.</span><span class="sxs-lookup"><span data-stu-id="d1be6-104">The **\*** or **->** operator was used with a nonpointer type.</span></span> <span data-ttu-id="d1be6-105">Дополнительные сведения см. в разделе [Типы указателей](../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md).</span><span class="sxs-lookup"><span data-stu-id="d1be6-105">For more information, see [Pointer types](../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md).</span></span>  
  
 <span data-ttu-id="d1be6-106">Следующий пример приводит к возникновению ошибки CS0193:</span><span class="sxs-lookup"><span data-stu-id="d1be6-106">The following sample generates CS0193:</span></span>  
  
```  
// CS0193.cs  
using System;  
  
public struct Age  
{  
   public int AgeYears;  
   public int AgeMonths;  
   public int AgeDays;  
}  
  
public class MyClass  
{  
   public static void SetAge(ref Age anAge, int years, int months, int days)  
   {  
      anAge->Months = 3;   // CS0193, anAge is not a pointer  
      // try the following line instead  
      // anAge.AgeMonths = 3;  
   }  
  
   public static void Main()  
   {  
      Age MyAge = new Age();  
      Console.WriteLine(MyAge.AgeMonths);  
      SetAge(ref MyAge, 22, 4, 15);  
      Console.WriteLine(MyAge.AgeMonths);  
   }  
}  
```
