# <a name="compiler-error-cs0155"></a><span data-ttu-id="5895c-102">Ошибка компилятора CS0155</span><span class="sxs-lookup"><span data-stu-id="5895c-102">Compiler Error CS0155</span></span>
<span data-ttu-id="5895c-103">Тип в операторах caught или thrown должен быть производным от System.Exception.</span><span class="sxs-lookup"><span data-stu-id="5895c-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="5895c-104">Выполнена попытка передачи типа данных, который не является производным от **System.Exception** , в блок [catch](../../csharp/language-reference/keywords/try-catch.md) .</span><span class="sxs-lookup"><span data-stu-id="5895c-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="5895c-105">В блок **catch** можно передавать только типы данных, которые являются производными от **System.Exception** .</span><span class="sxs-lookup"><span data-stu-id="5895c-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="5895c-106">Дополнительные сведения см. в разделе [операторы обработки исключений](../../csharp/language-reference/keywords/exception-handling-statements.md) и [исключения и обработка исключений](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="5895c-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="5895c-107">Следующий пример приводит к возникновению ошибки CS0155:</span><span class="sxs-lookup"><span data-stu-id="5895c-107">The following sample generates CS0155:</span></span>  
  
```  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
