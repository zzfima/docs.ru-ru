# <a name="compiler-error-cs1918"></a><span data-ttu-id="c1e19-102">Ошибка компилятора CS1918</span><span class="sxs-lookup"><span data-stu-id="c1e19-102">Compiler Error CS1918</span></span>
<span data-ttu-id="c1e19-103">Членам свойства "имя" типа "тип" не могут быть присвоены значения с помощью инициализатора объекта, поскольку они имеют тип значения</span><span class="sxs-lookup"><span data-stu-id="c1e19-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="c1e19-104">Эта ошибка возникает при попытке использовать инициализатор объекта для инициализации свойств структуры, которая сама является свойством инициализируемого класса.</span><span class="sxs-lookup"><span data-stu-id="c1e19-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c1e19-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="c1e19-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="c1e19-106">Если в инициализаторе объекта необходимо полностью инициализировать поля свойства, измените структуру на класс.</span><span class="sxs-lookup"><span data-stu-id="c1e19-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="c1e19-107">В противном случае после создания объекта с помощью инициализатора объекта инициализируйте члены структуры в вызове отдельного метода.</span><span class="sxs-lookup"><span data-stu-id="c1e19-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c1e19-108">Пример</span><span class="sxs-lookup"><span data-stu-id="c1e19-108">Example</span></span>  
 <span data-ttu-id="c1e19-109">В следующем примере возникает ошибка CS1918:</span><span class="sxs-lookup"><span data-stu-id="c1e19-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
