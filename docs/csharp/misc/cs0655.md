## <a name="example"></a><span data-ttu-id="96348-105">Пример</span><span class="sxs-lookup"><span data-stu-id="96348-105">Example</span></span>  
 <span data-ttu-id="96348-106">Следующий пример приводит к возникновению ошибки CS0655:</span><span class="sxs-lookup"><span data-stu-id="96348-106">The following sample generates CS0655:</span></span>  
  
```  
// CS0655.cs  
using System;  
  
class MyAttribute : Attribute  
{  
    // decimal is not valid attribute parameter type  
    public decimal d = 0;  
    public int e = 0;  
}  
  
[My(d = 0)]   // CS0655  
// Try the following line instead:  
// [My(e = 0)]  
class C  
{  
    public static void Main()  
    {  
    }  
}  
```
