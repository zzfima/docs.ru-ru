# <a name="compiler-error-cs0466"></a><span data-ttu-id="567c6-102">Ошибка компилятора CS0466</span><span class="sxs-lookup"><span data-stu-id="567c6-102">Compiler Error CS0466</span></span>
<span data-ttu-id="567c6-103">"метод1" не должен иметь параметр params, так как у "метод2" его нет</span><span class="sxs-lookup"><span data-stu-id="567c6-103">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="567c6-104">Нельзя применять параметр `params` для члена класса, если он не используется реализуемым интерфейсом.</span><span class="sxs-lookup"><span data-stu-id="567c6-104">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="567c6-105">Пример</span><span class="sxs-lookup"><span data-stu-id="567c6-105">Example</span></span>  
 <span data-ttu-id="567c6-106">Следующий пример приводит к возникновению ошибки CS0466:</span><span class="sxs-lookup"><span data-stu-id="567c6-106">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
