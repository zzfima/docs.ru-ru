# <a name="compiler-error-cs0100"></a>Ошибка компилятора CS0100
Имя параметра "имя_параметра" является дубликатом  
  
 В объявлении метода несколько раз используется одно и то же имя параметра. Имена параметров должны быть уникальными в объявлении метода. Дополнительные сведения см. в статье [Методы](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 В следующем примере возникает ошибка CS0100:  
  
```  
// CS0100.cs  
namespace x  
{  
   public class a  
   {  
      public static void f(int i, char i)   // CS0100  
      // try the following line instead  
      // public static void f(int i, char j)  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
