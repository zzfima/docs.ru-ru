# <a name="compiler-error-cs1578"></a><span data-ttu-id="17869-102">Ошибка компилятора CS1578</span><span class="sxs-lookup"><span data-stu-id="17869-102">Compiler Error CS1578</span></span>
<span data-ttu-id="17869-103">Требуется имя файла, однострочный комментарий или признак конца строки</span><span class="sxs-lookup"><span data-stu-id="17869-103">Filename, single-line comment or end-of-line expected</span></span>  
  
 <span data-ttu-id="17869-104">После директивы [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) допускается только имя файла (заключенное в двойные кавычки) или однострочный комментарий.</span><span class="sxs-lookup"><span data-stu-id="17869-104">After a [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="17869-105">Следующий пример приводит к возникновению ошибки CS1578:</span><span class="sxs-lookup"><span data-stu-id="17869-105">The following sample generates CS1578:</span></span>  
  
```csharp  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
