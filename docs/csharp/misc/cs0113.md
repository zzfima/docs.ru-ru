# <a name="compiler-error-cs0113"></a><span data-ttu-id="208a2-102">Ошибка компилятора CS0113</span><span class="sxs-lookup"><span data-stu-id="208a2-102">Compiler Error CS0113</span></span>
<span data-ttu-id="208a2-103">Член "функция", помеченный как override, не может быть помечен как new или virtual</span><span class="sxs-lookup"><span data-stu-id="208a2-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="208a2-104">Пометка метода ключевыми словами [new](../language-reference/keywords/new-modifier.md) и [override](../language-reference/keywords/override.md) является взаимоисключающей.</span><span class="sxs-lookup"><span data-stu-id="208a2-104">It is mutually exclusive to mark a method with the [new](../language-reference/keywords/new-modifier.md) and [override](../language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="208a2-105">Следующий пример приводит к возникновению ошибки CS0113:</span><span class="sxs-lookup"><span data-stu-id="208a2-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
