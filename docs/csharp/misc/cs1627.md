# <a name="compiler-error-cs1627"></a><span data-ttu-id="a010d-102">Ошибка компилятора CS1627</span><span class="sxs-lookup"><span data-stu-id="a010d-102">Compiler Error CS1627</span></span>
<span data-ttu-id="a010d-103">Требуется выражение после оператора yield return.</span><span class="sxs-lookup"><span data-stu-id="a010d-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="a010d-104">Эта ошибка возникает, если `yield` используется без выражения.</span><span class="sxs-lookup"><span data-stu-id="a010d-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="a010d-105">Чтобы избежать этой ошибки, вставьте в оператор соответствующее выражение.</span><span class="sxs-lookup"><span data-stu-id="a010d-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="a010d-106">Следующий пример приводит к возникновению ошибки CS1627:</span><span class="sxs-lookup"><span data-stu-id="a010d-106">The following sample generates CS1627:</span></span>  
  
```  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
