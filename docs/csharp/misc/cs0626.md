# <a name="compiler-warning-level-1-cs0626"></a><span data-ttu-id="57894-102">Предупреждение компилятора (уровень 1) CS0626</span><span class="sxs-lookup"><span data-stu-id="57894-102">Compiler Warning (level 1) CS0626</span></span>
<span data-ttu-id="57894-103">Метод, оператор или метод доступа "метод" помечен как внешний и не имеет атрибутов.</span><span class="sxs-lookup"><span data-stu-id="57894-103">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="57894-104">Для указания на внешнюю реализацию, возможно, следует добавить атрибут DllImport</span><span class="sxs-lookup"><span data-stu-id="57894-104">Consider adding a DllImport attribute to specify the external implementation</span></span>  
  
 <span data-ttu-id="57894-105">Метод, помеченный `extern` должен также быть помечен атрибутом, например, [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) атрибута.</span><span class="sxs-lookup"><span data-stu-id="57894-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>
  
 <span data-ttu-id="57894-106">Атрибут указывает, где реализуется метод.</span><span class="sxs-lookup"><span data-stu-id="57894-106">The attribute specifies where the method is implemented.</span></span> <span data-ttu-id="57894-107">Во время выполнения программе потребуется эта информация.</span><span class="sxs-lookup"><span data-stu-id="57894-107">At run time, the program will need this information.</span></span>  
  
 <span data-ttu-id="57894-108">При компиляции следующего примера будет выдано предупреждение CS0626:</span><span class="sxs-lookup"><span data-stu-id="57894-108">The following sample generates CS0626:</span></span>  
  
```csharp
// CS0626.cs  
// compile with: /warnaserror  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   static extern public void M(); // CS0626  
   // try the following line  
   // [DllImport("mydll.dll")] static extern public void M();  
  
   public static void Main()  
   {  
   }  
}  
```
