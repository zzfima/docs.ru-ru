# <a name="compiler-error-cs1515"></a>Ошибка компилятора CS1515
Требуется "in"  
  
 В операторе [foreach](../../csharp/language-reference/keywords/foreach-in.md) отсутствует часть in.  
  
## <a name="example"></a>Пример  
 В следующем примере возникает ошибка CS1515:  
  
```csharp 
using System;  
  
class Driver  
{  
   static void Main()  
   {  
      int[] arr = new int[] {1, 2, 3};  
  
      // try the following line instead  
      // foreach (int x in arr)  
      foreach (int x arr)      // CS1515, "in" is missing  
      {  
         Console.WriteLine(x);  
      }  
   }  
}  
```
