## <a name="example"></a><span data-ttu-id="0cde6-106">Пример</span><span class="sxs-lookup"><span data-stu-id="0cde6-106">Example</span></span>  
 <span data-ttu-id="0cde6-107">Следующий пример приводит к возникновению ошибки CS0535.</span><span class="sxs-lookup"><span data-stu-id="0cde6-107">The following sample generates CS0535.</span></span>  
  
```  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
