# <a name="compiler-error-cs0535"></a><span data-ttu-id="a5aaf-102">Ошибка компилятора CS0535</span><span class="sxs-lookup"><span data-stu-id="a5aaf-102">Compiler Error CS0535</span></span>
<span data-ttu-id="a5aaf-103">Класс "класс" не реализует член интерфейса "член"</span><span class="sxs-lookup"><span data-stu-id="a5aaf-103">'class' does not implement interface member 'member'</span></span>  
  
 <span data-ttu-id="a5aaf-104">[Класс](../language-reference/keywords/class.md) является производным от [интерфейса](../language-reference/keywords/interface.md), но не реализует один или несколько членов интерфейса.</span><span class="sxs-lookup"><span data-stu-id="a5aaf-104">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="a5aaf-105">Класс должен реализовывать все члены интерфейсов, от которых он является производным, или должен объявляться как `abstract`.</span><span class="sxs-lookup"><span data-stu-id="a5aaf-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a5aaf-106">Пример</span><span class="sxs-lookup"><span data-stu-id="a5aaf-106">Example</span></span>  
 <span data-ttu-id="a5aaf-107">Следующий пример приводит к возникновению ошибки CS0535.</span><span class="sxs-lookup"><span data-stu-id="a5aaf-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
