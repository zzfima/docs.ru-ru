# <a name="compiler-error-cs0216"></a><span data-ttu-id="94609-102">Ошибка компилятора CS0216</span><span class="sxs-lookup"><span data-stu-id="94609-102">Compiler Error CS0216</span></span>
<span data-ttu-id="94609-103">Оператор "оператор" требует, чтобы был также определен соответствующий оператор "пропущенный_оператор"</span><span class="sxs-lookup"><span data-stu-id="94609-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="94609-104">Пользовательский оператор [true](../../csharp/language-reference/keywords/true.md) требует пользовательский оператор [false](../../csharp/language-reference/keywords/false.md) и наоборот.</span><span class="sxs-lookup"><span data-stu-id="94609-104">A user-defined [true](../../csharp/language-reference/keywords/true.md) operator requires a user-defined [false](../../csharp/language-reference/keywords/false.md) operator, and vice versa.</span></span> <span data-ttu-id="94609-105">Дополнительные сведения см. в разделе [Операторы](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span><span class="sxs-lookup"><span data-stu-id="94609-105">For more information, see [Operators](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span></span>  
  
 <span data-ttu-id="94609-106">Следующий пример приводит к возникновению ошибки CS0216:</span><span class="sxs-lookup"><span data-stu-id="94609-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
