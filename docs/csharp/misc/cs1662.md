# <a name="compiler-error-cs1662"></a><span data-ttu-id="13b6e-102">Ошибка компилятора CS1662</span><span class="sxs-lookup"><span data-stu-id="13b6e-102">Compiler Error CS1662</span></span>
<span data-ttu-id="13b6e-103">Не удается преобразовать блок анонимного метода в тип делегата "тип_делегата", так как некоторые типы возврата в блоке не могут быть неявно преобразованы в тип возврата делегата.</span><span class="sxs-lookup"><span data-stu-id="13b6e-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>  
  
 <span data-ttu-id="13b6e-104">Эта ошибка возникает, если оператор return блока анонимного метода имеет тип, который не может быть неявно преобразован в тип возврата делегата.</span><span class="sxs-lookup"><span data-stu-id="13b6e-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>  
  
 <span data-ttu-id="13b6e-105">В следующем примере возникает ошибка CS1662:</span><span class="sxs-lookup"><span data-stu-id="13b6e-105">The following sample generates CS1662:</span></span>  
  
```csharp  
// CS1662.cs  
  
delegate int MyDelegate(int i);  
  
class C  
{  
  
  public static void Main()  
  {  
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662  
     // Try this instead:  
     // MyDelegate d = dekegate(int i) { return (int)1.0; };  
  }  
}  
```
