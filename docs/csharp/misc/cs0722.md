 <span data-ttu-id="7195b-104">Использовать статический тип в качестве типа возвращаемого значения не имеет смысла, так как экземпляры статических типов создавать нельзя.</span><span class="sxs-lookup"><span data-stu-id="7195b-104">A static type as a return type is not meaningful since instances of static types cannot be created.</span></span>

```csharp  
// CS0722.cs

{  
}

{  
   public SC F()  // CS0722

      return null;  
   }

   {  
   }  
}  
```
