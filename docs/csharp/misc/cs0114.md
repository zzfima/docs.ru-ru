 <span data-ttu-id="2a705-108">Следующий пример приводит к возникновению ошибки CS0114:</span><span class="sxs-lookup"><span data-stu-id="2a705-108">The following sample generates CS0114:</span></span>  
  
```csharp  
// CS0114.cs

abstract public class clx  
{  
   public abstract void f();  
}  
  
public class cly : clx  
{  
   public void f() // CS0114, hides base class member  
   // try the following line instead  
   // override public void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
