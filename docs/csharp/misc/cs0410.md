## <a name="example"></a><span data-ttu-id="24a04-106">Пример</span><span class="sxs-lookup"><span data-stu-id="24a04-106">Example</span></span>

```csharp

class Test  
{  
    delegate void D(double d );

    {  
        D d = new D(F);  // CS0410
```
