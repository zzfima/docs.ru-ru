<span data-ttu-id="77857-103">"конструктор": статический конструктор не может иметь явный вызов конструктора с this или base</span><span class="sxs-lookup"><span data-stu-id="77857-103">'constructor' : static constructor cannot have an explicit 'this' or 'base' constructor call</span></span>  
  
 <span data-ttu-id="77857-104">Не допускается вызывать `this` в статическом конструкторе, так как статический конструктор вызывается автоматически до создания любого экземпляра класса.</span><span class="sxs-lookup"><span data-stu-id="77857-104">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="77857-105">Кроме того, статические конструкторы не наследуются и не могут вызываться непосредственно.</span><span class="sxs-lookup"><span data-stu-id="77857-105">Also, static constructors are not inherited, and cannot be called directly.</span></span>  
  
 <span data-ttu-id="77857-106">Дополнительные сведения см. в разделе [это](../../csharp/language-reference/keywords/this.md) и [базового](../../csharp/language-reference/keywords/base.md).</span><span class="sxs-lookup"><span data-stu-id="77857-106">For more information, see [this](../../csharp/language-reference/keywords/this.md) and [base](../../csharp/language-reference/keywords/base.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="77857-107">Пример</span><span class="sxs-lookup"><span data-stu-id="77857-107">Example</span></span>  
 <span data-ttu-id="77857-108">Следующий пример приводит к возникновению ошибки CS0514:</span><span class="sxs-lookup"><span data-stu-id="77857-108">The following example generates CS0514:</span></span>  
  
```

    static A() : base(0) // CS0514  
    {  
    }  
  
    public A(object o)  
    {  
    }

{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
