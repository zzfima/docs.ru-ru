<span data-ttu-id="e7c05-103">Разделяемый метод должен быть объявлен в разделяемом классе или разделяемой структуре</span><span class="sxs-lookup"><span data-stu-id="e7c05-103">A partial method must be declared in a partial class or partial struct</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="e7c05-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="e7c05-105">To correct this error</span></span>
