# <a name="compiler-error-cs1037"></a>Ошибка компилятора CS1037
Требуется перегружаемый оператор.  
  
 При определении комментария с помощью параметра [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md), компилятор обнаружил неправильную ссылку.
