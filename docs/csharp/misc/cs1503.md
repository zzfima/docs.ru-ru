# <a name="compiler-error-cs1503"></a>Ошибка компилятора CS1503
Аргумент "число" невозможно преобразовать из TypeA в TypeB  
  
 Тип одного аргумента в методе не совпадает с типом, переданным при создании экземпляра класса. Эта ошибка обычно появляется с CS1502. Сведения о том, как устранить эту ошибку, см. в описании ошибки [CS1502](../../csharp/language-reference/compiler-messages/cs1502.md) .
