 <span data-ttu-id="15731-104">Метка повторялась в операторе [переключения](../language-reference/keywords/switch.md) .</span><span class="sxs-lookup"><span data-stu-id="15731-104">A label was repeated in a [switch](../language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="15731-105">Для получения дополнительной информации см.  <span data-ttu-id="15731-106">В следующем примере возникает ошибка CS0152:</span><span class="sxs-lookup"><span data-stu-id="15731-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
