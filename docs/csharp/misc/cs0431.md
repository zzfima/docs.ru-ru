<span data-ttu-id="d5a9b-103">Использование псевдонима "идентификатор" с "::" невозможно, так как псевдоним ссылается на тип.</span><span class="sxs-lookup"><span data-stu-id="d5a9b-103">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="d5a9b-104">Вместо этого используйте ".".</span><span class="sxs-lookup"><span data-stu-id="d5a9b-104">Use '.' instead.</span></span>

 <span data-ttu-id="d5a9b-107">При компиляции следующего примера возникнет ошибка CS0431:</span><span class="sxs-lookup"><span data-stu-id="d5a9b-107">The following sample generates CS0431:</span></span>  
  
```  
// CS0431.cs  
using A = Outer;  
  
public class Outer   
{  
   public class Inner   
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
