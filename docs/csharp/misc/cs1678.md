# <a name="compiler-error-cs1678"></a><span data-ttu-id="4e256-102">Ошибка компилятора CS1678</span><span class="sxs-lookup"><span data-stu-id="4e256-102">Compiler Error CS1678</span></span>
<span data-ttu-id="4e256-103">Параметр "число" объявлен как тип "тип 1", но должен быть "тип2"</span><span class="sxs-lookup"><span data-stu-id="4e256-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="4e256-104">Эта ошибка возникает, когда тип параметра в анонимном методе отличается от объявления делегата, к которому выполняется приведение метода.</span><span class="sxs-lookup"><span data-stu-id="4e256-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="4e256-105">В следующем примере возникает ошибка CS1678:</span><span class="sxs-lookup"><span data-stu-id="4e256-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
