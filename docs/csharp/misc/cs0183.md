# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="199b6-102">Предупреждение компилятора (уровень 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="199b6-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="199b6-103">Указанное выражение всегда имеет указанный тип ("тип")</span><span class="sxs-lookup"><span data-stu-id="199b6-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="199b6-104">Если условное выражение всегда равно **true**, вам не нужен условный оператор.</span><span class="sxs-lookup"><span data-stu-id="199b6-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="199b6-105">Это предупреждение возникает, когда вы пытаетесь проверить тип с помощью оператора **is** .</span><span class="sxs-lookup"><span data-stu-id="199b6-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="199b6-106">Если вычисление является типом значения, такая проверка не требуется.</span><span class="sxs-lookup"><span data-stu-id="199b6-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="199b6-107">При компиляции следующего примера будет выдано предупреждение CS0183:</span><span class="sxs-lookup"><span data-stu-id="199b6-107">The following sample generates CS0183:</span></span>  
  
```  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);   
   }  
}  
```
