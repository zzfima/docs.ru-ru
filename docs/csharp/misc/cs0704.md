## <a name="example"></a><span data-ttu-id="64422-106">Пример</span><span class="sxs-lookup"><span data-stu-id="64422-106">Example</span></span>  
 <span data-ttu-id="64422-107">Следующий пример приводит к возникновению ошибки CS0704:</span><span class="sxs-lookup"><span data-stu-id="64422-107">The following sample generates CS0704:</span></span>  
  
```csharp  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
