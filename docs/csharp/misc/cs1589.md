 <span data-ttu-id="e8880-104">Синтаксис (*фрагмент*) из [ \<включают >](../../csharp/programming-guide/xmldoc/include.md) тег, который ссылается на файл (`file`), был неправильным по указанной ***Причина***.</span><span class="sxs-lookup"><span data-stu-id="e8880-104">The syntax (*fragment*) of a [\<include>](../../csharp/programming-guide/xmldoc/include.md) tag, which referenced a file (`file`), was incorrect for the specified ***reason***.</span></span>

```csharp  
   {  
   }  
}  
```
