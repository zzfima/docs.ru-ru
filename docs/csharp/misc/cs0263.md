# <a name="compiler-error-cs0263"></a>Ошибка компилятора CS0263
Частичные объявления типа "тип" не должны указывать разные базовые классы  
  
 При определении типа в частичных объявлениях укажите одни и те же базовые типы в каждом частичном объявлении. Дополнительные сведения см. в разделе [Разделяемые классы и методы](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 В следующем примере возникает ошибка CS0263:  
  
```csharp  
// CS0263.cs  
// compile with: /target:library  
class B1  
{  
}  
  
class B2  
{  
}  
partial class C : B1  // CS0263 – is the base class B1 or B2?  
{  
}  
  
partial class C : B2  
{  
}  
```
