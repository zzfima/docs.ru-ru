# <a name="compiler-error-cs0564"></a><span data-ttu-id="abbfb-102">Ошибка компилятора CS0564</span><span class="sxs-lookup"><span data-stu-id="abbfb-102">Compiler Error CS0564</span></span>
<span data-ttu-id="abbfb-103">Тип первого операнда переопределенного оператора сдвига должен совпадать с вмещающим типом, а тип второго операнда должен быть int.</span><span class="sxs-lookup"><span data-stu-id="abbfb-103">The first operand of an overloaded shift operator must have the same type as the containing type, and the type of the second operand must be int</span></span>  
  
 <span data-ttu-id="abbfb-104">Предпринята попытка перегрузить оператор сдвига (<\< или >>) с неверно типизированными операндами.</span><span class="sxs-lookup"><span data-stu-id="abbfb-104">You attempted to overload a shift operator (<\< or >>) with incorrectly typed operands.</span></span> <span data-ttu-id="abbfb-105">Первый операнд должен быть типом, а второй операнд должен иметь тип `int`.</span><span class="sxs-lookup"><span data-stu-id="abbfb-105">The first operand must be the type and the second operand must be of the type `int`.</span></span>  
  
 <span data-ttu-id="abbfb-106">В следующем примере возникает ошибка CS0564:</span><span class="sxs-lookup"><span data-stu-id="abbfb-106">The following sample generates CS0564:</span></span>  
  
```csharp

class C  
{  
   public static int operator << (C c1, C c2) // CS0564

// public static int operator << (C c1, int c2)  
   {  
      return 0;  
   }  
   static void Main()   
   {  
   }  
}  
```
