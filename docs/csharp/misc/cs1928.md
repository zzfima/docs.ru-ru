# <a name="compiler-error-cs1928"></a><span data-ttu-id="361f6-102">Ошибка компилятора CS1928</span><span class="sxs-lookup"><span data-stu-id="361f6-102">Compiler Error CS1928</span></span>
<span data-ttu-id="361f6-103">Тип "тип" не содержит определение для метода "метод", и наилучшая перегрузка метода расширения "метод" имеет некоторые недопустимые аргументы.</span><span class="sxs-lookup"><span data-stu-id="361f6-103">'Type' does not contain a definition for 'method' and the best extension method overload 'method' has some invalid arguments.</span></span>  
  
 <span data-ttu-id="361f6-104">Эта ошибка возникает, когда компилятор не может найти член класса с именем метода, который был вызван.</span><span class="sxs-lookup"><span data-stu-id="361f6-104">This error is produced when the compiler cannot find a class member with the name of the method you have called.</span></span> <span data-ttu-id="361f6-105">Он может найти метод расширения с таким именем, но не с сигнатурой, которая соответствует типам, переданным в вызове метода.</span><span class="sxs-lookup"><span data-stu-id="361f6-105">It can find an extension method with that name, but not with a signature that matches the types you passed in with your method call.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="361f6-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="361f6-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="361f6-107">Передайте типы, которые соответствуют существующему методу расширения или методу класса.</span><span class="sxs-lookup"><span data-stu-id="361f6-107">Pass in types that match an existing extension method or class method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="361f6-108">Пример</span><span class="sxs-lookup"><span data-stu-id="361f6-108">Example</span></span>  
 <span data-ttu-id="361f6-109">Следующий код приводит к возникновению ошибки CS1928:</span><span class="sxs-lookup"><span data-stu-id="361f6-109">The following code generates CS1928:</span></span>  
  
```csharp  
// cs1928.cs  
class Test  
{  
    static void Main()  
    {  
        Test t = new Test();  
        t.M("hi"); // CS1928  
    }  
}  
static class Ext  
{  
    public static void M(this Test t, int i)  
    {  
    }  
}  
```  
  
 <span data-ttu-id="361f6-110">Эта ошибка часто сопровождается ошибкой CS1503: аргумент n: не удается преобразовать из "типА" в "типБ".</span><span class="sxs-lookup"><span data-stu-id="361f6-110">This error is often accompanied by CS1503: Argument 'n': cannot convert from 'typeA' to 'typeB'.</span></span>
