 <span data-ttu-id="79c40-104">Оператор [foreach](../language-reference/keywords/foreach-in.md) составлен неправильно.</span><span class="sxs-lookup"><span data-stu-id="79c40-104">A [foreach](../language-reference/keywords/foreach-in.md) statement was poorly formed.</span></span>

```csharp

{  
   public static void Main()  
   {  
      int[] myarray = new int[3] {1,2,3};  
  
      foreach (int in myarray)   // CS0230  
      // try the following line instead  
      // foreach (int x in myarray)

      }  
   }  
}  
```
