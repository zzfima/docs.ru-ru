# <a name="compiler-error-cs0573"></a>Ошибка компилятора CS0573
"объявление поля": в структуре не может содержаться инициализатор полей экземпляров  
  
 Инициализировать поле экземпляра [структуры](../language-reference/keywords/struct.md)невозможно. Поля типов значения инициализируются значениями по умолчанию, а поля ссылочного типа инициализируются значением `null`.  
  
## <a name="example"></a>Пример  
 Следующий пример приводит к возникновению ошибки CS0573:  
  
```csharp  
// CS0573.cs  
namespace x  
{  
    public class clx  
    {  
        public static void Main()  
        {  
        }  
    }  
  
    public struct cly  
    {  
        clx a = new clx();   // CS0573  
        // clx a;            // OK  
        int i = 7;           // CS0573  
        // int i;            // OK  
    }  
}  
```
