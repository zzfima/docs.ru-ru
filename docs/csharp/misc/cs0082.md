 <span data-ttu-id="25d87-104">Во время компиляции свойства преобразуются в методы с префиксами `get_` или `set_` , стоящими перед идентификаторами.</span><span class="sxs-lookup"><span data-stu-id="25d87-104">Properties at compile time are translated to methods with `get_` and/or `set_` in front of the identifier.</span></span> <span data-ttu-id="25d87-105">Если определить собственный метод, который конфликтует с именем преобразованного метода, возникает ошибка.</span><span class="sxs-lookup"><span data-stu-id="25d87-105">If you define your own method that conflicts with the method name, an error is generated.</span></span>  
  
## <a name="example"></a><span data-ttu-id="25d87-106">Пример</span><span class="sxs-lookup"><span data-stu-id="25d87-106">Example</span></span>  
 <span data-ttu-id="25d87-107">Следующий пример приводит к возникновению ошибки CS0082:</span><span class="sxs-lookup"><span data-stu-id="25d87-107">The following example generates CS0082:</span></span>  
  
```  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
