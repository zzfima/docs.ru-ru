## <a name="example"></a><span data-ttu-id="52ee5-107">Пример</span><span class="sxs-lookup"><span data-stu-id="52ee5-107">Example</span></span>  
 <span data-ttu-id="52ee5-108">Следующий код приводит к возникновению ошибки CS1952.</span><span class="sxs-lookup"><span data-stu-id="52ee5-108">The following code produces CS1952:</span></span>  
  
```  
// cs1952.cs  
using System;  
using System.Linq.Expressions;  
  
class Test  
{  
    public static int M(__arglist)  
    {  
        return 1;  
    }  
  
    static int Main()  
    {  
        Expression<Func<int, int>> f = x => Test.M(__arglist(x)); // CS1952  
        return 1;  
    }  
}  
```
