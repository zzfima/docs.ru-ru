# <a name="compiler-error-cs1623"></a><span data-ttu-id="e473c-102">Ошибка компилятора CS1623</span><span class="sxs-lookup"><span data-stu-id="e473c-102">Compiler Error CS1623</span></span>
<span data-ttu-id="e473c-103">Итераторы не могут иметь параметры с ключевым словом ref или out.</span><span class="sxs-lookup"><span data-stu-id="e473c-103">Iterators cannot have ref or out parameters</span></span>  
  
 <span data-ttu-id="e473c-104">Эта ошибка возникает, когда метод итератора принимает параметр `ref` или `out` .</span><span class="sxs-lookup"><span data-stu-id="e473c-104">This error occurs if an iterator method takes a `ref` or `out` parameter.</span></span> <span data-ttu-id="e473c-105">Чтобы избежать этой ошибки, удалите ключевое слово `ref` или `out` из сигнатуры метода.</span><span class="sxs-lookup"><span data-stu-id="e473c-105">To avoid this error, remove the `ref` or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e473c-106">Пример</span><span class="sxs-lookup"><span data-stu-id="e473c-106">Example</span></span>  
 <span data-ttu-id="e473c-107">Следующий пример приводит к возникновению ошибки CS1623:</span><span class="sxs-lookup"><span data-stu-id="e473c-107">The following sample generates CS1623:</span></span>  
  
```  
// CS1623.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {  
        yield return i;  
    }  
  
    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {  
        f = 0.0F;  
        yield return f;  
    }  
}  
```
