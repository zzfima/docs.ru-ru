# <a name="compiler-error-cs1555"></a>Ошибка компилятора CS1555
Не удалось найти "класс", указанный для метода Main  
  
 Класс был указан в параметре [-Main](../language-reference/compiler-options/main-compiler-option.md) компилятора, но имя класса не найдено в исходном коде.
