# <a name="compiler-error-cs1555"></a>Ошибка компилятора CS1555
Не удалось найти "класс", указанный для метода Main  
  
 Класс был указан для параметра компилятора [/main](../language-reference/compiler-options/main-compiler-option.md) , но имя класса не найдено в исходном коде.
