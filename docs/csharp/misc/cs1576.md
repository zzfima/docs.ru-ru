 <span data-ttu-id="25288-104">Компилятор обнаружил ошибку в значении, переданном в директиву [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) .</span><span class="sxs-lookup"><span data-stu-id="25288-104">The compiler detected an error with the value passed to the [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive.</span></span>

      #line "abc.sc"         // CS1576

   }
