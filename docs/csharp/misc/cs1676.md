# <a name="compiler-error-cs1676"></a><span data-ttu-id="c5a9d-102">Ошибка компилятора CS1676</span><span class="sxs-lookup"><span data-stu-id="c5a9d-102">Compiler Error CS1676</span></span>
<span data-ttu-id="c5a9d-103">Параметр "число" не должен объявляться с ключевым словом "ключевое_слово"</span><span class="sxs-lookup"><span data-stu-id="c5a9d-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="c5a9d-104">Эта ошибка возникает, когда модификатор типа параметра в анонимном методе отличается от используемого в объявлении делегата, к которому выполняется приведение метода.</span><span class="sxs-lookup"><span data-stu-id="c5a9d-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="c5a9d-105">В следующем примере возникает ошибка CS1676:</span><span class="sxs-lookup"><span data-stu-id="c5a9d-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
