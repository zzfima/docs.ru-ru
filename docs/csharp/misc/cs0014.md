# <a name="compiler-error-cs0014"></a>Ошибка компилятора CS0014
Не удалось найти требуемый файл "файл"  
  
 В системе отсутствует файл, необходимый для компилятора. Проверьте правильность пути. Если требуется системный файл [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] , то, возможно, необходимо исправить или полностью переустановить [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] .
