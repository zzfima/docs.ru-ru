# <a name="compiler-error-cs0014"></a>Ошибка компилятора CS0014
Не удалось найти требуемый файл "файл"  
  
 В системе отсутствует файл, необходимый для компилятора. Проверьте правильность пути. Если файл является файлом системы Visual Studio, может потребоваться восстановить установку либо удалить и полностью переустановить Visual Studio.
