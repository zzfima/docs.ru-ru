# <a name="compiler-error-cs0014"></a>Ошибка компилятора CS0014
Не удалось найти требуемый файл "файл"  
  
 В системе отсутствует файл, необходимый для компилятора. Проверьте правильность пути. Если файл [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] системный файл, то может потребоваться восстановление установки или else удаление и переустановка [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] полностью.
