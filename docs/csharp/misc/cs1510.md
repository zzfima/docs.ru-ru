## <a name="example"></a><span data-ttu-id="1b23d-106">Пример</span><span class="sxs-lookup"><span data-stu-id="1b23d-106">Example</span></span>  
 <span data-ttu-id="1b23d-107">Следующий пример приводит к возникновению ошибки CS1510:</span><span class="sxs-lookup"><span data-stu-id="1b23d-107">The following sample generates CS1510:</span></span>  
  
```csharp

public class C

   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
