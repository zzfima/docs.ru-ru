## <a name="example"></a><span data-ttu-id="032db-105">Пример</span><span class="sxs-lookup"><span data-stu-id="032db-105">Example</span></span>  
 <span data-ttu-id="032db-106">Следующий код приводит к возникновению ошибки CS0443:</span><span class="sxs-lookup"><span data-stu-id="032db-106">The following code generates CS0443.</span></span>  
  
```csharp  
// CS0443.cs   
using System;   
class MyClass   
{  
    public static void Main()      
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}   
    }  
}  
```
