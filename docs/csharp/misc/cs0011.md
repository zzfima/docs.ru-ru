## <a name="example"></a><span data-ttu-id="d5cd5-107">Пример</span><span class="sxs-lookup"><span data-stu-id="d5cd5-107">Example</span></span>

## <a name="example"></a><span data-ttu-id="d5cd5-108">Пример</span><span class="sxs-lookup"><span data-stu-id="d5cd5-108">Example</span></span>  
 <span data-ttu-id="d5cd5-109">Второй файл создает библиотеку DLL, которая определяет класс `C` , производный от класса `B` , созданного в предыдущем примере.</span><span class="sxs-lookup"><span data-stu-id="d5cd5-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

```csharp  
// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll

```  
  
## <a name="example"></a><span data-ttu-id="d5cd5-110">Пример</span><span class="sxs-lookup"><span data-stu-id="d5cd5-110">Example</span></span>
