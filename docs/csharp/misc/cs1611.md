# <a name="compiler-error-cs1611"></a>Ошибка компилятора CS1611
Параметр params не может объявляться как ref или out  
  
 Ключевые слова [в](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) или [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) не может использоваться с [params](../../csharp/language-reference/keywords/params.md) ключевое слово.  
  
 В следующем примере возникает ошибка CS1611:  
  
```csharp  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
