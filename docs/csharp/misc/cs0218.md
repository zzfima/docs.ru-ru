# <a name="compiler-error-cs0218"></a>Ошибка компилятора CS0218
Тип ("тип") должен содержать объявления оператора true и оператора false  
  
Если определяемый пользователем тип перегружает [& оператор](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) или [ &#124; оператор](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), он должен также определять [true и false](../language-reference/keywords/true-false-operators.md) операторы, чтобы сделать сокращенного вычисления [& & оператор](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) или [ &#124; &#124; оператор](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) определен.
  
 В следующем примере возникает ошибка CS0218:  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
