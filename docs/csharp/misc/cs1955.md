## <a name="example"></a><span data-ttu-id="ecb79-108">Пример</span><span class="sxs-lookup"><span data-stu-id="ecb79-108">Example</span></span>

<span data-ttu-id="ecb79-109">В следующем коде возникает ошибка CS1955 так, как происходит попытка вызвать поле и свойство с помощью [оператор вызова `()` ](../language-reference/operators/member-access-operators.md#invocation-operator-).</span><span class="sxs-lookup"><span data-stu-id="ecb79-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="ecb79-110">Поле или свойство вызвать невозможно.</span><span class="sxs-lookup"><span data-stu-id="ecb79-110">You cannot call a field or a property.</span></span> <span data-ttu-id="ecb79-111">Используйте [оператор доступа к члену `.` ](../language-reference/operators/member-access-operators.md#member-access-operator-) для доступа к хранящемуся в нем значению.</span><span class="sxs-lookup"><span data-stu-id="ecb79-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
