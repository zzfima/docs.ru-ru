<span data-ttu-id="c1f5b-103">Требуется постоянное значение.</span><span class="sxs-lookup"><span data-stu-id="c1f5b-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="c1f5b-104">Переменная найдена там, где ожидается константа.</span><span class="sxs-lookup"><span data-stu-id="c1f5b-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="c1f5b-105">Для получения дополнительной информации см. [переключения](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="c1f5b-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="c1f5b-106">Следующий пример приводит к возникновению ошибки CS0150.</span><span class="sxs-lookup"><span data-stu-id="c1f5b-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="c1f5b-107">Эта ошибка может также возникать, если размер массива указан с помощью значения переменной и инициализирован с помощью инициализатора массива.</span><span class="sxs-lookup"><span data-stu-id="c1f5b-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="c1f5b-108">Чтобы устранить эту ошибку, инициализируйте массив в отдельной инструкции или инструкциях.</span><span class="sxs-lookup"><span data-stu-id="c1f5b-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
