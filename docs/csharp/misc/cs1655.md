# <a name="compiler-error-cs1655"></a><span data-ttu-id="7b65b-102">Ошибка компилятора CS1655</span><span class="sxs-lookup"><span data-stu-id="7b65b-102">Compiler Error CS1655</span></span>
<span data-ttu-id="7b65b-103">Невозможно передать поля "переменная" как аргумент с ключевым словом ref или out, потому что это "тип_переменной_только_для_чтения"</span><span class="sxs-lookup"><span data-stu-id="7b65b-103">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="7b65b-104">Эта ошибка происходит при попытке передать в функцию в качестве аргумента ref или out член переменной [foreach](../language-reference/keywords/foreach-in.md) , [using](../language-reference/keywords/using-statement.md) или [fixed](../language-reference/keywords/fixed-statement.md) .</span><span class="sxs-lookup"><span data-stu-id="7b65b-104">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="7b65b-105">Так как в данных контекстах эти переменные доступны только для чтения, такая передача не разрешается.</span><span class="sxs-lookup"><span data-stu-id="7b65b-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="7b65b-106">При компиляции следующего примера возникнет ошибка CS1655:</span><span class="sxs-lookup"><span data-stu-id="7b65b-106">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
