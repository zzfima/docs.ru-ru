<span data-ttu-id="d0ed9-103">Тип "имя_типа" должен быть ссылочным типом для его использования в качестве параметра "имя_параметра" в универсальном типе или методе "идентификатор_универсального_объекта".</span><span class="sxs-lookup"><span data-stu-id="d0ed9-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="d0ed9-104">Эта ошибка возникает при передаче типа значения, такого как `struct` или `int` , в качестве параметра в универсальный тип или метод, имеющий ограничение ссылочного типа.</span><span class="sxs-lookup"><span data-stu-id="d0ed9-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>
