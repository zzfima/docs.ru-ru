# <a name="compiler-error-cs1527"></a><span data-ttu-id="d3f5f-102">Ошибка компилятора CS1527</span><span class="sxs-lookup"><span data-stu-id="d3f5f-102">Compiler Error CS1527</span></span>
<span data-ttu-id="d3f5f-103">Элементы, определенные в пространстве имен не может быть объявлено явно виде как частные, защищен, защищенные внутренние или защищенные закрытые.</span><span class="sxs-lookup"><span data-stu-id="d3f5f-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span> 
  
 <span data-ttu-id="d3f5f-104">Объявления типов в пространстве имен могут иметь только [открытый](../../csharp/language-reference/keywords/public.md) или [внутренний](../../csharp/language-reference/keywords/internal.md) доступ.</span><span class="sxs-lookup"><span data-stu-id="d3f5f-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="d3f5f-105">Если модификатор доступа не указан, по умолчанию используется **внутренний** тип.</span><span class="sxs-lookup"><span data-stu-id="d3f5f-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="d3f5f-106">Следующий пример приводит к возникновению ошибки CS1527:</span><span class="sxs-lookup"><span data-stu-id="d3f5f-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="d3f5f-107">В приведенном ниже примере возникает ошибка CS1527, так как если в коде программы пространство имен не объявлено явно, все объявления типов неявно помещаются в глобальное пространство имен.</span><span class="sxs-lookup"><span data-stu-id="d3f5f-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4{}  
private struct S1{}  
```
