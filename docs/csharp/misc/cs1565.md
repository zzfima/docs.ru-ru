# <a name="compiler-error-cs1565"></a>Ошибка компилятора CS1565
Заданы несовместимые параметры: файл ресурсов Win32; значок Win32.  
  
 Параметр компилятора [-win32res](../language-reference/compiler-options/win32res-compiler-option.md) и параметр компилятора [/win32ico](../language-reference/compiler-options/win32icon-compiler-option.md) нельзя использовать в той же компиляции.
