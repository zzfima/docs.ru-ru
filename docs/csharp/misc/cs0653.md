 <span data-ttu-id="15c6a-104">Пользовательский класс атрибутов [abstract](../language-reference/keywords/abstract.md) нельзя использовать в качестве атрибута.</span><span class="sxs-lookup"><span data-stu-id="15c6a-104">An [abstract](../language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>
