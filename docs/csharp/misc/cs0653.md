 <span data-ttu-id="6a721-104">Пользовательский класс атрибутов [abstract](../../csharp/language-reference/keywords/abstract.md) нельзя использовать в качестве атрибута.</span><span class="sxs-lookup"><span data-stu-id="6a721-104">An [abstract](../../csharp/language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>
