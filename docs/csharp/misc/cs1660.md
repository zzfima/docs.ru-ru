# <a name="compiler-error-cs1660"></a><span data-ttu-id="bcd6b-102">Ошибка компилятора CS1660</span><span class="sxs-lookup"><span data-stu-id="bcd6b-102">Compiler Error CS1660</span></span>
<span data-ttu-id="bcd6b-103">Невозможно преобразовать блок анонимного метода в тип "тип", поскольку он не является типом делегата</span><span class="sxs-lookup"><span data-stu-id="bcd6b-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="bcd6b-104">Эта ошибка возникает при попытке назначить блоку анонимного метода тип или преобразовать этот блок в тип, который не является типом делегата.</span><span class="sxs-lookup"><span data-stu-id="bcd6b-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="bcd6b-105">Следующий пример приводит к возникновению ошибки CS1660:</span><span class="sxs-lookup"><span data-stu-id="bcd6b-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
