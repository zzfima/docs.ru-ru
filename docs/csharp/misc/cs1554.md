 <span data-ttu-id="3ae7f-104">Тип возвращаемых данных для пользовательского [оператора](../../csharp/language-reference/keywords/operator.md) должен находиться перед ключевым словом operator.</span><span class="sxs-lookup"><span data-stu-id="3ae7f-104">The return type for a user-defined [operator](../../csharp/language-reference/keywords/operator.md) must appear before the keyword operator.</span></span>

   }  
}
