# <a name="compiler-error-cs0568"></a><span data-ttu-id="a8c50-102">Ошибка компилятора CS0568</span><span class="sxs-lookup"><span data-stu-id="a8c50-102">Compiler Error CS0568</span></span>
<span data-ttu-id="a8c50-103">Структуры не могут содержать явных конструкторов без параметров.</span><span class="sxs-lookup"><span data-stu-id="a8c50-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="a8c50-104">Каждый [структуры](../../csharp/language-reference/keywords/struct.md) уже имеет конструктор без параметров, который инициализирует объект с нулевым значением.</span><span class="sxs-lookup"><span data-stu-id="a8c50-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="a8c50-105">Таким образом, конструкторы, которые можно создать для структуры, должны принимать один или несколько параметров.</span><span class="sxs-lookup"><span data-stu-id="a8c50-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="a8c50-106">Следующий пример приводит к возникновению ошибки CS0568:</span><span class="sxs-lookup"><span data-stu-id="a8c50-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
