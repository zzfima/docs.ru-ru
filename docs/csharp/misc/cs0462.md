# <a name="compiler-error-cs0462"></a><span data-ttu-id="1155a-102">Ошибка компилятора CS0462</span><span class="sxs-lookup"><span data-stu-id="1155a-102">Compiler Error CS0462</span></span>
<span data-ttu-id="1155a-103">Наследуемые члены "член1" и "член2" имеют одинаковую сигнатуру в типе "тип", поэтому их нельзя переопределить</span><span class="sxs-lookup"><span data-stu-id="1155a-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="1155a-104">Эта ошибка возникает при введении универсальных типов.</span><span class="sxs-lookup"><span data-stu-id="1155a-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="1155a-105">Как правило, наличие в классе двух версий одного метода с одинаковыми сигнатурами невозможно.</span><span class="sxs-lookup"><span data-stu-id="1155a-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="1155a-106">Но в случае универсальных типов можно определить универсальный метод, который дублирует другой метод в случае создания экземпляра с определенным типом.</span><span class="sxs-lookup"><span data-stu-id="1155a-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1155a-107">Пример</span><span class="sxs-lookup"><span data-stu-id="1155a-107">Example</span></span>  
 <span data-ttu-id="1155a-108">При создании экземпляра типа `C<int>` создаются две версии метода `F` с одинаковыми сигнатурами, поэтому при выполнении переопределения в классе `D` компилятор не может решить, какой из них нужно переопределять.</span><span class="sxs-lookup"><span data-stu-id="1155a-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="1155a-109">Следующий пример приводит к возникновению ошибки CS0462:</span><span class="sxs-lookup"><span data-stu-id="1155a-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
