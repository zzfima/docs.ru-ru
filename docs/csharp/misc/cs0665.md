<span data-ttu-id="e3599-103">В условных выражениях присваивание всегда постоянное. Предполагалось использовать ==, а не = ?</span><span class="sxs-lookup"><span data-stu-id="e3599-103">Assignment in conditional expression is always constant; did you mean to use == instead of = ?</span></span>  
  
 <span data-ttu-id="e3599-104">Условное выражение, использующее [оператор =-](../../csharp/language-reference/operators/assignment-operator.md) , но не [оператор ==-](../../csharp/language-reference/operators/equality-comparison-operator.md).</span><span class="sxs-lookup"><span data-stu-id="e3599-104">A conditional expression used the [= operator](../../csharp/language-reference/operators/assignment-operator.md) and not the [== operator](../../csharp/language-reference/operators/equality-comparison-operator.md).</span></span>  
  
 <span data-ttu-id="e3599-105">В следующем примере возникает ошибка CS0665.</span><span class="sxs-lookup"><span data-stu-id="e3599-105">The following sample generates CS0665:</span></span>  
  
```csharp

{  
   public static void Main()  
   {  
      bool i = false;  
  
      if (i = true)   // CS0665  
      // try the following line instead  
      // if (i == true)  
      {  
      }  
  
      System.Console.WriteLine(i);  
   }  
}  
```
