# Предупреждение компилятора (уровень&#160;1) CS1590
Недопустимый элемент включения для XML — отсутствует атрибут file  
  
 Переданный тегу [\<include\>](../../csharp/programming-guide/xmldoc/include.md) атрибут path или doc отсутствует или является неполным.  
  
 Следующий пример приводит к возникновению предупреждения CS1590:  
  
```  
// CS1590.cs // compile with: /W:1 /doc:x.xml /// <include path='MyDocs/MyMembers[@name="test"]/*' />   // CS1590 // try the following line instead // /// <include file='x.doc' path='MyDocs/MyMembers[@name="test"]/*' /> class Test { public static void Main() { } }  
```
