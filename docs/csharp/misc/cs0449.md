# <a name="compiler-error-cs0449"></a><span data-ttu-id="44b06-102">Ошибка компилятора CS0449</span><span class="sxs-lookup"><span data-stu-id="44b06-102">Compiler Error CS0449</span></span>
<span data-ttu-id="44b06-103">Все другие ограничения должны следовать после ограничения "class" или "struct".</span><span class="sxs-lookup"><span data-stu-id="44b06-103">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="44b06-104">Ограничения параметра общего типа или метода должны применяться в следующем порядке: сначала `class` или `struct` (если таковые имеются), затем все ограничения интерфейса, и наконец — все ограничения конструктора.</span><span class="sxs-lookup"><span data-stu-id="44b06-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="44b06-105">Эта ошибка возникает в том случае, когда ограничение `class` или `struct` следует не первым.</span><span class="sxs-lookup"><span data-stu-id="44b06-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="44b06-106">Чтобы устранить эту ошибку, измените порядок предложений ограничений.</span><span class="sxs-lookup"><span data-stu-id="44b06-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="44b06-107">Пример</span><span class="sxs-lookup"><span data-stu-id="44b06-107">Example</span></span>  
 <span data-ttu-id="44b06-108">Следующий пример приводит к возникновению ошибки CS0449:</span><span class="sxs-lookup"><span data-stu-id="44b06-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
