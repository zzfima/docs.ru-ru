## <a name="example"></a><span data-ttu-id="37445-105">Пример</span><span class="sxs-lookup"><span data-stu-id="37445-105">Example</span></span>  
 <span data-ttu-id="37445-106">В следующем примере возникает ошибка CS0401.</span><span class="sxs-lookup"><span data-stu-id="37445-106">The following sample generates CS0401.</span></span>  
  
```csharp  
// CS0401.cs  
// compile with: /target:library  
using System;  
 class C<T> where T : new(), IDisposable {}  // CS0401

{  
   static void F<U>() where U : new(), IDisposable{}   // CS0401  
}  
```
