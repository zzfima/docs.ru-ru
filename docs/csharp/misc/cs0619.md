# <a name="compiler-error-cs0619"></a>Ошибка компилятора CS0619
"член" устарел: "текст"  
  
 Член класса был помечен атрибутом [устарело] ((.. /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) атрибута, таким образом, будет выведено сообщение об ошибке, при ссылке на член класса.  
  
 Следующий пример приводит к возникновению ошибки CS0619:  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
