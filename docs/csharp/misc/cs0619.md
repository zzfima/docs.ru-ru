# <a name="compiler-error-cs0619"></a>Ошибка компилятора CS0619
"член" устарел: "текст"  
  
 Член класса, помеченного с [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) атрибута, таким образом, будет выведено сообщение об ошибке, если имеется ссылка на член класса.  
  
 Следующий пример приводит к возникновению ошибки CS0619:  
  
```  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
