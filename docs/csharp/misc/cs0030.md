 <span data-ttu-id="b4a7a-104">Необходимо предоставить подпрограммы преобразования для поддержки перегрузки определенных операторов.</span><span class="sxs-lookup"><span data-stu-id="b4a7a-104">You must provide conversion routines to support certain operator overloads.</span></span> <span data-ttu-id="b4a7a-105">Дополнительные сведения см. в разделе [Операторы преобразования](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="b4a7a-105">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="b4a7a-106">Следующий пример приводит к возникновению ошибки CS0030:</span><span class="sxs-lookup"><span data-stu-id="b4a7a-106">The following sample generates CS0030:</span></span>  
  
```  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
