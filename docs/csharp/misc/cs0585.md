# <a name="compiler-error-cs0585"></a>Ошибка компилятора CS0585
Внутренняя ошибка компилятора: этап "этап"  
  
 Возможно, ошибка возникает из-за сбоя компилятора при попытке проанализировать неправильный синтаксис. Если это не так, попробуйте [обращайтесь к нам](https://docs.microsoft.com/visualstudio/ide/talk-to-us).
