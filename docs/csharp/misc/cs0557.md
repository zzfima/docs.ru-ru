 <span data-ttu-id="48957-104">Повторяющиеся процедуры преобразования не разрешены в классе.</span><span class="sxs-lookup"><span data-stu-id="48957-104">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="48957-105">В следующем примере возникает ошибка CS0557:</span><span class="sxs-lookup"><span data-stu-id="48957-105">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
