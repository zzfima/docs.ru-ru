# <a name="compiler-error-cs0177"></a><span data-ttu-id="88ea8-102">Ошибка компилятора CS0177</span><span class="sxs-lookup"><span data-stu-id="88ea8-102">Compiler Error CS0177</span></span>
<span data-ttu-id="88ea8-103">До передачи управления из текущего метода выходному параметру "параметр" должно быть присвоено значение</span><span class="sxs-lookup"><span data-stu-id="88ea8-103">The out parameter 'parameter' must be assigned to before control leaves the current method</span></span>  
  
 <span data-ttu-id="88ea8-104">Параметру, помеченному ключевым словом [out](../language-reference/keywords/out-parameter-modifier.md) , не было присвоено значение в теле метода.</span><span class="sxs-lookup"><span data-stu-id="88ea8-104">A parameter marked with the [out](../language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="88ea8-105">Дополнительные сведения см. в разделе [Передача параметров](../programming-guide/classes-and-structs/passing-parameters.md) .</span><span class="sxs-lookup"><span data-stu-id="88ea8-105">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md)</span></span>  
  
 <span data-ttu-id="88ea8-106">Следующий пример приводит к возникновению ошибки CS0177:</span><span class="sxs-lookup"><span data-stu-id="88ea8-106">The following sample generates CS0177:</span></span>  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
