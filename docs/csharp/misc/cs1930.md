## <a name="example"></a><span data-ttu-id="c8acd-108">Пример</span><span class="sxs-lookup"><span data-stu-id="c8acd-108">Example</span></span>  
 <span data-ttu-id="c8acd-109">Приведенный ниже пример приводит к возникновению ошибки CS1930, так как идентификатор `num` используется для переменной диапазона в предложении `from` и для переменной диапазона, представленной предложением `let` .</span><span class="sxs-lookup"><span data-stu-id="c8acd-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;
    }  
}  
```
