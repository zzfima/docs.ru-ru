# <a name="compiler-error-cs0244"></a>Ошибка компилятора CS0244
Ни "is", ни "as" недопустимы в типах указателей.  
  
 Операторы [-](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) и [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) недопустимы для использования в типах указателей. Дополнительные сведения см. в разделе [Небезопасный код и указатели](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 В следующем примере возникает ошибка CS0244:  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
