# <a name="compiler-error-cs0244"></a><span data-ttu-id="b3bfe-102">Ошибка компилятора CS0244</span><span class="sxs-lookup"><span data-stu-id="b3bfe-102">Compiler Error CS0244</span></span>
<span data-ttu-id="b3bfe-103">Ни "is", ни "as" недопустимы в типах указателей.</span><span class="sxs-lookup"><span data-stu-id="b3bfe-103">Neither 'is' nor 'as' is valid on pointer types</span></span>  
  
 <span data-ttu-id="b3bfe-104">Ключевые слова [is](../../csharp/language-reference/keywords/is.md) и [as](../../csharp/language-reference/keywords/as.md) не разрешено использовать в типах указателей.</span><span class="sxs-lookup"><span data-stu-id="b3bfe-104">The [is](../../csharp/language-reference/keywords/is.md) and [as](../../csharp/language-reference/keywords/as.md) keywords are not valid for use on pointer types.</span></span> <span data-ttu-id="b3bfe-105">Дополнительные сведения см. в разделе [Небезопасный код и указатели](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="b3bfe-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="b3bfe-106">В следующем примере возникает ошибка CS0244:</span><span class="sxs-lookup"><span data-stu-id="b3bfe-106">The following sample generates CS0244:</span></span>  
  
```  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
