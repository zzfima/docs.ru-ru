# <a name="compiler-error-cs0601"></a><span data-ttu-id="d23cd-102">Ошибка компилятора CS0601</span><span class="sxs-lookup"><span data-stu-id="d23cd-102">Compiler Error CS0601</span></span>
<span data-ttu-id="d23cd-103">Атрибут DllImport должен быть указан для метода, который помечен как "static" и "extern".</span><span class="sxs-lookup"><span data-stu-id="d23cd-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="d23cd-104">Атрибут `DllImport` был использован для метода, который не имеет правильные ключевые слова доступа.</span><span class="sxs-lookup"><span data-stu-id="d23cd-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="d23cd-105">В следующем примере возникает ошибка CS0601:</span><span class="sxs-lookup"><span data-stu-id="d23cd-105">The following sample generates CS0601:</span></span>  
  
```  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
