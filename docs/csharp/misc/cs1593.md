# <a name="compiler-error-cs1593"></a><span data-ttu-id="c13bd-102">Ошибка компилятора CS1593</span><span class="sxs-lookup"><span data-stu-id="c13bd-102">Compiler Error CS1593</span></span>
<span data-ttu-id="c13bd-103">Делегат "делегат" не принимает "число" аргументов</span><span class="sxs-lookup"><span data-stu-id="c13bd-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="c13bd-104">Число аргументов, переданных при вызове [делегата](../../csharp/language-reference/keywords/delegate.md) , не совпадает с числом параметров, указанных в объявлении делегата.</span><span class="sxs-lookup"><span data-stu-id="c13bd-104">The number of arguments passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="c13bd-105">Следующий пример приводит к возникновению ошибки CS1593:</span><span class="sxs-lookup"><span data-stu-id="c13bd-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
