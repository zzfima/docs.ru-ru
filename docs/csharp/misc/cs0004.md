# <a name="compiler-error-cs0004"></a>Ошибка компилятора CS0004
Предупреждение, обрабатываемое как ошибка  
  
 Вы выполнили компиляцию с параметром компиляции `/warnaserror` , который приводит к отображению предупреждений компилятора как ошибок.
