# <a name="compiler-error-cs0553"></a><span data-ttu-id="b03fa-102">Ошибка компилятора CS0553</span><span class="sxs-lookup"><span data-stu-id="b03fa-102">Compiler Error CS0553</span></span>
<span data-ttu-id="b03fa-103">"подпрограмма преобразования": определенное пользователем преобразование в базовый класс/из базового класса</span><span class="sxs-lookup"><span data-stu-id="b03fa-103">'conversion routine' : user defined conversion to/from base class</span></span>  
  
 <span data-ttu-id="b03fa-104">Пользовательские преобразования в значения базового класса не допускаются; в операторе подобного типа нет необходимости.</span><span class="sxs-lookup"><span data-stu-id="b03fa-104">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="b03fa-105">Следующий пример приводит к возникновению ошибки CS0553:</span><span class="sxs-lookup"><span data-stu-id="b03fa-105">The following sample generates CS0553:</span></span>  
  
```  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
