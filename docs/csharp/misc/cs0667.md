# <a name="compiler-error-cs0667"></a>Ошибка компилятора CS0667
Функция "недопустимая функция" является нерекомендуемой Вместо нее используйте функцию "допустимая функция".  
  
 Вы пытаетесь использовать устаревшую функцию. Обновите код, чтобы использовать вместо нее допустимую функцию.
