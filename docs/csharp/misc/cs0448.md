# <a name="compiler-error-cs0448"></a><span data-ttu-id="fdf3c-102">Ошибка компилятора CS0448</span><span class="sxs-lookup"><span data-stu-id="fdf3c-102">Compiler Error CS0448</span></span>
<span data-ttu-id="fdf3c-103">Возвращаемый тип оператора ++ или -- должен быть вмещающего типа или типа, производного от вмещающего типа</span><span class="sxs-lookup"><span data-stu-id="fdf3c-103">The return type for ++ or -- operator must be the containing type or derived from the containing type</span></span>  
  
 <span data-ttu-id="fdf3c-104">При переопределении операторов `++` или `--` они должны возвращать тот же тип, что и содержащий их тип, или возвращать тип, который является производным от содержащего типа.</span><span class="sxs-lookup"><span data-stu-id="fdf3c-104">When you override the `++` or `--` operators, they must return the same type as the containing type, or return a type that is derived from the containing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fdf3c-105">Пример</span><span class="sxs-lookup"><span data-stu-id="fdf3c-105">Example</span></span>  
 <span data-ttu-id="fdf3c-106">В следующем примере возникает ошибка CS0448.</span><span class="sxs-lookup"><span data-stu-id="fdf3c-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="fdf3c-107">Пример</span><span class="sxs-lookup"><span data-stu-id="fdf3c-107">Example</span></span>  
 <span data-ttu-id="fdf3c-108">В следующем примере возникает ошибка CS0448.</span><span class="sxs-lookup"><span data-stu-id="fdf3c-108">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
