# <a name="compiler-error-cs0231"></a><span data-ttu-id="65a5e-102">Ошибка компилятора CS0231</span><span class="sxs-lookup"><span data-stu-id="65a5e-102">Compiler Error CS0231</span></span>
<span data-ttu-id="65a5e-103">Параметр params должен быть указан последним в списке формальных параметров.</span><span class="sxs-lookup"><span data-stu-id="65a5e-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="65a5e-104">Параметр [params](../../csharp/language-reference/keywords/params.md) поддерживает переменное число аргументов и должен указываться после всех остальных параметров.</span><span class="sxs-lookup"><span data-stu-id="65a5e-104">The [params](../../csharp/language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="65a5e-105">Дополнительные сведения см. в статье [Методы](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="65a5e-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="65a5e-106">Следующий пример приводит к возникновению ошибки CS0231:</span><span class="sxs-lookup"><span data-stu-id="65a5e-106">The following sample generates CS0231:</span></span>  
  
```  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}   
  
   static void Main()   
   {  
   }  
}  
```
