# <a name="compiler-error-cs0572"></a><span data-ttu-id="03447-102">Ошибка компилятора CS0572</span><span class="sxs-lookup"><span data-stu-id="03447-102">Compiler Error CS0572</span></span>
<span data-ttu-id="03447-103">"тип": невозможно сослаться на тип через выражение; попытайтесь использовать "путь_к_типу"</span><span class="sxs-lookup"><span data-stu-id="03447-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="03447-104">Предпринята попытка доступа к члену класса посредством идентификатора, который не разрешен в этой ситуации.</span><span class="sxs-lookup"><span data-stu-id="03447-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="03447-105">В следующем примере возникает ошибка CS0572:</span><span class="sxs-lookup"><span data-stu-id="03447-105">The following sample generates CS0572:</span></span>  
  
```  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
