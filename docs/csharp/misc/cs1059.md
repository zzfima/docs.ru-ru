# <a name="compiler-error-cs1059"></a><span data-ttu-id="5e692-102">Ошибка компилятора CS1059</span><span class="sxs-lookup"><span data-stu-id="5e692-102">Compiler Error CS1059</span></span>
<span data-ttu-id="5e692-103">Операндом оператора инкремента или декремента должна быть переменная, свойство или индексатор.</span><span class="sxs-lookup"><span data-stu-id="5e692-103">The operand of an increment or decrement operator must be a variable, property or indexer.</span></span>  
  
 <span data-ttu-id="5e692-104">Эта ошибка возникает при попытке увеличения или уменьшения постоянного значения.</span><span class="sxs-lookup"><span data-stu-id="5e692-104">This error is raised when you try to increment or decrement a constant value.</span></span> <span data-ttu-id="5e692-105">Также она возникает при попытке увеличения значения такого выражения, как `(a+b)++`.</span><span class="sxs-lookup"><span data-stu-id="5e692-105">It can also occur if you try to increment an expression such as `(a+b)++`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5e692-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="5e692-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="5e692-110">Пример</span><span class="sxs-lookup"><span data-stu-id="5e692-110">Example</span></span>  
 <span data-ttu-id="5e692-111">Приведенный ниже пример приводит к возникновению ошибки CS1059, так как `i` является не переменной, а константой, а `E` имеет тип `Enum` , элементы которого также представляют собой постоянные значения.</span><span class="sxs-lookup"><span data-stu-id="5e692-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
