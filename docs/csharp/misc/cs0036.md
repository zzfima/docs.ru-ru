# <a name="compiler-error-cs0036"></a><span data-ttu-id="65e8c-102">Ошибка компилятора CS0036</span><span class="sxs-lookup"><span data-stu-id="65e8c-102">Compiler Error CS0036</span></span>
<span data-ttu-id="65e8c-103">Выходной параметр не может иметь атрибут "[In]"</span><span class="sxs-lookup"><span data-stu-id="65e8c-103">An out parameter cannot have the '[In]' attribute</span></span>  
  
 <span data-ttu-id="65e8c-104">В настоящее время атрибут **In** запрещен в параметре [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) .</span><span class="sxs-lookup"><span data-stu-id="65e8c-104">Currently, the **In** attribute is not permitted on an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span>  
  
 <span data-ttu-id="65e8c-105">Следующий пример приводит к возникновению ошибки CS0036:</span><span class="sxs-lookup"><span data-stu-id="65e8c-105">The following sample generates CS0036:</span></span>  
  
```csharp  
// CS0036.cs  
  
using System;  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   public static void TestOut([In] out char TestChar)   // CS0036  
   // try the following line instead  
   // public static void TestOut(out char TestChar)  
   {  
      TestChar = 'b';  
      Console.WriteLine(TestChar);  
   }  
  
   public static void Main()  
   {  
      char i;           //variable to receive the value  
      TestOut(out i);   // the arg must be passed as out  
      Console.WriteLine(i);  
   }  
}  
```
