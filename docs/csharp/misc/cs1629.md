 <span data-ttu-id="08340-104">Спецификация языка C# не позволяет небезопасный код в итераторах.</span><span class="sxs-lookup"><span data-stu-id="08340-104">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="08340-105">В следующем примере возникает ошибка CS1629:</span><span class="sxs-lookup"><span data-stu-id="08340-105">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
