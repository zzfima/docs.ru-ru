# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="e509a-102">Предупреждение компилятора (уровень 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="e509a-102">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="e509a-103">"член": новый защищенный член объявлен в запечатанном классе</span><span class="sxs-lookup"><span data-stu-id="e509a-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="e509a-104">[Запечатанный](../../csharp/language-reference/keywords/sealed.md) класс не может вводить [защищенный](../../csharp/language-reference/keywords/protected.md) член, так как никакому другому классу не удастся наследовать от класса с атрибутом `sealed` и использовать член с атрибутом `protected` .</span><span class="sxs-lookup"><span data-stu-id="e509a-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="e509a-105">При компиляции следующего примера будет выдано предупреждение CS0628:</span><span class="sxs-lookup"><span data-stu-id="e509a-105">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
