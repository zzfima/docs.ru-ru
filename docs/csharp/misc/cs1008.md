# <a name="compiler-error-cs1008"></a>Ошибка компилятора CS1008
Требуется тип byte, sbyte, short, ushort, int, uint, long или ulong.  
  
 Некоторые типы данных, например [перечисления](../language-reference/keywords/enum.md), могут быть объявлены только для хранения данных определенных типов.  
  
 В следующем примере возникает ошибка CS1008:  
  
```csharp  
// CS1008.cs  
abstract public class clx  
{  
   enum splitch : char   // CS1008, char not valid type for enums  
   {  
      x, y, z  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
