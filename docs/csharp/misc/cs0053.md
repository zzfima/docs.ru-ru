# <a name="compiler-error-cs0053"></a>Ошибка компилятора CS0053
Несогласованность по доступности: свойство типа "тип" менее доступно, чем свойство "свойство"  
  
 Открытая конструкция должна возвращать общедоступный объект. Дополнительные сведения см. в статье [Модификаторы доступа](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 В следующем примере возникает ошибка CS0053:  
  
```csharp  
// CS0053.cs  
class MyClass //defaults to private accessibility  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class MyClass2  
{  
   public MyClass myProperty   // CS0053  
   {  
      get  
      {  
         return new MyClass();  
      }  
      set  
      {  
      }  
   }  
}  
  
public class MyClass3  
{  
   public static void Main()  
   {  
   }  
}  
```
