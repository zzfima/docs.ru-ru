# <a name="compiler-error-cs1511"></a><span data-ttu-id="eabd2-102">Ошибка компилятора CS1511</span><span class="sxs-lookup"><span data-stu-id="eabd2-102">Compiler Error CS1511</span></span>
<span data-ttu-id="eabd2-103">Ключевое слово "base" неприменимо в статическом методе.</span><span class="sxs-lookup"><span data-stu-id="eabd2-103">Keyword 'base' is not available in a static method</span></span>  
  
 <span data-ttu-id="eabd2-104">Ключевое слово [base](../../csharp/language-reference/keywords/base.md) использовалось в [статическом](../../csharp/language-reference/keywords/static.md) методе.</span><span class="sxs-lookup"><span data-stu-id="eabd2-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="eabd2-105">`base` может вызываться только в конструкторе экземпляра, методе экземпляра или методе доступа экземпляра.</span><span class="sxs-lookup"><span data-stu-id="eabd2-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="eabd2-106">Пример</span><span class="sxs-lookup"><span data-stu-id="eabd2-106">Example</span></span>  
 <span data-ttu-id="eabd2-107">Следующий пример приводит к возникновению ошибки CS1511:</span><span class="sxs-lookup"><span data-stu-id="eabd2-107">The following sample generates CS1511.</span></span>  
  
```  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
