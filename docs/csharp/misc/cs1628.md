# <a name="compiler-error-cs1628"></a><span data-ttu-id="d8fa1-102">Ошибка компилятора CS1628</span><span class="sxs-lookup"><span data-stu-id="d8fa1-102">Compiler Error CS1628</span></span>
<span data-ttu-id="d8fa1-103">Нельзя использовать параметр "параметр" с ключевым словом ref или out внутри анонимного метода, лямбда-выражения или выражения запроса.</span><span class="sxs-lookup"><span data-stu-id="d8fa1-103">Cannot use ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="d8fa1-104">Эта ошибка возникает при использовании параметра ref или out внутри блока анонимного метода.</span><span class="sxs-lookup"><span data-stu-id="d8fa1-104">This error occurs if you use a ref or out parameter inside an anonymous method block.</span></span> <span data-ttu-id="d8fa1-105">Чтобы избежать этой ошибки, используйте локальную переменную или другую конструкцию.</span><span class="sxs-lookup"><span data-stu-id="d8fa1-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="d8fa1-106">Следующий пример приводит к возникновению ошибки CS1628:</span><span class="sxs-lookup"><span data-stu-id="d8fa1-106">The following sample generates CS1628:</span></span>  
  
```  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
