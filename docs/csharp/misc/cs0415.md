<span data-ttu-id="e6d34-103">Атрибут IndexerName применим только в индексаторе, который не является явным объявлением члена интерфейса</span><span class="sxs-lookup"><span data-stu-id="e6d34-103">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="e6d34-104">Эта ошибка возникает, если атрибут IndexerName используется в индексаторе, который является явной реализацией интерфейса.</span><span class="sxs-lookup"><span data-stu-id="e6d34-104">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="e6d34-105">Этой ошибки можно избежать, удалив имя интерфейса из объявления индексатора, если это возможно.</span><span class="sxs-lookup"><span data-stu-id="e6d34-105">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="e6d34-106">Дополнительные сведения см. в разделе [Класс IndexerNameAttribute](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="e6d34-106">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="e6d34-107">В следующем примере возникает ошибка CS0415:</span><span class="sxs-lookup"><span data-stu-id="e6d34-107">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
