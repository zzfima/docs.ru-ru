# <a name="compiler-error-cs0023"></a>Ошибка компилятора CS0023
Оператор "оператор" не может применяться к операнду типа "тип"  
  
 Предпринята попытка применить оператор к переменной, тип которой не предназначен для работы с этим оператором. Дополнительные сведения см. в разделе [типы](../../csharp/programming-guide/types/index.md) и [ C# операторы](../../csharp/language-reference/operators/index.md).  
  
 При компиляции следующего примера возникнет ошибка CS0023:  
  
```csharp  
// CS0023.cs  
namespace x  
{  
   public class a  
   {  
      public static void Main()  
      {  
         string s = "hello";  
         s = -s;   // CS0023, minus operator not allowed on strings  
      }  
   }  
}  
```
