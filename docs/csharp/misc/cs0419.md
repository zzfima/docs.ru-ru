# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="06c44-102">Предупреждение компилятора (уровень 3) CS0419</span><span class="sxs-lookup"><span data-stu-id="06c44-102">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="06c44-103">Неоднозначная ссылка в атрибуте cref: "Имя метода 1".</span><span class="sxs-lookup"><span data-stu-id="06c44-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="06c44-104">Предполагается "Имя метода 2", но может также соответствовать другим перегрузкам, включая "Имя метода 3".</span><span class="sxs-lookup"><span data-stu-id="06c44-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="06c44-105">Не удалось разрешить ссылку в комментарии документации XML в коде.</span><span class="sxs-lookup"><span data-stu-id="06c44-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="06c44-106">Это может происходить, если метод перегружен или если найдено два разных идентификатора с одинаковым именем.</span><span class="sxs-lookup"><span data-stu-id="06c44-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="06c44-107">Чтобы разрешить предупреждение, используйте полное имя для устранения неоднозначности ссылки или заключите определенную перегрузку в скобки.</span><span class="sxs-lookup"><span data-stu-id="06c44-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="06c44-108">Следующий пример приводит к возникновению предупреждения CS0419:</span><span class="sxs-lookup"><span data-stu-id="06c44-108">The following sample generates CS0419.</span></span>  
  
```  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
