# <a name="compiler-error-cs0220"></a><span data-ttu-id="04d20-102">Ошибка компилятора CS0220</span><span class="sxs-lookup"><span data-stu-id="04d20-102">Compiler Error CS0220</span></span>
<span data-ttu-id="04d20-103">Переполнение при выполнении операции во время компиляции в режиме с проверкой</span><span class="sxs-lookup"><span data-stu-id="04d20-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="04d20-104">Операция была обнаружена по [checked](../../csharp/language-reference/keywords/checked.md), который используется по умолчанию, и привела к потере данных.</span><span class="sxs-lookup"><span data-stu-id="04d20-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="04d20-105">Исправьте входные значения для назначения или используйте [unchecked](../../csharp/language-reference/keywords/unchecked.md) , чтобы устранить эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="04d20-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="04d20-106">Дополнительные сведения см. в разделе [Checked и Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="04d20-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="04d20-107">В следующем примере возникает ошибка CS0220:</span><span class="sxs-lookup"><span data-stu-id="04d20-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
