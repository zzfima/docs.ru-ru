# Предупреждение компилятора (уровень&#160;2) CS0162
Обнаружен недостижимый код  
  
 Компилятор обнаружил код, который никогда не будет выполняться.  
  
 Следующий пример приводит к возникновению предупреждения CS0162:  
  
```  
// CS0162.cs // compile with: /W:2 public class A { public static void Main() { goto lab1; { // The following statements cannot be reached: int i = 9;   // CS0162 i++; } lab1: { } } }  
```
