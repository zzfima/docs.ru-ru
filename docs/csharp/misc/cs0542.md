# <a name="compiler-error-cs0542"></a><span data-ttu-id="a27c3-102">Ошибка компилятора CS0542</span><span class="sxs-lookup"><span data-stu-id="a27c3-102">Compiler Error CS0542</span></span>
<span data-ttu-id="a27c3-103">"определенный пользователем тип": имена членов не могут совпадать с именем типа, в который они входят</span><span class="sxs-lookup"><span data-stu-id="a27c3-103">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="a27c3-104">Члены класса или структуры не могут иметь то же имя, что у класса или структуры, если член не является конструктором.</span><span class="sxs-lookup"><span data-stu-id="a27c3-104">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="a27c3-105">Следующий пример приводит к возникновению ошибки CS0542:</span><span class="sxs-lookup"><span data-stu-id="a27c3-105">The following sample generates CS0542:</span></span>  
  
```csharp  
// CS0542.cs  
class C  
{  
    public int C;  
}  
```  
  
 <span data-ttu-id="a27c3-106">Эта ошибка может возникнуть, если вы случайно задали для конструктора тип возвращаемого значения, что фактически превращает его в обычный метод.</span><span class="sxs-lookup"><span data-stu-id="a27c3-106">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="a27c3-107">Следующий пример приводит к возникновению ошибки CS0542, так как `F` является методом, а не конструктором, поскольку имеет тип возвращаемого значения:</span><span class="sxs-lookup"><span data-stu-id="a27c3-107">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp  
// CS0542.cs  
class F  
{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="a27c3-108">Эта ошибка может возникнуть, если ваш класс называется Item и содержит индексатор, объявленный как `this`.</span><span class="sxs-lookup"><span data-stu-id="a27c3-108">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="a27c3-109">Индексатору по умолчанию дается имя Item в порожденном коде, что создает конфликт.</span><span class="sxs-lookup"><span data-stu-id="a27c3-109">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>  
  
```csharp

{  
   public int this[int i]  // CS0542  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```
