# <a name="compiler-warning-level-3-cs0660"></a><span data-ttu-id="e94ee-102">Предупреждение компилятора (уровень 3) CS0660</span><span class="sxs-lookup"><span data-stu-id="e94ee-102">Compiler Warning (level 3) CS0660</span></span>

<span data-ttu-id="e94ee-104">Компилятор обнаружил определяемый пользователем оператор равенства или неравенства, но не имеет переопределения для метода <xref:System.Object.Equals%2A?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="e94ee-104">The compiler detected the user-defined equality or inequality operator, but no override for the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="e94ee-105">Определяемый пользователем оператор равенства или неравенства подразумевает также необходимость переопределения метода <xref:System.Object.Equals%2A>.</span><span class="sxs-lookup"><span data-stu-id="e94ee-105">A user-defined equality or inequality operator implies that you also want to override the <xref:System.Object.Equals%2A> method.</span></span> <span data-ttu-id="e94ee-106">Дополнительные сведения см. [в разделе Определение равенства значений для типа](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span><span class="sxs-lookup"><span data-stu-id="e94ee-106">For more information, see [How to define value equality for a type](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span></span>
  
<span data-ttu-id="e94ee-107">В следующем примере возникает ошибка CS0660:</span><span class="sxs-lookup"><span data-stu-id="e94ee-107">The following sample generates CS0660:</span></span>  

```csharp
// CS0660.cs

class Test   // CS0660  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   // uncomment the Equals function to resolve  
   // public override bool Equals(object o)  
   // {  
   //    return true;  
   // }  
  
   public override int GetHashCode()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
