 <span data-ttu-id="bd7f9-104">При объявлении [свойства](../../csharp/programming-guide/classes-and-structs/using-properties.md)также необходимо объявить его методы доступа.</span><span class="sxs-lookup"><span data-stu-id="bd7f9-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="bd7f9-105">Однако свойство не может иметь несколько методов доступа `get` или несколько методов доступа `set` .</span><span class="sxs-lookup"><span data-stu-id="bd7f9-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bd7f9-106">Пример</span><span class="sxs-lookup"><span data-stu-id="bd7f9-106">Example</span></span>

```csharp  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method

            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
