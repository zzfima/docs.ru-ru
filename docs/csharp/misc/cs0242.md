# <a name="compiler-error-cs0242"></a>Ошибка компилятора CS0242
При указателях на объекты неизвестного типа данная операция не определена.  
  
 Увеличение указателя типа void не разрешено. Дополнительные сведения см. в разделе [Небезопасный код и указатели](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 В следующем примере возникает ошибка CS0242:  
  
```csharp  
// CS0242.cs  
// compile with: /unsafe  
class TestClass  
{  
   public unsafe void Test()  
   {  
      void * p = null;  
      p++;   // CS0242, incrementing a void pointer not allowed  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
