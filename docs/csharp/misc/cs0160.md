# <a name="compiler-error-cs0160"></a><span data-ttu-id="4b2f6-102">Ошибка компилятора CS0160</span><span class="sxs-lookup"><span data-stu-id="4b2f6-102">Compiler Error CS0160</span></span>
<span data-ttu-id="4b2f6-103">Предыдущее предложение catch уже перехватывает все исключения этого типа или супертипа ("тип").</span><span class="sxs-lookup"><span data-stu-id="4b2f6-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="4b2f6-104">Ряд операторов [catch](../../csharp/language-reference/keywords/try-catch.md) должен быть указан в порядке убывания поколения наследования.</span><span class="sxs-lookup"><span data-stu-id="4b2f6-104">A series of [catch](../../csharp/language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="4b2f6-105">Например, наиболее производные объекты должны быть первыми.</span><span class="sxs-lookup"><span data-stu-id="4b2f6-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="4b2f6-106">Дополнительные сведения см. в разделе [исключения и обработка исключений](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="4b2f6-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="4b2f6-107">В следующем примере возникает ошибка CS0160:</span><span class="sxs-lookup"><span data-stu-id="4b2f6-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
