# <a name="compiler-error-cs0710"></a><span data-ttu-id="a1a1e-102">Ошибка компилятора CS0710</span><span class="sxs-lookup"><span data-stu-id="a1a1e-102">Compiler Error CS0710</span></span>
<span data-ttu-id="a1a1e-103">Статические классы не могут иметь конструкторы экземпляров.</span><span class="sxs-lookup"><span data-stu-id="a1a1e-103">Static classes cannot have instance constructors</span></span>  
  
 <span data-ttu-id="a1a1e-104">Для статических классов нельзя создавать экземпляры, поэтому для них не нужны конструкторы.</span><span class="sxs-lookup"><span data-stu-id="a1a1e-104">A static class cannot be instantiated, hence it has no need for constructors.</span></span> <span data-ttu-id="a1a1e-105">Чтобы избежать возникновения этой ошибки, удалите конструкторы из статических классов или, если все же необходимо создавать экземпляры, сделайте соответствующие классы не статическими.</span><span class="sxs-lookup"><span data-stu-id="a1a1e-105">To avoid this error, remove any constructors from static classes, or if you really want to construct instances, make the class non-static.</span></span>  
  
 <span data-ttu-id="a1a1e-106">Следующий пример приводит к возникновению ошибки CS0710:</span><span class="sxs-lookup"><span data-stu-id="a1a1e-106">The following sample generates CS0710:</span></span>  
  
```csharp  
// CS0710.cs  
public static class C  
{  
   public C()  // CS0710  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
