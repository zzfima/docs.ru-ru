 <span data-ttu-id="45a8c-104">При использовании оператора `goto case`должно выполняться неявное преобразование из значения goto case в тип параметра.</span><span class="sxs-lookup"><span data-stu-id="45a8c-104">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="45a8c-105">Пример</span><span class="sxs-lookup"><span data-stu-id="45a8c-105">Example</span></span>  
 <span data-ttu-id="45a8c-106">При компиляции следующего примера будет выдано предупреждение CS0469:</span><span class="sxs-lookup"><span data-stu-id="45a8c-106">The following sample generates CS0469.</span></span>  
  
```  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:   
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
