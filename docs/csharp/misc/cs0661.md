# <a name="compiler-warning-level-3-cs0661"></a><span data-ttu-id="90287-102">Предупреждение компилятора (уровень 3) CS0661</span><span class="sxs-lookup"><span data-stu-id="90287-102">Compiler Warning (level 3) CS0661</span></span>
<span data-ttu-id="90287-103">Класс "класс" определяет оператор "==" или оператор "!=", но не переопределяет Object.GetHashCode().</span><span class="sxs-lookup"><span data-stu-id="90287-103">'class' defines operator == or operator != but does not override Object.GetHashCode()</span></span>  
  
 <span data-ttu-id="90287-104">Компилятор обнаружил задаваемый пользователем оператор равенства или неравенства, но переопределение для функции **GetHashCode** отсутствует.</span><span class="sxs-lookup"><span data-stu-id="90287-104">The compiler detected the user-defined equality or inequality operator, but no override for the **GetHashCode** function.</span></span> <span data-ttu-id="90287-105">Задаваемый пользователем оператор равенства или неравенства подразумевает, что вы хотите переопределить функцию **GetHashCode** .</span><span class="sxs-lookup"><span data-stu-id="90287-105">A user-defined equality or inequality operator implies that you also want to override the **GetHashCode** function.</span></span>  
  
 <span data-ttu-id="90287-106">В следующем примере происходит ошибка CS0661:</span><span class="sxs-lookup"><span data-stu-id="90287-106">The following sample generates CS0661:</span></span>  
  
```csharp  
// CS0661.cs  
// compile with: /W:3  
class Test   // CS0661  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   public static bool operator != (object o, Test t)  
   {  
      return true;  
   }  
  
   public override bool Equals(object o)  
   {  
      return true;  
   }  
  
   // uncomment the GetHashCode function to resolve  
   // public override int GetHashCode()  
   // {  
   //    return 0;  
   // }  
  
   public static void Main()  
   {  
   }  
}  
```
