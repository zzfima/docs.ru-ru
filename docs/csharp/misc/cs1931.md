## <a name="example"></a><span data-ttu-id="0d5d4-107">Пример</span><span class="sxs-lookup"><span data-stu-id="0d5d4-107">Example</span></span>  
 <span data-ttu-id="0d5d4-108">Следующий код создает ошибку CS1931, так как идентификатор `x` используется и как локальная переменная в `Main` , и как переменная диапазона в выражении запроса:</span><span class="sxs-lookup"><span data-stu-id="0d5d4-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>

```  
class Test  
    {  
        static void Main()  
        {  
            int x = 1;  
            var y = from x in Enumerable.Range(1, 100) // CS1931  
                    select x;  
        }  
    }  
```
