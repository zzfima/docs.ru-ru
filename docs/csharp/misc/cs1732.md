<span data-ttu-id="44b6d-103">Требуется параметр.</span><span class="sxs-lookup"><span data-stu-id="44b6d-103">Expected parameter.</span></span>  
  
 <span data-ttu-id="44b6d-104">Эта ошибка возникает, если лямбда-выражение содержит запятую, следующую за входным параметром, однако следующий параметр не указывается.</span><span class="sxs-lookup"><span data-stu-id="44b6d-104">This error is produced when a lambda expression contains a comma following an input parameter but does not specify the following parameter.</span></span>
