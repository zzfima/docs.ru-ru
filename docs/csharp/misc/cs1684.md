<span data-ttu-id="333c8-103">Не удается найти тип "Имя типа", который, согласно ссылке, должен быть определен в "Пространство имен".</span><span class="sxs-lookup"><span data-stu-id="333c8-103">Reference to type 'Type Name' claims it is defined in 'Namespace', but it could not be found</span></span>  
  
 <span data-ttu-id="333c8-104">Эта ошибка может быть вызвана ссылкой в одном пространстве имен на тип, который, как подразумевается, существует в другом пространства имен, но этот тип не существует.</span><span class="sxs-lookup"><span data-stu-id="333c8-104">This error can be caused by a reference inside one namespace referring to a type that it says exists inside a second namespace, but the type does not exist.</span></span> <span data-ttu-id="333c8-105">Например, mydll.dll говорит, что тип `A` существует в yourdll.dll, но в yourdll.dll такой тип отсутствует.</span><span class="sxs-lookup"><span data-stu-id="333c8-105">For example, mydll.dll says that type `A` exists inside yourdll.dll, but no such type exists inside yourdll.dll.</span></span> <span data-ttu-id="333c8-106">Одной из возможных причин этой ошибки является использование слишком старой версии yourdll.dll, в которой еще не был определен тип `A` .</span><span class="sxs-lookup"><span data-stu-id="333c8-106">One possible cause of this error is that the version of yourdll.dll you are using is too old and `A` has not yet been defined.</span></span>  
  
 <span data-ttu-id="333c8-107">В следующем примере возникает ошибка CS1684.</span><span class="sxs-lookup"><span data-stu-id="333c8-107">The following sample generates CS1684.</span></span>

```

   public void Test() {}

```  
  
## <a name="example"></a><span data-ttu-id="333c8-109">Пример</span><span class="sxs-lookup"><span data-stu-id="333c8-109">Example</span></span>
