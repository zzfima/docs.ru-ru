# <a name="compiler-error-cs0121"></a>Ошибка компилятора CS0121

Вызов неоднозначнмежду следующими методами или свойствами: 'method1' и 'method2'

Из-за неявного преобразования компилятор не смог вызвать одну форму перегруженного метода. Эту ошибку можно устранить одним из следующих способов:

- Указать параметры метода таким образом, чтобы неявное преобразование не происходило.
- Удалите все перегрузки для метода.
- Перед вызовом метода в ролях в надлежащем типе.
- Используйте именованные аргументы.

## <a name="example"></a>Пример

Следующие примеры генерируют ошибку компилятора CS0121:

```csharp
public class Program
{
    static void f(int i, double d)
    {
    }

    static void f(double d, int i)
    {
    }

    public static void Main()
    {
        f(1, 1);   // CS0121

        // Try the following code instead:
        // f(1, 1.0);
        // or
        // f(1.0, 1);
        // or
        // f(1, (double)1);   // Cast and specify which method to call.
        // or
        // f(i: 1, 1);
        // or
        // f(d: 1, 1);

        // f(i: 1, d: 1); // This still gives CS0121
    }
}
```

## <a name="example"></a>Пример

```csharp
class Program2
{
    static int ol_invoked = 0;

    delegate int D1(int x);
    delegate T D1<T>(T x);
    delegate T D1<T, U>(U u);

    static void F(D1 d1) { ol_invoked = 1; }
    static void F<T>(D1<T> d1t) { ol_invoked = 2; }
    static void F<T, U>(D1<T, U> d1t) { ol_invoked = 3; }

    static int Test001()
    {
        F(delegate(int x) { return 1; }); // CS0121
        if (ol_invoked == 1)
            return 0;
        else
            return 1;
    }

    static int Main()
    {
        return Test001();
    }
}
```
