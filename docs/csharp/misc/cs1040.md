<span data-ttu-id="a73dc-103">Директивами препроцессору могут находиться в качестве первого символа пустое пространство в строке</span><span class="sxs-lookup"><span data-stu-id="a73dc-103">Preprocessor directives must appear as the first non-white-space character on a line</span></span>  
  
 <span data-ttu-id="a73dc-104">В строке обнаружена [директива препроцессора](../../csharp/language-reference/preprocessor-directives/index.md) , однако эта директива не является первым токеном в строке.</span><span class="sxs-lookup"><span data-stu-id="a73dc-104">A [preprocessor directive](../../csharp/language-reference/preprocessor-directives/index.md) was found on a line and was not the first token on the line.</span></span> <span data-ttu-id="a73dc-105">Директива должна быть первым токеном в строке.</span><span class="sxs-lookup"><span data-stu-id="a73dc-105">A directive must be the first token on the line.</span></span>  
  
 <span data-ttu-id="a73dc-106">Следующий пример приводит к возникновению ошибки CS1040:</span><span class="sxs-lookup"><span data-stu-id="a73dc-106">The following sample generates CS1040:</span></span>  
  
```csharp

// try the following two lines instead  
// /* Define a symbol, X */  
// #define X

{  
   public static void Main()  
   {  
   }  
}  
```
