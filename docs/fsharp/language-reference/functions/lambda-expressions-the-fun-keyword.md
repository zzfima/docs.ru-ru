# <a name="lambda-expressions-the-fun-keyword-f"></a>Лямбда-выражения: ключевое слово fun (F#)

`fun` Ключевое слово используется для определения лямбда-выражения, то есть анонимную функцию.

## <a name="syntax"></a>Синтаксис

```fsharp
fun parameter-list -> expression
```

## <a name="remarks"></a>Примечания

*Список параметров* обычно состоит из имен и, возможно, типы параметров. Как правило *список параметров* может состоять из любых шаблонов F #. Полный список шаблонов, см. в разделе [сопоставление шаблонов](../pattern-matching.md). Списки допустимых параметров включает следующие варианты.

```fsharp
// Lambda expressions with parameter lists.
fun a b c -> ...
fun (a: int) b c -> ...
fun (a : int) (b : string) (c:float) -> ...

// A lambda expression with a tuple pattern.
fun (a, b) -> …

// A lambda expression with a list pattern.
fun head :: tail -> …
```

*Выражение* представляет собой тело функции, последнее выражение которого формирует возвращаемое значение. Следующие примеры допустимых лямбда-выражений.
