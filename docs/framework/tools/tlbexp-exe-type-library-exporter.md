## <a name="examples"></a><span data-ttu-id="93734-188">Примеры</span><span class="sxs-lookup"><span data-stu-id="93734-188">Examples</span></span>  
 <span data-ttu-id="93734-189">Следующая команда создает библиотеку типов с таким же именем, что и у сборки, найденной в `myTest.dll`.</span><span class="sxs-lookup"><span data-stu-id="93734-189">The following command generates a type library with the same name as the assembly found in `myTest.dll`.</span></span>

```  
  
 <span data-ttu-id="93734-190">Следующая команда создает библиотеку типов с именем `clipper.tlb`.</span><span class="sxs-lookup"><span data-stu-id="93734-190">The following command generates a type library with the name `clipper.tlb`.</span></span>

```  
  
 <span data-ttu-id="93734-191">В следующем примере показано использование программы Tlbexp.exe для экспорта библиотеки типов из сборки, на которую ссылаются сборки, импортированные с помощью программы Tlbimp.exe.</span><span class="sxs-lookup"><span data-stu-id="93734-191">The following example illustrates using Tlbexp.exe to export a type library from an assembly that references assemblies that were imported using Tlbimp.exe.</span></span>  
  
 <span data-ttu-id="93734-192">Сначала с помощью программы Tlbimp.exe импортируйте библиотеку типов `myLib.tlb` и сохраните ее как `myLib.dll`.</span><span class="sxs-lookup"><span data-stu-id="93734-192">First use Tlbimp.exe to import the type library `myLib.tlb` and save it as `myLib.dll`.</span></span>

```  
  
 <span data-ttu-id="93734-193">Следующая команда использует компилятор C# для компиляции библиотеки `Sample.dll,`, которая ссылается на библиотеку `myLib.dll`, созданную в предыдущем примере.</span><span class="sxs-lookup"><span data-stu-id="93734-193">The following command uses the C# compiler to compile the `Sample.dll,` which references `myLib.dll` created in the previous example.</span></span>

```
