 <span data-ttu-id="456df-223">Следующая команда создает сборку с таким же именем, как у библиотеки типов в `myTest.tlb`, и расширением DLL.</span><span class="sxs-lookup"><span data-stu-id="456df-223">The following command generates an assembly with the same name as the type library found in `myTest.tlb` and with the .dll extension.</span></span>
