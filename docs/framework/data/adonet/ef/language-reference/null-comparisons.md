```  
-- Find order details and orders with no ship date.  
SELECT h.SalesOrderID  
FROM Sales.SalesOrderHeader h  
JOIN Sales.SalesOrderDetail o ON o.SalesOrderID = h.SalesOrderID  
WHERE h.ShipDate IS Null  
```  
  
 <span data-ttu-id="03b1a-111">Это совсем не похоже на Null-семантику среды CRL, где сравнение «Null равняется Null» возвращает значение True.</span><span class="sxs-lookup"><span data-stu-id="03b1a-111">This is very different from the CLR null semantics, where the null-equals-null comparison returns true.</span></span>
