<span data-ttu-id="f0b1f-103">В ADO.NET можно записать XML-представление объекта <xref:System.Data.DataSet> вместе со схемой или без нее.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-103">In ADO.NET you can write an XML representation of a <xref:System.Data.DataSet>, with or without its schema.</span></span> <span data-ttu-id="f0b1f-104">Если информация схемы встраивается внутрь XML, она записываются на языке XSD.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-104">If schema information is included inline with the XML, it is written using the XML Schema definition language (XSD).</span></span> <span data-ttu-id="f0b1f-105">Схема содержит определения таблиц для <xref:System.Data.DataSet>, а также определения связей и ограничений.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-105">The schema contains the table definitions of the <xref:System.Data.DataSet> as well as the relation and constraint definitions.</span></span>  
  
 <span data-ttu-id="f0b1f-106">Если <xref:System.Data.DataSet> записан как XML-данные, строки в <xref:System.Data.DataSet> записываются в своей текущей версии.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-106">When a <xref:System.Data.DataSet> is written as XML data, the rows in the <xref:System.Data.DataSet> are written in their current versions.</span></span> <span data-ttu-id="f0b1f-107">Однако <xref:System.Data.DataSet> может быть записан как DiffGram, так что будут включены и текущие, и исходные данные строк.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-107">However, the <xref:System.Data.DataSet> can also be written as a DiffGram so that both the current and the original values of the rows will be included.</span></span>  
  
 <span data-ttu-id="f0b1f-108">XML-представление <xref:System.Data.DataSet> могут записываться в файл, поток, **XmlWriter**, или строка.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-108">The XML representation of the <xref:System.Data.DataSet> can be written to a file, a stream, an **XmlWriter**, or a string.</span></span> <span data-ttu-id="f0b1f-109">Эти возможности обеспечивают большую гибкость способа переноса XML-представления для <xref:System.Data.DataSet>.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-109">These choices provide great flexibility for how you transport the XML representation of the <xref:System.Data.DataSet>.</span></span> <span data-ttu-id="f0b1f-110">Чтобы получить XML-представление <xref:System.Data.DataSet> как строку, используйте **GetXml** метод, как показано в следующем примере.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-110">To obtain the XML representation of the <xref:System.Data.DataSet> as a string, use the **GetXml** method as shown in the following example.</span></span>

  
```csharp  
string xmlDS = custDS.GetXml();
```  

 <span data-ttu-id="f0b1f-111">**GetXml** возвращает XML-представление <xref:System.Data.DataSet> без сведений о схеме.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-111">**GetXml** returns the XML representation of the <xref:System.Data.DataSet> without schema information.</span></span> <span data-ttu-id="f0b1f-112">Для записи сведений о схеме из <xref:System.Data.DataSet> (в виде XML-схемы) в строку, используйте **GetXmlSchema**.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-112">To write the schema information from the <xref:System.Data.DataSet> (as XML Schema) to a string, use **GetXmlSchema**.</span></span>  
  
 <span data-ttu-id="f0b1f-113">Для записи <xref:System.Data.DataSet> в файл, поток, или **XmlWriter**, использовать **WriteXml** метод.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-113">To write a <xref:System.Data.DataSet> to a file, stream, or **XmlWriter**, use the **WriteXml** method.</span></span> <span data-ttu-id="f0b1f-114">Первый параметр, передаваемый **WriteXml** — место назначения выходных данных XML.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-114">The first parameter you pass to **WriteXml** is the destination of the XML output.</span></span> <span data-ttu-id="f0b1f-115">Например, передается строка, содержащая имя файла, **System.IO.TextWriter** объекта и т. д.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-115">For example, pass a string containing a file name, a **System.IO.TextWriter** object, and so on.</span></span> <span data-ttu-id="f0b1f-116">Можно передать необязательный второй параметр **XmlWriteMode** для указания, как должны записываться выходные данные XML.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-116">You can pass an optional second parameter of an **XmlWriteMode** to specify how the XML output is to be written.</span></span>  
  
 <span data-ttu-id="f0b1f-117">В следующей таблице показаны параметры для **XmlWriteMode**.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-117">The following table shows the options for **XmlWriteMode**.</span></span>  
  
|<span data-ttu-id="f0b1f-118">Параметр XmlWriteMode</span><span class="sxs-lookup"><span data-stu-id="f0b1f-118">XmlWriteMode option</span></span>|<span data-ttu-id="f0b1f-119">Описание:</span><span class="sxs-lookup"><span data-stu-id="f0b1f-119">Description</span></span>|  
|-------------------------|-----------------|  
|<span data-ttu-id="f0b1f-120">**IgnoreSchema**</span><span class="sxs-lookup"><span data-stu-id="f0b1f-120">**IgnoreSchema**</span></span>|<span data-ttu-id="f0b1f-121">Записывает текущее содержимое <xref:System.Data.DataSet> как XML-данные, без схемы XML.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-121">Writes the current contents of the <xref:System.Data.DataSet> as XML data, without an XML Schema.</span></span> <span data-ttu-id="f0b1f-122">Это значение по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="f0b1f-122">This is the default.</span></span>|

```vb

```  
  
```csharp  
custDS.WriteXml("Customers.xml", XmlWriteMode.WriteSchema);  
```  
  
```vb  
Dim xmlSW As System.IO.StreamWriter = New System.IO.StreamWriter("Customers.xml")

```  
  
```csharp  
System.IO.StreamWriter xmlSW = new System.IO.StreamWriter("Customers.xml");

```
