 `http://MovieServer/MovieService.svc/Movies(1)/Rate`

}

> [!WARNING]
> <span data-ttu-id="836a1-171">Приведенный выше образец кода будет работать только со службами данных WCF 5.2 и более поздней версии, которые поддерживают облегченный формат JSON.</span><span class="sxs-lookup"><span data-stu-id="836a1-171">The above sample code will work only with WCF Data Services 5.2 and later which has support for JSON light.</span></span> <span data-ttu-id="836a1-172">При использовании более ранней версии служб данных WCF необходимо указать подробный тип содержимого json следующим образом: `application/json;odata=verbose`.</span><span class="sxs-lookup"><span data-stu-id="836a1-172">If using an earlier version of WCF Data Services, you must specify the json verbose content-type as follows: `application/json;odata=verbose`.</span></span>

 <span data-ttu-id="836a1-174">В приведенном выше фрагменте кода класс `MoviesModel` был сформирован с помощью Visual Studio для добавления ссылки на службу данных WCF.</span><span class="sxs-lookup"><span data-stu-id="836a1-174">In the code snippet above, the `MoviesModel` class was generated by using Visual Studio to Add Service Reference to a WCF Data Service.</span></span>
