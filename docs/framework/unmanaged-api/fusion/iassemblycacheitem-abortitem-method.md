# <a name="iassemblycacheitemabortitem-method"></a>Метод IAssemblyCacheItem::AbortItem
Позволяет сборку в глобальный кэш сборок для выполнения операции очистки, перед выпуском.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT AbortItem ();  
```
