# <a name="profiling-structures"></a>Структуры профилирования
В этом разделе описаны неуправляемые структуры, которые использует API профилирования.  
  
## <a name="in-this-section"></a>В этом разделе  
 [Структура COR_PRF_ASSEMBLY_REFERENCE_INFO](../../../../docs/framework/unmanaged-api/profiling/cor-prf-assembly-reference-info-structure.md)  
 Обеспечивает среду CLR информацией о ссылке на сборку, которую ей следует учитывать при выполнении обхода замыкания.  
  
 [Структура COR_PRF_CODE_INFO](../../../../docs/framework/unmanaged-api/profiling/cor-prf-code-info-structure.md)  
 Представляет один непрерывный блок машинного кода, хранящийся в памяти.  
  
 [Структура COR_PRF_EX_CLAUSE_INFO](../../../../docs/framework/unmanaged-api/profiling/cor-prf-ex-clause-info-structure.md)  
 Хранит сведения об определенном экземпляре исключительного предложения и связанном с ним кадре.  
  
 [Структура COR_PRF_FUNCTION](../../../../docs/framework/unmanaged-api/profiling/cor-prf-function-structure.md)  
 Выдает уникальное представление функции, объединяя ее идентификатор с идентификатором перекомпилированной версии этой функции.  
  
 [Структура COR_PRF_FUNCTION_ARGUMENT_INFO](../../../../docs/framework/unmanaged-api/profiling/cor-prf-function-argument-info-structure.md)  
 Представляет аргументы функции слева направо.  
  
 [Структура COR_PRF_FUNCTION_ARGUMENT_RANGE](../../../../docs/framework/unmanaged-api/profiling/cor-prf-function-argument-range-structure.md)  
 Представляет блок аргументов функции, которые сохраняются в памяти последовательно слева направо.  
  
 [Структура COR_PRF_GC_GENERATION_RANGE](../../../../docs/framework/unmanaged-api/profiling/cor-prf-gc-generation-range-structure.md)  
 Описывает диапазон (т. е., блок) памяти, который занимается сборкой мусора.
