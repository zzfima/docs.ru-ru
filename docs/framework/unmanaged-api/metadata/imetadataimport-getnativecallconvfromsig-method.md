# <a name="imetadataimportgetnativecallconvfromsig-method"></a>Метод IMetaDataImport::GetNativeCallConvFromSig
Возвращает собственное соглашение о вызовах для метода, представленного заданным указателем на подпись.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT GetNativeCallConvFromSig (  
   [in]  void const  *pvSig,  
   [in]  ULONG       cbSig,  
   [out] ULONG       *pCallConv  
);  
```  
  
## <a name="parameters"></a>Параметры  
 `pvSig`  
 [in] A pointer to the metadata signature of the method to return the calling convention for.  
  
 `cbSig`  
 [in] The size in bytes of `pvSig`.  
  
 `pCallConv`  
 [out] A pointer to the native calling convention.
