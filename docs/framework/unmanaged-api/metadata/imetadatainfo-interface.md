# <a name="imetadatainfo-interface"></a>Интерфейс IMetaDataInfo
Предоставляет метод, который получает сведения о сопоставлении метаданных из файла на диске с памятью.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод GetFileMapping](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|Возвращает область памяти сопоставленного файла и тип сопоставления.|
