# <a name="icordebughandlevaluedispose-method"></a>Метод ICorDebugHandleValue::Dispose
Освобождает дескриптор, который ссылается данный объект ICorDebugHandleValue без явного освобождения указатель на интерфейс.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT Dispose ();  
```
