    > [!NOTE]

### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="3bd31-108">Чтобы изменить размер нескольких элементов управления в форму</span><span class="sxs-lookup"><span data-stu-id="3bd31-108">To resize multiple controls on a form</span></span>  
  
1.  <span data-ttu-id="3bd31-109">Удерживая нажатой клавишу CTRL или SHIFT и выберите элементы управления, которые вы хотите изменить.</span><span class="sxs-lookup"><span data-stu-id="3bd31-109">Hold down the CTRL or SHIFT key and select the controls you want to resize.</span></span> <span data-ttu-id="3bd31-110">Размер первый выбранный элемент управления используется для других элементов управления.</span><span class="sxs-lookup"><span data-stu-id="3bd31-110">The size of the first control you select is used for the other controls.</span></span>  
  
2.  <span data-ttu-id="3bd31-111">На **формат** меню, выберите **установить тот же размер**, а затем выберите один из четырех вариантов.</span><span class="sxs-lookup"><span data-stu-id="3bd31-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="3bd31-112">Первые три команды изменения размеров элементов управления в соответствии с первого выбранного элемента управления.</span><span class="sxs-lookup"><span data-stu-id="3bd31-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
