### <a name="to-make-a-column-read-only-by-using-the-designer"></a><span data-ttu-id="eb2a6-111">Чтобы сделать столбец только для чтения с помощью конструктора</span><span class="sxs-lookup"><span data-stu-id="eb2a6-111">To make a column read-only by using the designer</span></span>  
  
1.  <span data-ttu-id="eb2a6-112">Щелкните глиф смарт-тега (![глиф смарт-тега](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) в правом верхнем углу <xref:System.Windows.Forms.DataGridView> управления, а затем выберите **Правка столбцов**.</span><span class="sxs-lookup"><span data-stu-id="eb2a6-112">Click the smart tag glyph (![Smart Tag Glyph](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control, and then select **Edit Columns**.</span></span>  
  
2.  <span data-ttu-id="eb2a6-113">Выберите столбец из **выбранные столбцы** списка.</span><span class="sxs-lookup"><span data-stu-id="eb2a6-113">Select a column from the **Selected Columns** list.</span></span>  
  
3.  <span data-ttu-id="eb2a6-114">В **свойства столбца** сетки, задайте <xref:System.Windows.Forms.DataGridViewColumn.ReadOnly%2A> свойства `true`.</span><span class="sxs-lookup"><span data-stu-id="eb2a6-114">In the **Column Properties** grid, set the <xref:System.Windows.Forms.DataGridViewColumn.ReadOnly%2A> property to `true`.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="eb2a6-115">Вы также можете столбец только для чтения при добавлении, выбрав **только для чтения** флажок в **добавить столбец** диалоговое окно.</span><span class="sxs-lookup"><span data-stu-id="eb2a6-115">You can also make a column read-only when you add it by selecting the **Read Only** check box in the **Add Column** dialog box.</span></span>
