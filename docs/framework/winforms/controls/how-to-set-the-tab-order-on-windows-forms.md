### <a name="to-set-the-tab-order-of-a-control"></a><span data-ttu-id="83829-110">Чтобы задать последовательность перехода элемента управления</span><span class="sxs-lookup"><span data-stu-id="83829-110">To set the tab order of a control</span></span>
