### <a name="to-add-a-table-to-the-datagrid-control-in-the-designer"></a><span data-ttu-id="ade1a-116">Чтобы добавить таблицу в элемент управления DataGrid в конструкторе</span><span class="sxs-lookup"><span data-stu-id="ade1a-116">To add a table to the DataGrid control in the designer</span></span>  
  
1.  <span data-ttu-id="ade1a-117">Для отображения данных в таблице, необходимо сначала привязать <xref:System.Windows.Forms.DataGrid> элемента управления к набору данных.</span><span class="sxs-lookup"><span data-stu-id="ade1a-117">In order to display data in the table, you must first bind the <xref:System.Windows.Forms.DataGrid> control to a dataset.</span></span> <span data-ttu-id="ade1a-118">Дополнительные сведения см. в разделе [как: привязка элемента управления DataGrid в Windows Forms к источнику данных с помощью конструктора](../../../../docs/framework/winforms/controls/bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span><span class="sxs-lookup"><span data-stu-id="ade1a-118">For more information, see [How to: Bind the Windows Forms DataGrid Control to a Data Source Using the Designer](../../../../docs/framework/winforms/controls/bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span></span>  
  
2.  <span data-ttu-id="ade1a-119">Выберите <xref:System.Windows.Forms.DataGrid> элемента управления <xref:System.Windows.Forms.DataGrid.TableStyles%2A> свойства в окне «Свойства» и затем нажмите кнопку с многоточием (![экрана VisualStudioEllipsesButton](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) рядом с полем свойство отображения **редактор коллекции DataGridTableStyle**.</span><span class="sxs-lookup"><span data-stu-id="ade1a-119">Select the <xref:System.Windows.Forms.DataGrid> control's <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property in the Properties window, and then click the ellipsis button (![VisualStudioEllipsesButton screenshot](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) next to the property to display the **DataGridTableStyle Collection Editor**.</span></span>  
  
3.  <span data-ttu-id="ade1a-120">В редакторе коллекции щелкните **добавить** Вставляемый стиль таблицы.</span><span class="sxs-lookup"><span data-stu-id="ade1a-120">In the collection editor, click **Add** to insert a table style.</span></span>  
  
4.  <span data-ttu-id="ade1a-121">Нажмите кнопку **ОК** чтобы закрыть редактор коллекции, а затем повторно открыть его, нажав кнопку с многоточием рядом с полем <xref:System.Windows.Forms.DataGrid.TableStyles%2A> свойство.</span><span class="sxs-lookup"><span data-stu-id="ade1a-121">Click **OK** to close the collection editor, and then reopen it by clicking the ellipsis button next to the <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property.</span></span>  
  
     <span data-ttu-id="ade1a-122">При повторном открытии редактора коллекции, все таблицы данных, привязанных к элементу управления будет отображаться в раскрывающемся списке для <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> свойство стиля таблицы.</span><span class="sxs-lookup"><span data-stu-id="ade1a-122">When you reopen the collection editor, any data tables bound to the control will appear in the drop-down list for the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> property of the table style.</span></span>  
  
5.  <span data-ttu-id="ade1a-123">В **члены** окно редактора коллекции, выберите стиль таблицы.</span><span class="sxs-lookup"><span data-stu-id="ade1a-123">In the **Members** box of the collection editor, click the table style.</span></span>  
  
6.  <span data-ttu-id="ade1a-124">В **свойства** редактора коллекции, выберите <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> значение для таблицы будет отображаться.</span><span class="sxs-lookup"><span data-stu-id="ade1a-124">In the **Properties** box of the collection editor, select the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> value for the table you want to display.</span></span>
