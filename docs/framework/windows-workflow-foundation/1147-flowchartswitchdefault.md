|||

## <a name="description"></a>Описание  
 Указывает, что был выбран вариант по умолчанию в параметре блок-схемы.  
  
## <a name="message"></a>Сообщение  
 Блок-схема «%1»/FlowSwitch: выбрано действие Default Case.
