желтая стрелка.</span><span class="sxs-lookup"><span data-stu-id="92a4b-114">The rehosted workflow designer is then instructed to highlight the activity on the designer surface and use the same method as the workflow debugger, specifically drawing a yellow border around the activity and displaying a yellow arrow along the left side of the designer.</span></span>
