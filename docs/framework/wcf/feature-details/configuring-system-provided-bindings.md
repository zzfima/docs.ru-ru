<span data-ttu-id="781bd-103">Привязки указывают, какой механизм связи использовать при взаимодействии с конечной точкой, а также способ подключения к конечной точке.</span><span class="sxs-lookup"><span data-stu-id="781bd-103">Bindings specify the communication mechanism to use when talking to an endpoint and indicate how to connect to an endpoint.</span></span> <span data-ttu-id="781bd-104">Привязки состоят из элементов, определяющих способ деления каналов Windows Communication Foundation (WCF) для предоставления требуемых возможностей связи.</span><span class="sxs-lookup"><span data-stu-id="781bd-104">Bindings consist of elements that define how the Windows Communication Foundation (WCF) channels are layered up to provide the required communication features.</span></span> <span data-ttu-id="781bd-105">Привязка содержит три типа элементов.</span><span class="sxs-lookup"><span data-stu-id="781bd-105">A binding contains three types of elements:</span></span>
