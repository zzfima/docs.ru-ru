## <a name="untyped-message-programming"></a><span data-ttu-id="fa3ae-116">Программирование нетипизированных сообщений</span><span class="sxs-lookup"><span data-stu-id="fa3ae-116">Untyped Message Programming</span></span>  
 <span data-ttu-id="fa3ae-117">При работе напрямую с объектами нетипизированного сообщения следует явно задать свойства нетипизированного сообщения для его сериализации в виде JSON.</span><span class="sxs-lookup"><span data-stu-id="fa3ae-117">When working directly with untyped Message objects, you must explicitly set the properties on the untyped message to serialize it as JSON.</span></span> <span data-ttu-id="fa3ae-118">В следующем фрагменте кода показано, как это сделать.</span><span class="sxs-lookup"><span data-stu-id="fa3ae-118">The following code snippet shows how to do this.</span></span>  
  
```csharp
 Message response = Message.CreateMessage(  
                  MessageVersion.None,    // No SOAP message version  
                             "*",                     // SOAP action, ignored since this is JSON  
                             "Response string: JSON format specified", // Message body  
                             new DataContractJsonSerializer(typeof(string))); // Specify DataContractJsonSerializer  
      response.Properties.Add( WebBodyFormatMessageProperty.Name,   
                    new WebBodyFormatMessageProperty(WebContentFormat.Json)); // Use JSON format  
```
