# <a name="internet-information-service-hosting-instructions"></a>Инструкции по размещению в службах IIS
Для выполнения примеров, размещаемых в службах IIS, следует убедиться, что службы IIS правильно установлены и запущены.  
  
### <a name="to-install-iis-version-75-on-windows-server-2008-r2"></a>Установка служб IIS версии 7.5 в Windows Server 2008 R2  
  
1.  Из **диспетчера сервера**выберите **ролей.** В разделе **Сводка по ролям**, нажмите кнопку **Добавить роли**.  
  
2.  Нажмите кнопку **Далее** для отображения **Выбор ролей сервера** диалогового окна.  
  
3.  Выберите **сервера приложений** из **ролей** , а затем нажмите **Далее** дважды, чтобы отобразить **Выбор служб ролей** диалоговое окно для Роль сервера приложений.  
  
4.  Выберите **веб-сервер (IIS)** флажок. Если будет предложено установить дополнительные службы ролей и компоненты, нажмите кнопку **добавить необходимые компоненты**. Нажмите кнопку **Далее** дважды, чтобы отобразить **Выбор служб ролей** диалоговое окно для роли веб-сервер (IIS).  
  
5.  Разверните **средства управления**, а затем разверните **совместимость управления IIS 6**. Выберите **IIS 6 сценариев средства**. Если будет предложено установить дополнительные службы ролей и компоненты, нажмите кнопку **добавить требуемые службы роли**. Нажмите кнопку **Далее**.  
  
6.  Если элементы выбраны правильно, нажмите кнопку **установить**.  
  
7.  По завершении установки нажмите кнопку **закрыть**.  
  
### <a name="to-install-iis-version-75-on-windows-7"></a>Установка служб IIS версии 7.5 в Windows 7  
  
1.  Нажмите кнопку **запустить**, а затем нажмите кнопку **панели управления**.  
  
2.  Откройте **программы** группы.  
  
3.  В разделе **программы и компоненты**, нажмите кнопку **Включение или отключение компонентов Windows**.  
  
4.  **Контроль учетных записей пользователей** отображается диалоговое окно. Нажмите кнопку **Продолжить**.  
  
5.  **Компоненты Windows** отображается диалоговое окно. Разверните элемент **Internet Information Services**.  
  
6.  Разверните элемент **World Wide Web Services**.  
  
7.  Разверните элемент **компоненты разработки приложений**.  
  
8.  Убедитесь, что выбраны следующие элементы.  
  
    1.  **Расширяемость .NET**  
  
    2.  **ASP.NET**  
  
    3.  **Расширения ISAPI**  
  
    4.  **Фильтры ISAPI**  
  
9. В элементе **World Wide Web Services**, разверните **основные возможности Http**.  
  
10. Убедитесь, что **статического содержимого** выбран.  
  
11. В элементе **World Wide Web Services**, разверните **безопасности**.  
  
12. Убедитесь, что **проверки подлинности Windows** выбран.  
  
13. В разделе **Internet Information Services** каталога, разверните элемент **веб-средства управления**и выберите **консоль управления IIS**.  
  
14. Разверните элемент **совместимость управления IIS 6**, а затем выберите **сценариев IIS 6**.  
  
15. В разделе **Internet Information Services** каталога, разверните элемент **Microsoft .NET Framework 3.5.1**, а затем выберите **активация Windows Communication Foundation Http**.  
  
16. Нажмите кнопку **ОК**.

### <a name="to-install-iis-version-60-on-windows-server-2003"></a>Установка служб IIS версии 6.0 в Windows Server 2003  
  
1.  Из **управление сервером**, нажмите кнопку **добавить или удалить роль**, а затем нажмите кнопку **Далее**.  
  
2.  Выберите **сервер приложений (IIS, ASP.NET)** из **роли сервера** , а затем нажмите **Далее**.  
  
3.  Выберите **включить ASP.NET** флажок и нажмите кнопку **Далее**.  
  
4.  Если элементы выбраны правильно, нажмите кнопку Далее.  
  
### <a name="to-install-iis-version-51-on-windows-xp-with-service-pack-2-and-service-pack-3-installed"></a>Установка служб IIS версии 5.1 в Windows XP с установленными пакетами обновления 2 и 3 (SP2 и SP3)  
  
1.  В панели управления выберите **Установка и удаление программ**.  
  
2.  В **Установка и удаление программ** диалоговое окно, нажмите кнопку **компонентов Windows**.  
  
3.  В **мастер компонентов Windows**выберите **Internet Information Services (IIS)** флажок и нажмите кнопку **Далее**.  
  
4.  Если **файлы, необходимые** диалоговое окно, вставьте диск установки операционной системы, перейдите в папку i386 и нажмите кнопку **ОК**.  
  
5.  По завершении установки нажмите кнопку **Готово**.  
  
6.  Закройте **Установка и удаление программ** диалоговое окно, а затем закройте **панели управления**.  
  
### <a name="to-verify-the-installation-of-iis-and-aspnet"></a>Проверка установки служб IIS и ASP.NET  
  
1.  Сохраните HTML-файл, указанный в конце данного раздела, в корневом каталоге \InetPub\wwwroot и назовите его Default.aspx.  
  
2.  Откройте окно браузера.  
  
3.  Тип `http://localhost/Default.aspx` в адресной строке и нажмите клавишу ВВОД.  
  
4.  Должна появиться веб-страница с текстом "Здравствуй, мир!".  
  
> [!NOTE]
>  При каждой установке новой версии [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] необходимо повторно регистрировать aspnet_isapi как расширение веб-службы для IIS. Чтобы сделать это, выполните команду `aspnet_regiis –I –enable`.
