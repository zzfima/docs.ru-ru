 <span data-ttu-id="d0077-119">Например, в следующем примере кода показана операция, которая принимает два целых числа и возвращает другое целое число.</span><span class="sxs-lookup"><span data-stu-id="d0077-119">For example, the following code example shows an operation that takes two integers and returns another integer.</span></span> <span data-ttu-id="d0077-120">Здесь может быть создано несколько исключений, поэтому при разработке контракта ошибок нужно определить, какие состояния ошибки важны для клиента.</span><span class="sxs-lookup"><span data-stu-id="d0077-120">Several exceptions can be thrown here, so when designing the fault contract, you must determine which error conditions are important for your client.</span></span> <span data-ttu-id="d0077-121">В этом случае служба должна обнаруживать исключение <xref:System.DivideByZeroException?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="d0077-121">In this case, the service should detect the <xref:System.DivideByZeroException?displayProperty=nameWithType> exception.</span></span>  
  
```csharp

{  
    [OperationContract]   
    int Divide(int a, int b)  
    {  
      if (b==0) throw new Exception("Division by zero!");  
      return a/b;  
    }  
}  
```
