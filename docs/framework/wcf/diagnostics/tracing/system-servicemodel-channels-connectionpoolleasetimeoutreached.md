# System.ServiceModel.Channels.ConnectionPoolLeaseTimeoutReached
System.ServiceModel.Channels.ConnectionPoolLeaseTimeoutReached  
  
## Описание  
 Подключение превысило время ожидания аренды подключения для данного пула подключений и было закрыто.
