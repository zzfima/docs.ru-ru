# System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached
System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached  
  
## Описание  
 Достигнута квота MaxOutboundConnectionsPerEndpoint, поэтому соединение было закрыто и повторно не использовано прослушивателем.
