|||  
|-|-|  
|Идентификатор|3819|  
|Ключевые слова|Службы маршрутизации|  
|Уровень|Сведения|  
|Канал|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Описание  
 Событие возникает, если служба маршрутизации обращается к резервным конечным точкам.  
  
## <a name="message"></a>Сообщение  
 Сообщение службы маршрутизации с идентификатором «%1» [операция %2] перемещено в резервную конечную точку «%3».
