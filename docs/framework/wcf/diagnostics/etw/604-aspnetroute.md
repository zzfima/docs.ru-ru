## <a name="description"></a>Описание  
 Это событие создается при добавлении нового маршрута Asp.Net «%1» с типом службы «%2» и типом serviceHostFactoryType «%3».

 Добавлен новый маршрут Asp.Net «%1» с типом службы «%2» и типом serviceHostFactoryType «%3».
