# <a name="3805---routingservicedisplayconfig"></a>3805 - RoutingServiceDisplayConfig
## <a name="properties"></a>Свойства  
  
|||  
|-|-|  
|ID|3805|  
|Ключевые слова|Службы маршрутизации|  
|Уровень|Verbose|  
|Канал|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Описание  
 Событие возникает, если служба маршрутизации настроена с помощью параметра RouteOnHeadersOnly.  
  
## <a name="message"></a>Сообщение  
 Служба маршрутизации настраивается с помощью RouteOnHeadersOnly: %1, SoapProcessingEnabled: %2, EnsureOrderedDispatch: %3.
