# <a name="calls-failed"></a>Сбои вызовов
Имя счетчика: Calls Failed  
  
## <a name="description"></a>Описание:  
 Количество вызовов с необработанными исключениями в данной операции. Значение этого счетчика увеличивается при каждом обнаружении необработанного исключения в данной операции.
