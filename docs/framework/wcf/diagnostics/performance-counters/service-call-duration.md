# Служба: длительность обработки вызова
Имя счетчика: Call Duration  
  
## Описание  
 Средняя длительность вызовов к этой службе.  Средняя длительность вычисляется по следующей формуле: \(N1\-N0\)\/\(D1\-D0\).
