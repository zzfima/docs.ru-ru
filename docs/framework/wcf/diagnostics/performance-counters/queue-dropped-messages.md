## <a name="description"></a>Описание:  
 Количество сообщений, отброшенных использующим очереди транспортом в этой службе.  
  
 См. в разделе [обработка подозрительных сообщений](https://go.microsoft.com/fwlink/?LinkId=96233) Дополнительные сведения о при сообщения будут удаляться.
