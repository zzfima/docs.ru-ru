# <a name="failedtotraceevent"></a>FailedToTraceEvent
ИД: 104  
  
 "Уровень серьезности" — Ошибка  
  
 Категори Трассировка  
  
## <a name="description"></a>Описание  
 Одно или несколько событий не трассировались. В событии указаны исходная строка события, имя и идентификатор процесса.
