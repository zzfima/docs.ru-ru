# <a name="wasconnectiontimedout"></a>WasConnectionTimedout
Идентификатор: 163  
  
 Важность: ошибка  
  
 Категория: SMSvcHost  
  
## <a name="description"></a>Описание  
 Данное событие указывает, что время ожидания подключения, необходимого службе активации Windows (WAS), вышло, и содержит имя и идентификатор процесса.
