2.  <span data-ttu-id="00c92-110">В конце четвертой секунды используется экземпляр <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> класс для резкого перемещения прямоугольника в следующую позицию.</span><span class="sxs-lookup"><span data-stu-id="00c92-110">At the end of the fourth second, uses an instance of the <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> class to suddenly move the rectangle to the next position.</span></span> <span data-ttu-id="00c92-111">Дискретные ключевые кадры, например <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> , создают скачкообразные переходы между значениями.</span><span class="sxs-lookup"><span data-stu-id="00c92-111">Discrete key frames like <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> create sudden jumps between values.</span></span> <span data-ttu-id="00c92-112">В этом примере прямоугольник находится в начальной позиции, а затем внезапно появляется в позиции 500.</span><span class="sxs-lookup"><span data-stu-id="00c92-112">In this example, the rectangle is at the starting position and then suddenly appears at the 500 position.</span></span>  
  
3.  <span data-ttu-id="00c92-113">В последних двух секунд используется экземпляр <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame> класс для перемещения прямоугольника обратно в исходное положение.</span><span class="sxs-lookup"><span data-stu-id="00c92-113">In the final two seconds, uses an instance of the <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame> class to move the rectangle back to its starting position.</span></span> <span data-ttu-id="00c92-114">Ключевые кадры сплайна, например <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame> , создают переменный переход между значениями согласно значению <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame.KeySpline%2A> свойство.</span><span class="sxs-lookup"><span data-stu-id="00c92-114">Spline key frames like <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame> create a variable transition between values according to the value of the <xref:System.Windows.Media.Animation.SplineDoubleKeyFrame.KeySpline%2A> property.</span></span> <span data-ttu-id="00c92-115">В этом примере прямоугольник начинает двигаться медленно и ускоряется экспоненциально к концу временного отрезка.</span><span class="sxs-lookup"><span data-stu-id="00c92-115">In this example, the rectangle begins by moving slowly and then speeds up exponentially toward the end of the time segment.</span></span>
