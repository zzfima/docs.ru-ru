## <a name="graphics-hardware"></a><span data-ttu-id="51afa-104">Графическое оборудование</span><span class="sxs-lookup"><span data-stu-id="51afa-104">Graphics Hardware</span></span>  
 <span data-ttu-id="51afa-105">Ниже перечислены возможности графического оборудования, которые сильнее всего влияют на уровни отрисовки.</span><span class="sxs-lookup"><span data-stu-id="51afa-105">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>  
  
- <span data-ttu-id="51afa-106">**Видеопамять**.
