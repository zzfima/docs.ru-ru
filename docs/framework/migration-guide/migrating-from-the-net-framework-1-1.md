## <a name="retargeting-or-recompiling"></a><span data-ttu-id="b3eee-107">Переназначение и перекомпиляция</span><span class="sxs-lookup"><span data-stu-id="b3eee-107">Retargeting or Recompiling</span></span>

-   <span data-ttu-id="b3eee-109">Можно перенацелить приложение для запуска в платформе [!INCLUDE[net_v40_long](../../../includes/net-v40-long-md.md)].</span><span class="sxs-lookup"><span data-stu-id="b3eee-109">You can retarget the application to run under [!INCLUDE[net_v40_long](../../../includes/net-v40-long-md.md)].</span></span> <span data-ttu-id="b3eee-110">Для изменения целевой платформы необходимо добавить в файл конфигурации приложения элемент [\<supportedRuntime>](../../../docs/framework/configure-apps/file-schema/startup/supportedruntime-element.md), который позволит приложению запускаться в платформе [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)].</span><span class="sxs-lookup"><span data-stu-id="b3eee-110">Retargeting requires that you add a [\<supportedRuntime>](../../../docs/framework/configure-apps/file-schema/startup/supportedruntime-element.md) element to the application's configuration file that allows it to run under [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)].</span></span> <span data-ttu-id="b3eee-111">Такой файл конфигурации имеет следующий вид:</span><span class="sxs-lookup"><span data-stu-id="b3eee-111">Such a configuration file takes the following form:</span></span>

    ```xml  
       </startup>  
    </configuration>  
    ```  
  
-   <span data-ttu-id="b3eee-112">Можно перекомпилировать приложение с помощью компилятора, предназначенного для [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)].</span><span class="sxs-lookup"><span data-stu-id="b3eee-112">You can recompile the application with a compiler that targets the [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)].</span></span> <span data-ttu-id="b3eee-113">Если для разработки и компиляции решения первоначально использовалась среда Visual Studio 2003, можно открыть решение в среде [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)] и использовать диалоговое окно **Совместимость проекта** для конвертации решения и файлов проекта из форматов, используемых Visual Studio 2003, в формат Microsoft Build Engine (MSBuild), используемый средой [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)].</span><span class="sxs-lookup"><span data-stu-id="b3eee-113">If you originally used Visual Studio 2003 to develop and compile your solution, you can open the solution in [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)] and use the **Project Compatibility** dialog box to convert the solution and project files from the formats used by Visual Studio 2003 to the Microsoft Build Engine (MSBuild) format used by [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)].</span></span>  
  
 <span data-ttu-id="b3eee-114">Независимо от того, перекомпилируется ли или перенацеливается приложение, необходимо определить, затронуто ли оно какими-либо изменениями, введенными в более поздних версиях платформы .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="b3eee-114">Regardless of whether you prefer to recompile or retarget your application, you must determine whether your application is affected by any changes introduced in later versions of the .NET Framework.</span></span> <span data-ttu-id="b3eee-115">Эти изменения могут быть двух типов:</span><span class="sxs-lookup"><span data-stu-id="b3eee-115">These changes are of two kinds:</span></span>

-   <span data-ttu-id="b3eee-117">отметка типов и членов типов как нерекомендуемых или устаревших в более поздних относительно [!INCLUDE[net_v11_short](../../../includes/net-v11-short-md.md)] версиях платформы .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="b3eee-117">Types and type members that have been marked as deprecated or obsolete between the [!INCLUDE[net_v11_short](../../../includes/net-v11-short-md.md)] and later versions of the .NET Framework.</span></span>
