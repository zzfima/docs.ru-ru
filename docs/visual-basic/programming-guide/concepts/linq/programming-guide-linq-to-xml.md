# <a name="programming-guide-linq-to-xml-visual-basic"></a>Руководством по программированию (LINQ to XML) (Visual Basic)
Этот раздел содержит теоретические и практические сведения о программировании при помощи [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)].  
  
## <a name="who-should-read-this-documentation"></a>Кому необходимо ознакомиться с этой документацией  
 Эта документация предназначена для разработчиков, которые уже понимают Visual Basic и некоторые основные аспекты .NET Framework.  
  
 Цель этой документации — упростить работу с [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] для любых разработчиков. [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] позволяет упростить программирование XML. Чтобы использовать это средство, не обязательно быть высокопрофессиональным разработчиком.  
  
 [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] основывается на использовании универсальных классов. Поэтому очень важно понимать использование универсальных типов и классов. Дополнительные сведения см. [в разделе Универсальные типы в Visual Basic](../../../../visual-basic/programming-guide/language-features/data-types/generic-types.md).
