## <a name="changes-during-reference-type-conversions"></a><span data-ttu-id="0a520-179">Изменения во время преобразования ссылочного типа</span><span class="sxs-lookup"><span data-stu-id="0a520-179">Changes During Reference Type Conversions</span></span>  
 <span data-ttu-id="0a520-180">При преобразовании из *ссылочного типа* копируется только указатель на значение.</span><span class="sxs-lookup"><span data-stu-id="0a520-180">A conversion from a *reference type* copies only the pointer to the value.</span></span> <span data-ttu-id="0a520-181">Само значение не копируется и не изменяется каким-либо образом.</span><span class="sxs-lookup"><span data-stu-id="0a520-181">The value itself is neither copied nor changed in any way.</span></span> <span data-ttu-id="0a520-182">Единственное, что может изменить, — это тип данных переменной, содержащей указатель.</span><span class="sxs-lookup"><span data-stu-id="0a520-182">The only thing that can change is the data type of the variable holding the pointer.</span></span> <span data-ttu-id="0a520-183">В следующем примере тип данных преобразуется из производного класса в его базовый класс, но объект, на который теперь указывают обе переменные, не изменяется.</span><span class="sxs-lookup"><span data-stu-id="0a520-183">In the following example, the data type is converted from the derived class to its base class, but the object that both variables now point to is unchanged.</span></span>  
  
```vb  
' Assume class cSquare inherits from class cShape.  
Dim shape As cShape  
Dim square As cSquare = New cSquare  
' The following statement performs a widening  
' conversion from a derived class to its base class.  
shape = square  
```
