## <a name="an-anonymous-type-definition"></a><span data-ttu-id="03cfa-180">Определение анонимного типа</span><span class="sxs-lookup"><span data-stu-id="03cfa-180">An Anonymous Type Definition</span></span>  
 <span data-ttu-id="03cfa-181">В ответ на объявление экземпляра анонимного типа компилятор создает новое определение класса, которое содержит указанные свойства.</span><span class="sxs-lookup"><span data-stu-id="03cfa-181">In response to the declaration of an instance of an anonymous type, the compiler creates a new class definition that contains the specified properties.</span></span>  
  
 <span data-ttu-id="03cfa-182">Если анонимный тип содержит по крайней мере одно ключевое свойство, определение переопределяет три члена, унаследованные от <xref:System.Object>: <xref:System.Object.Equals%2A>, <xref:System.Object.GetHashCode%2A>и <xref:System.Object.ToString%2A>.</span><span class="sxs-lookup"><span data-stu-id="03cfa-182">If the anonymous type contains at least one key property, the definition overrides three members inherited from <xref:System.Object>: <xref:System.Object.Equals%2A>, <xref:System.Object.GetHashCode%2A>, and <xref:System.Object.ToString%2A>.</span></span> <span data-ttu-id="03cfa-183">Код, созданный для проверки равенства и определения значения хэш-кода, рассматривает только ключевые свойства.</span><span class="sxs-lookup"><span data-stu-id="03cfa-183">The code produced for testing equality and determining the hash code value considers only the key properties.</span></span> <span data-ttu-id="03cfa-184">Если анонимный тип не содержит свойств ключа, переопределяется только <xref:System.Object.ToString%2A>.</span><span class="sxs-lookup"><span data-stu-id="03cfa-184">If the anonymous type contains no key properties, only <xref:System.Object.ToString%2A> is overridden.</span></span> <span data-ttu-id="03cfa-185">Явно именованные свойства анонимного типа не могут конфликтовать с этими созданными методами.</span><span class="sxs-lookup"><span data-stu-id="03cfa-185">Explicitly named properties of an anonymous type cannot conflict with these generated methods.</span></span> <span data-ttu-id="03cfa-186">То есть нельзя использовать `.Equals`, `.GetHashCode`или `.ToString` для именования свойства.</span><span class="sxs-lookup"><span data-stu-id="03cfa-186">That is, you cannot use `.Equals`, `.GetHashCode`, or `.ToString` to name a property.</span></span>  
  
 <span data-ttu-id="03cfa-187">Определения анонимных типов, имеющие по крайней мере одно ключевое свойство, также реализуют интерфейс <xref:System.IEquatable%601?displayProperty=nameWithType>, где `T` является типом анонимного типа.</span><span class="sxs-lookup"><span data-stu-id="03cfa-187">Anonymous type definitions that have at least one key property also implement the <xref:System.IEquatable%601?displayProperty=nameWithType> interface, where `T` is the type of the anonymous type.</span></span>
