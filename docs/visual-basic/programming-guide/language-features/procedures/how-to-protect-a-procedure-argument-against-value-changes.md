<span data-ttu-id="9f99d-103">Если процедура объявляет параметр как [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic предоставляет код процедуры прямую ссылку на элемент программирования, в аргументе в вызывающем коде.</span><span class="sxs-lookup"><span data-stu-id="9f99d-103">If a procedure declares a parameter as [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic gives the procedure code a direct reference to the programming element underlying the argument in the calling code.</span></span> <span data-ttu-id="9f99d-104">Это позволяет процедуре для изменения значения в аргументе в вызывающем коде.</span><span class="sxs-lookup"><span data-stu-id="9f99d-104">This permits the procedure to change the value underlying the argument in the calling code.</span></span> <span data-ttu-id="9f99d-105">В некоторых случаях вызывающий код может потребоваться защита от таких изменений.</span><span class="sxs-lookup"><span data-stu-id="9f99d-105">In some cases the calling code might want to protect against such a change.</span></span>  
  
 <span data-ttu-id="9f99d-106">Всегда можно защитить аргумент от изменений, объявив соответствующий параметр [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) в процедуре.</span><span class="sxs-lookup"><span data-stu-id="9f99d-106">You can always protect an argument from change by declaring the corresponding parameter [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) in the procedure.</span></span> <span data-ttu-id="9f99d-107">Если вы хотите иметь возможность изменять данный аргумент в некоторых случаях, но не другие, его можно объявить `ByRef` и позволить вызывающему коду определять механизм передачи в каждом вызове.</span><span class="sxs-lookup"><span data-stu-id="9f99d-107">If you want to be able to change a given argument in some cases but not others, you can declare it `ByRef` and let the calling code determine the passing mechanism in each call.</span></span> <span data-ttu-id="9f99d-108">Это достигается путем заключения в круглые скобки, чтобы передать его по значению соответствующего аргумента, или не заключать в круглые скобки, чтобы передавать по ссылке.</span><span class="sxs-lookup"><span data-stu-id="9f99d-108">It does this by enclosing the corresponding argument in parentheses to pass it by value, or not enclosing it in parentheses to pass it by reference.</span></span> <span data-ttu-id="9f99d-109">Дополнительные сведения см. в разделе [как: Принудительная аргумента, передаваемые по значению](./how-to-force-an-argument-to-be-passed-by-value.md).</span><span class="sxs-lookup"><span data-stu-id="9f99d-109">For more information, see [How to: Force an Argument to Be Passed by Value](./how-to-force-an-argument-to-be-passed-by-value.md).</span></span>
