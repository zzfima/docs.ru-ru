# <a name="registry-summary-visual-basic"></a>Сводка по реестру (Visual Basic)
Ключевые слова Visual Studio и члены библиотек времени выполнения упорядочены по назначению и использованию.  
  
 С помощью `My` предоставляет большую производительность и быстродействие в операциях реестра, чем эти элементы. Дополнительные сведения см. в разделе <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.
