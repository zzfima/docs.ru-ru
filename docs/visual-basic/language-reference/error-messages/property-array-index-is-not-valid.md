# <a name="property-array-index-is-not-valid"></a>Недопустимый индекс массива свойства
Предоставленное значение не является допустимым для индекса массива свойства.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   См. в документации компонента, чтобы убедиться в том, что индекс находится в допустимом диапазоне для указанного свойства.
