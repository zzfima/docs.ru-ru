# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>Имя файла или класса не найдено в ходе выполнения операции автоматизации (Visual Basic)
Имя, указанное для файла или класса в вызове `GetObject` не удалось найти функцию.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте имена и повторите попытку. Убедитесь, что имя, используемое для `class` параметр соответствует, зарегистрированные в системе.
