# <a name="xml-namespace-uri-httpwwww3orgxml1998namespace-can-be-bound-only-to-xmlns"></a>URI пространства имен XML `http://www.w3.org/XML/1998/namespace`; может быть связан только с «xmlns»
URI `http://www.w3.org/XML/1998/namespace` используется в объявление пространства имен XML. Этот URI является зарезервированным пространством имен и не может быть включен в объявление пространства имен XML.  
  
 **Идентификатор ошибки:** BC31183  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
Удалите объявление пространства имен XML или замените URI `http://www.w3.org/XML/1998/namespace` с допустимым URI пространства имен.
