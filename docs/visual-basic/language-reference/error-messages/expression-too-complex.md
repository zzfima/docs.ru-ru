# <a name="expression-too-complex"></a>Слишком сложное выражение
Выражение с плавающей запятой содержит слишком много вложенных подвыражения.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Разбейте выражение на количество отдельных выражений так, чтобы предотвратить возникновение ошибки.
