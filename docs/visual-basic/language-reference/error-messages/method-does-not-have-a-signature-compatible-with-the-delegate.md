# <a name="method-does-not-have-a-signature-compatible-with-the-delegate"></a>Метод не имеет подписи, совместимой с делегатом
Есть несоответствие между сигнатурами метода и делегата, к которому вы пытаетесь использовать. Типы параметров и значений, возвращаемых классом делегата, определяются оператором `Delegate` . Любая процедура, которая имеет соответствующие типами параметров и возвращаемые типы можно использовать для создания экземпляра этого типа делегата.  
  
 **Идентификатор ошибки**: BC36563
