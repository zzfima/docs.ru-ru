# <a name="argument-not-optional-visual-basic"></a>Аргумент не является обязательным (Visual Basic)
Количество и типы аргументов должны совпадать ожидалось. Неправильное число аргументов является либо пропущенный аргумент не является необязательным. Аргумент можно опустить из-за обращения к определяемой пользователем процедуры, только если он был объявлен `Optional` в определении процедуры.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Введите все необходимые аргументы.  
  
2.  Убедитесь, что пропущенные аргументы являются необязательными. Если они отсутствуют, либо аргумента в вызове, либо объявить параметр `Optional` в определении.
