# <a name="unable-to-find-required-file-39ltfilenamegt39"></a>Не удалось найти требуемый файл &#39; &lt;filename&gt;&#39;

 **Идентификатор ошибки:** BC30655  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переустановите Visual Studio.
