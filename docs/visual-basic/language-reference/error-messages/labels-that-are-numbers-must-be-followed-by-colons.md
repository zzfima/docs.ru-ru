# <a name="labels-that-are-numbers-must-be-followed-by-colons"></a>После меток, представляющих собой числа, следует ставить двоеточие
Номера строк следуют тем же правилам, что другие виды меток и должны содержать двоеточие.  
  
 **Идентификатор ошибки:** BC30801  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Поместить число, за которым следует двоеточие в начале строки кода; Например:  
  
    ```  
    400:    X += 1  
    ```
