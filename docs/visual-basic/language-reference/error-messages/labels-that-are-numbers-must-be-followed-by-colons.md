# <a name="labels-that-are-numbers-must-be-followed-by-colons"></a>После меток, представляющих собой числа, следует ставить двоеточие
Номера строк подчиняются тем же правилам, как другие виды меток и должны содержать двоеточие.  
  
 **Идентификатор ошибки:** BC30801  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Поместите число, за которым следует двоеточие в начале строки кода; Например:  
  
    ```  
    400:    X += 1  
    ```
