# <a name="unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>Запись во временный файл невозможна, поскольку его каталог недоступен
Visual Basic не удалось определить путь, где хранятся временные файлы.  
  
 **Идентификатор ошибки:** BC30698  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Перезапустите [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
2.  Если эта ошибка повторится, переустановите [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].
