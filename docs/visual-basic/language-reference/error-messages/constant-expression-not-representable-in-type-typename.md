# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>Константное выражение, которое нельзя представить в типе &#39; &lt;typename&gt;&#39;
Вы пытаетесь вычислить константу, которая не соответствует целевому типу, обычно из-за переполнения диапазона.  
  
 **Идентификатор ошибки:** BC30439  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Измените тип целевого объекта, который может обрабатывать константы.
