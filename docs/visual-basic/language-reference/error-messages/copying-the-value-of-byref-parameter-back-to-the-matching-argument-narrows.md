 <span data-ttu-id="a1de9-107">Если передается аргумент [ByRef](../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic иногда копирует его значение в локальную переменную в процедуре вместо передачи ссылки.</span><span class="sxs-lookup"><span data-stu-id="a1de9-107">If you pass the argument [ByRef](../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic sometimes copies the argument value into a local variable in the procedure instead of passing a reference.</span></span> <span data-ttu-id="a1de9-108">В этом случае когда процедура возвращает результат, Visual Basic необходимо скопировать значение локальной переменной обратно в аргумент в вызывающем коде.</span><span class="sxs-lookup"><span data-stu-id="a1de9-108">In such a case, when the procedure returns, Visual Basic must then copy the local variable value back into the argument in the calling code.</span></span>  
  
 <span data-ttu-id="a1de9-109">Если значение аргумента `ByRef` копируется в процедуру, а аргумент и параметр имеют один и тот же тип, то преобразование не требуется.</span><span class="sxs-lookup"><span data-stu-id="a1de9-109">If a `ByRef` argument value is copied into the procedure and the argument and parameter are of the same type, no conversion is necessary.</span></span> <span data-ttu-id="a1de9-110">Но если типы различны, Visual Basic необходимо преобразовать в обоих направлениях.</span><span class="sxs-lookup"><span data-stu-id="a1de9-110">But if the types are different, Visual Basic must convert in both directions.</span></span> <span data-ttu-id="a1de9-111">Если один из типов является типом соответствующего класса или структуры, объектов Visual Basic необходимо преобразовать его в и из другого типа.</span><span class="sxs-lookup"><span data-stu-id="a1de9-111">If one of the types is your class or structure type, Visual Basic must convert it both to and from the other type.</span></span> <span data-ttu-id="a1de9-112">Если один из этих преобразований является расширяющим, обратное преобразование может быть сужающим.</span><span class="sxs-lookup"><span data-stu-id="a1de9-112">If one of these conversions is widening, the reverse conversion might be narrowing.</span></span>

-   <span data-ttu-id="a1de9-115">По возможности используйте аргумент вызова того же типа, что и параметр процедуры, поэтому Visual Basic не нужно выполнять никаких преобразований.</span><span class="sxs-lookup"><span data-stu-id="a1de9-115">If possible, use a calling argument of the same type as the procedure parameter, so Visual Basic does not need to do any conversion.</span></span>
