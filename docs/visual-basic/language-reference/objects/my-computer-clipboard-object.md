# <a name="mycomputerclipboard-object"></a>Объект My.Computer.Clipboard
Предоставляет методы для управления буфером обмена.  
  
## <a name="remarks"></a>Примечания  
 Дополнительные сведения о методах и свойствах `My.Computer.Clipboard` объекта см. в разделе <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>.</xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>  
  
 Дополнительные сведения см. в разделе [хранение данных и чтение из буфера обмена](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md).  
  
> [!NOTE]
>  Можно также использовать методы <xref:System.Windows.Forms.Clipboard?displayProperty=fullName>класс для работы в буфер обмена.</xref:System.Windows.Forms.Clipboard?displayProperty=fullName>
