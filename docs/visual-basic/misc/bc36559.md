<span data-ttu-id="aefef-103">Пока тип свойства анонимного типа не установлен, он не может использоваться для установки типа другого свойства.</span><span class="sxs-lookup"><span data-stu-id="aefef-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="aefef-104">Например, в следующем объявлении `.IDName = .LastName` недопустимо, поскольку еще не выполнена инициализация `.LastName` .</span><span class="sxs-lookup"><span data-stu-id="aefef-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>
