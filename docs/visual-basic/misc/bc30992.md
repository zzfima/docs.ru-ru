## <a name="example"></a><span data-ttu-id="6e08e-109">Пример</span><span class="sxs-lookup"><span data-stu-id="6e08e-109">Example</span></span>  
 <span data-ttu-id="6e08e-110">В приведенном ниже классе содержится используемое по умолчанию свойство `defaultProp`, для которого требуется целочисленный аргумент.</span><span class="sxs-lookup"><span data-stu-id="6e08e-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```

