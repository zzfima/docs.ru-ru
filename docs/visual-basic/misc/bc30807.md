# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39; Разрешить &#39; и &#39; Выбрать &#39; операторы присваивания больше не поддерживаются.
Операторы`Let` и `Set` нельзя использовать в присваиваниях.  
  
 **Идентификатор ошибки:** BC30807  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите `Let` и `Set` из операторов присваивания, например:  
  
     `X = 2`
