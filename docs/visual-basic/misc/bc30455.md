# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-ltmethodnamegt"></a>Не задан аргумент для параметра "&lt;parametername&gt;«из»&lt;имя_метода&gt;"
Аргумент для обязательного параметра не указан.  
  
 **Идентификатор ошибки:** BC30455  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Предоставьте аргумент для указанного параметра.
