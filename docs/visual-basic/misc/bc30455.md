# <a name="argument-not-specified-for-parameter-parametername-of-methodname"></a>Не задан аргумент для параметра '\<имя_параметра >' из '\<имя_метода >'
Аргумент для обязательного параметра не указан.  
  
 **Идентификатор ошибки:** BC30455  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Предоставьте аргумент для указанного параметра.
