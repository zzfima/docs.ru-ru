# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-39ltmethodnamegt39"></a>Не указан аргумент для параметра &#39; &lt;имя_параметра&gt;&#39; &#39;&lt; имя_метода&gt;&#39;
Аргумент для обязательного параметра не указан.  
  
 **Идентификатор ошибки:** BC30455  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Предоставьте аргумент для указанного параметра.
