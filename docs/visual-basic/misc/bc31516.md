# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Тип "\<typename >" не может использоваться как атрибут, так как он не объявлен как «Public»
Атрибуты должны быть открытыми.  
  
 **Идентификатор ошибки:** BC31516  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Добавьте модификатор `Public` в класс атрибута.
