# <a name="type-or-with-expected"></a><span data-ttu-id="26eaa-102">Требуется тип или "With"</span><span class="sxs-lookup"><span data-stu-id="26eaa-102">Type or 'With' expected</span></span>
<span data-ttu-id="26eaa-103">При объявлении экземпляра класса за ключевым словом `New` должно следовать имя типа или `With`.</span><span class="sxs-lookup"><span data-stu-id="26eaa-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="26eaa-104">Например, в следующих инструкциях объявляется `client` как экземпляр класса `Customer` .</span><span class="sxs-lookup"><span data-stu-id="26eaa-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="26eaa-105">Имя типа `Customer` , следующее за `New`.</span><span class="sxs-lookup"><span data-stu-id="26eaa-105">The type name `Customer` follows `New`.</span></span>  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="26eaa-106">Начиная с версии [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], вы можете объявлять объект как экземпляр анонимного типа, и в этом случае не указывать тип данных.</span><span class="sxs-lookup"><span data-stu-id="26eaa-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="26eaa-107">В объявлениях анонимных типов за `With` следует ключевое слово `New`.</span><span class="sxs-lookup"><span data-stu-id="26eaa-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```  
  
 <span data-ttu-id="26eaa-108">**Идентификатор ошибки:** BC30988</span><span class="sxs-lookup"><span data-stu-id="26eaa-108">**Error ID:** BC30988</span></span>
