# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Ожидались соответствующие закрывающие кавычки для значения атрибута XML
В значении атрибута, которое разделяется двойными кавычками, отсутствуют закрывающие кавычки.  
  
 **Идентификатор ошибки:** BC31164  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте закрывающие двойные кавычки, чтобы отделить значение атрибута.
