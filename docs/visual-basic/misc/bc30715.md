# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>Операторы 'Catch' недопустимы в окне интерпретации
`Catch`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30715  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `Catch` из отлаживаемого кода.
