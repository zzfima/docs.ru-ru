# Операторы Catch недопустимы в окне интерпретации
Операторы `Catch` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30715  
  
### Исправление ошибки  
  
-   Удалите операторы `Catch` из отлаживаемого кода.
