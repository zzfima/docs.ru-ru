# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>Контексты и операторы with недопустимы в окнах отладки
`With`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30726  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `With` из отлаживаемого кода.
