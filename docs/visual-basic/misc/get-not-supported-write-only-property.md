# Get не поддерживается (свойство доступно только для записи).
Не удалось прочитать свойство, поскольку оно доступно только для записи.  
  
### Исправление ошибки  
  
1.  Проверьте свойство и определите, при каких условиях возможно его задание.  
  
2.  Удалите ссылку на это свойство.
