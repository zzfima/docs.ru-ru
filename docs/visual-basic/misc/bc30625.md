# <a name="module-statement-must-end-with-a-matching-end-module"></a>Оператор Module должен заканчиваться соответствующим оператором End Module
Блоки модулей должны заканчиваться оператором `End Module` .  
  
 **Идентификатор ошибки:** BC30625  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте оператор `End Module` в конец блока `Module` .
