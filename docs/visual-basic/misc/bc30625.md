# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39; Модуль &#39; оператор должен заканчиваться соответствующим &#39; End Module &#39;
Блоки модулей должны заканчиваться оператором `End``Module` .  
  
 **Идентификатор ошибки:** BC30625  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте оператор `End Module` в конец блока `Module` .
