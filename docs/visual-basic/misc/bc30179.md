# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;Тип 1&gt; &#39;&lt; TypeName&gt;&#39; и &lt;тип2&gt; &#39;&lt; TypeName&gt;&#39; конфликта в &lt;имен&gt; &#39;&lt; Имя пространства имен&gt;&#39;
Два объявленных элемента, например класс и модуль, имеют одинаковые имена в одном пространстве имен.  
  
 **Идентификатор ошибки:** BC30179  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переместите объявление одного из элементов в другое пространство имен.  
  
-   Измените имя одного из элементов.
