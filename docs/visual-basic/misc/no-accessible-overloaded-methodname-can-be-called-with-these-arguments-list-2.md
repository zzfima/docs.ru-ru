# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion-ltlistgt"></a>Ни один из доступных перегруженных &#39; &lt;имя_метода&gt;&#39; может вызываться с этими аргументами без преобразования с сужением: &lt;списка&gt;
Вызван перегруженный метод, однако он не может быть сопоставлен со списком указанных аргументов без сужающего преобразования.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Укажите `Option``Strict` `Off`.  
  
2.  Измените аргументы таким образом, чтобы они соответствовали сигнатуре перегруженного метода.
