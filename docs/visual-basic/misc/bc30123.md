# <a name="39end39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Конец &#39; операторы недопустимы в окне интерпретации
Операторы `Stop` и `End` приостанавливают выполнение и не разрешены в контексте отладки.  
  
 **Идентификатор ошибки:** BC30123  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Не используйте операторы `End` и `Stop` в окне **интерпретации** .
