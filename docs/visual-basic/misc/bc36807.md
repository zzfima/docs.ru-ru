<span data-ttu-id="12c42-103">На дочерний элемент XML имеется ссылка для получения объекта, который не относится к типу <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>или `IEnumerable(Of XElement)`.</span><span class="sxs-lookup"><span data-stu-id="12c42-103">An XML child element has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="12c42-104">Для получения дополнительной информации см.
