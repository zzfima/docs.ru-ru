# Преобразование в &quot;&lt;тип&gt;&quot; невозможно
Предпринята попытка недопустимого преобразования.  
  
 **Идентификатор ошибки:** BC30748  
  
### Исправление ошибки  
  
-   Выберите совместимый тип преобразования.
