# <a name="structure-39ltstructurenamegt39-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Структура &#39; &lt;имя_структуры&gt;&#39; должен содержать по крайней мере один экземпляр переменной-члена или объявление событий
Обнаружена функционально пустая структура. Между оператором `Structure` и оператором `End Structure` должен быть объявлен по крайней мере один член-переменная.  
  
 **Идентификатор ошибки:** BC30281  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте оператор `Dim` или `Event` в структуру.
