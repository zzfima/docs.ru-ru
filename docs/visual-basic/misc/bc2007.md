# нераспознанный параметр &lt;имя_параметра&gt;; проигнорирован
В командной строке указан недопустимый параметр.  
  
 **Идентификатор ошибки:** BC2007  
  
### Исправление ошибки  
  
-   Проверьте указанные параметры, чтобы найти недопустимый.
