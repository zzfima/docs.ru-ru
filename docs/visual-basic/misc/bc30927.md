# <a name="mustoverride-cannot-be-specified-on-procedurename-because-it-is-in-a-partial-type-that-is-declared-notinheritable-in-another-partial-definition"></a><span data-ttu-id="55c10-102">«MustOverride» не может указываться для "\<имя_процедуры >", так как он находится в разделяемом типе, объявленном в другом частичном определении «NotInheritable»</span><span class="sxs-lookup"><span data-stu-id="55c10-102">'MustOverride' cannot be specified on '\<procedurename>' because it is in a partial type that is declared 'NotInheritable' in another partial definition</span></span>
<span data-ttu-id="55c10-103">Процедура или свойство объявляется как `MustOverride` в классе, который определен в нескольких частичных объявлениях, но одно из частичных определений задает `NotInheritable` для этого класса.</span><span class="sxs-lookup"><span data-stu-id="55c10-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="55c10-104">После разделения определения класса на несколько частичных объявлений компилятор обрабатывает класс как объединение всех частичных объявлений.</span><span class="sxs-lookup"><span data-stu-id="55c10-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="55c10-105">Это относится не только к членам, но и к реализации, наследованию и уровню доступа.</span><span class="sxs-lookup"><span data-stu-id="55c10-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="55c10-106">Чтобы переопределить процедуру или свойство, класс должен унаследовать этот объект от базового класса.</span><span class="sxs-lookup"><span data-stu-id="55c10-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="55c10-107">Таким образом, чтобы указать `MustOverride` для процедуры или свойства в базовом классе, необходимо указать `MustInherit` для этого класса.</span><span class="sxs-lookup"><span data-stu-id="55c10-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="55c10-108">Поскольку эти ключевые слова противоречат друг другу, вы не можете указать и `MustInherit` , и `NotInheritable` для одного и того же класса.</span><span class="sxs-lookup"><span data-stu-id="55c10-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="55c10-109">**Идентификатор ошибки:** BC30927</span><span class="sxs-lookup"><span data-stu-id="55c10-109">**Error ID:** BC30927</span></span>
