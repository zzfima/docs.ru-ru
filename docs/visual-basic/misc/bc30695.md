# <a name="lttypenamegt-39ltmembernamegt39-must-be-declared-39shadows39-because-another-member-with-this-name-is-declared-39shadows39"></a>&lt;TypeName&gt; &#39; &lt;membername&gt; &#39; должен быть объявлен &#39;Shadows&#39; поскольку другой член с таким именем объявлен &#39;тени&#39;
Это свойство или процедура имеет то же имя, что и другое свойство или процедура, помеченная модификатором `Shadows` .  
  
 **Идентификатор ошибки:** BC30695  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте модификатор `Shadows` .
