# <a name="argument-ltargumentnamegt-must-be-greater-than-or-equal-to--1"></a>Аргумент "&lt;имя_аргумента&gt;" должно быть больше или равно-1
Значение аргумента меньше -1.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь в том, что значение аргумента находится в допустимом диапазоне.  
  
2.  Если аргумент является результатом вычисления, проверьте значения, обрабатываемые при вычислении.
