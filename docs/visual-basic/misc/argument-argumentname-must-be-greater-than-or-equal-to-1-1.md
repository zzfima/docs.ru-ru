# Аргумент &quot;&lt;имя_аргумента&gt;&quot; должен быть больше или равен -1
Значение аргумента меньше \-1.  
  
### Исправление ошибки  
  
1.  Убедитесь, что значение аргумента находится в допустимом диапазоне.  
  
2.  Если аргумент является результатом вычисления, проверьте значения, обрабатываемые при вычислении.
