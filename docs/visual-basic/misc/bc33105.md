# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="86229-102">Операнды If не могут быть именованными аргументами</span><span class="sxs-lookup"><span data-stu-id="86229-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="86229-103">Использование именованных аргументов в операндах оператора `If` недопустимо.</span><span class="sxs-lookup"><span data-stu-id="86229-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="86229-104">Следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="86229-104">The following example causes this error:</span></span>  
  
```  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="86229-105">Это отличается от функции `IIf` , которая разрешает именованные аргументов, как показано в следующем коде:</span><span class="sxs-lookup"><span data-stu-id="86229-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="86229-106">**Идентификатор ошибки:** BC33105</span><span class="sxs-lookup"><span data-stu-id="86229-106">**Error ID:** BC33105</span></span>

-   <span data-ttu-id="86229-108">Удалите назначения имен из операндов, как показано в следующем коде.</span><span class="sxs-lookup"><span data-stu-id="86229-108">Remove the name assignments from the operands, as shown in the following code.</span></span>  
  
    ```  
    result = If(i > 0, "positive", "not positive")  
    ```
