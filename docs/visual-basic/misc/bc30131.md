# Атрибут безопасности &quot;&lt;имя_атрибута&gt;&quot; не может использоваться для модуля
Среда CLR не принимает атрибут безопасности для модуля.  
  
 **Идентификатор ошибки:** BC30131  
  
### Исправление ошибки  
  
1.  Удалите атрибут из объявления.
