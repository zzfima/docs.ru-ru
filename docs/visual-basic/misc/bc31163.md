# Ожидалась соответствующая закрывающая кавычка для значения атрибута XML
В значении атрибута, которое разделяется одинарными кавычками, отсутствует закрывающая кавычка.  
  
 **Идентификатор ошибки:** BC31163  
  
### Исправление ошибки  
  
-   Добавьте закрывающую одинарную кавычку \('\), чтобы отделить значение атрибута.
