# <a name="nullable-types-are-not-supported-because-the-current-version-of-39mscorlibdll39-does-not-define-the-type-39nullableof-t39"></a>Допускающие значение NULL типы не поддерживаются, так как текущая версия &#39;mscorlib.dll &#39; не определен тип &#39; Nullable(Of T) &#39;
Используемая версия платформы не поддерживает типы, допускающие значение NULL.  
  
 **Идентификатор ошибки:** BC32130
