# <a name="class-not-registered-on-local-machine"></a>Класс не зарегистрирован на локальном компьютере
Эта ошибка обычно возникает из-за поврежденного файла. Может потребоваться переустановка [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переустановите [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].
