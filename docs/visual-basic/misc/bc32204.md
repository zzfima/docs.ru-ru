# <a name="event-name-length-cannot-exceed-1011-characters"></a>Длина имени события не может превышать 1011 символов.
Имя или идентификатор каждого события, объявленного с помощью оператора `Event` , ограничено 1011 знаками.  
  
 **Идентификатор ошибки:** BC32204  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Уменьшите длину идентификатора.
