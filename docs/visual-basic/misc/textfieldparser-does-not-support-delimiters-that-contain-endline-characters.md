# TextFieldParser не поддерживает разделители, содержащие символы конца строки
Указан разделитель, содержащий символ конца строки, но `TextFieldParse`r не поддерживает разделители, содержащие символ конца строки.  
  
### Исправление ошибки  
  
-   Укажите правильный разделитель.
