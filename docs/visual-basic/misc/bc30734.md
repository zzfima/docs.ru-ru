# &quot;&lt;имя&gt;&quot; уже объявлен как параметр этого метода
Указанное имя параметра уже объявлено для этого метода.  
  
 **Идентификатор ошибки:** BC30734  
  
### Исправление ошибки  
  
-   Измените имя параметра, чтобы оно было уникальным.
