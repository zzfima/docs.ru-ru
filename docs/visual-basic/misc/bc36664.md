# <a name="nesting-of-parentheses-in-lambda-expressions-parameter-list-exceeds-maximum-allowable-depth"></a>Вложенность скобок в параметре лямбда-выражения превышает максимально допустимую глубину
Вложенные скобки в списке параметров лямбда-выражения не могут превышать глубину 128.  
  
 **Идентификатор ошибки:** BC36664
