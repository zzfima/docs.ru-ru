# Спецификатор параметра встречается второй раз
Спецификатор параметра применяется к параметру несколько раз.  
  
 **Идентификатор ошибки:** BC30785  
  
### Исправление ошибки  
  
-   Удалите повторяющиеся спецификаторы.
