# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>Целевая версия .NET Compact Framework не поддерживает &#39; Конец &#39; инструкции
Версия .NET Compact Framework, с которой вы работаете, не поддерживает оператор `End` .  
  
 **Идентификатор ошибки:** BC30769  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите оператор.
