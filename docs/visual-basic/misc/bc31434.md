# <a name="method-39ltmethodname1gt39-cannot-implement-partial-method-39ltmethodname2gt39-because-39ltmethodname3gt39-already-implements-it"></a>Метод &#39; &lt;имя_метода1&gt;&#39; не может реализовывать разделяемый метод &#39;&lt; имя_метода2&gt;&#39; так как &#39;&lt; имя_метода3&gt;&#39; уже реализует его
Метод "\<имя_метода1 >" не может реализовать частичный метод "\<имя_метода2 >" из-за "\<имя_метода3 >" уже реализует его. Только один метод может реализовывать разделяемый метод.  
  
 Нельзя иметь два разделяемых метода, реализующих одно и то же объявление разделяемого метода. Следующий код вызывает эту ошибку.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub ValueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' First implementation of the partial method.  
    Private Sub ValueChanged()  
        MsgBox(Value was changed to " & Me.Quantity)  
    End Sub  
  
    ' Second implementation of the partial method causes this error.  
    'Private Sub ValueChanged()  
    '    Console.WriteLine("Quantity was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **Идентификатор ошибки:** BC31434
