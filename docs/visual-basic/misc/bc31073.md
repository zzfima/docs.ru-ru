# <a name="39ltname1gt39-conflicts-with-a-lttypegt-by-the-same-name-defined-in-39ltname2gt39"></a>&#39; &lt;name1&gt;&#39; конфликтует с &lt;тип&gt; с тем же именем, определенные в &#39;&lt; Name2&gt;&#39;
Обнаружено имя, конфликтующее с типом с тем же именем.  
  
 **Идентификатор ошибки:** BC31073  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Объявите уникальное имя.  
  
2.  Заключите имя конфликтующего элемента в квадратные скобки.
