# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>События, объявленные с &#39; Как &#39; Предложение должно иметь тип делегата
Указанный тип не является делегатом.  
  
 **Идентификатор ошибки:** BC31044  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Удалите предложение `As` .  
  
2.  Укажите делегат с помощью предложения `As` .
