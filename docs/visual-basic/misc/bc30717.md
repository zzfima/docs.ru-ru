# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>Операторы 'Try' недопустимы в окне интерпретации
`Try`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30717  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `Try` из отлаживаемого кода.
