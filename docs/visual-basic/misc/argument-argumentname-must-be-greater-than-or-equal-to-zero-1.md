# Аргумент &lt;имя_аргумента&gt; должен быть больше или равен нулю
Значение аргумента меньше нуля. Значения аргумента должны быть больше или равны нулю.  
  
### Исправление ошибки  
  
1.  Убедитесь в том, что значение аргумента больше или равно нулю.  
  
2.  Если аргумент является результатом вычисления, проверьте значения, обрабатываемые при вычислении.
