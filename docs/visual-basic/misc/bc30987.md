# <a name="-expected"></a><span data-ttu-id="8c421-102">Требуется символ "{"</span><span class="sxs-lookup"><span data-stu-id="8c421-102">'{' expected</span></span>
<span data-ttu-id="8c421-103">Чтобы объявить экземпляр именованного или анонимного типа с помощью инициализатора объектов, необходимо заключить список полей или свойств и их начальные значения в фигурные скобки ({ }).</span><span class="sxs-lookup"><span data-stu-id="8c421-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>  
  
```vb  
Dim client As New Customer() With {.Name = "Microsoft", .City = "Seattle"}  
Dim emp = New Employee() With {.Name = "Rob Young", .ID = 55555}  
Dim anon = New With {.ID = 123456}  
```
