# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39;&lt;TypeName&gt; &#39; является неподдерживаемым типом
Указанный тип не может быть представлен Visual Studio.  
  
 **Идентификатор ошибки** : BC30649  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте поддерживаемый тип.  
  
-   Разработайте новый тип.
