<span data-ttu-id="4f83f-103">Когда предоставляются параметры в объявлении и реализации разделяемого метода, имена соответствующих параметров должны совпадать.</span><span class="sxs-lookup"><span data-stu-id="4f83f-103">When parameters are supplied for the declaration and implementation of a partial method, the names of the corresponding parameters must be the same.</span></span> <span data-ttu-id="4f83f-104">Например, следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="4f83f-104">For example, the following code causes this error.</span></span>

    Partial Private Sub valueChanged(ByVal newVal As Integer)

    ' reported for parameter val.

    ' End Sub  
  
End Class
