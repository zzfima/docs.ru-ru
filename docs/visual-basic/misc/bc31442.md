# <a name="parameter-name-39ltparametername1gt39-does-not-match-the-name-of-the-corresponding-parameter-39ltparametername2gt39-defined-on-the-partial-method-declaration-39ltmethodnamegt39"></a>Имя параметра &#39; &lt;имя_параметра_типа1&gt; &#39; не соответствует имени соответствующего параметра &#39; &lt;имя_параметра_типа2&gt;&#39;, определенного в объявлении разделяемого метода &#39; &lt;имя_метода&gt;&#39;
Когда предоставляются параметры в объявлении и реализации разделяемого метода, имена соответствующих параметров должны совпадать. Например, следующий код вызывает эту ошибку.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged(ByVal newVal As Integer)  
    End Sub  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method. This error is  
    ' reported for parameter val.  
    ' Private Sub valueChanged(ByVal val As Integer)  
    '     MsgBox(Value was changed to " & Me.Quantity)  
    ' End Sub  
  
End Class  
```  
  
 **Идентификатор ошибки:** BC31442  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Измените имя параметра или имена параметров в объявлении или реализации так, чтобы соответствующие параметры имели одинаковые имена.
