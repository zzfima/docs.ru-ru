# <a name="sub-main-is-declared-more-than-once-in-ltnamespacegt-ltmessagegt"></a>«Sub Main» объявлен более одного раза в "&lt;пространства имен&gt;": &lt;сообщения&gt;
`Sub Main` можно объявить в пространстве имен только один раз.  
  
 **Идентификатор ошибки:** BC30738  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Убедитесь, что в проекте имеется только одна процедура `Sub Main` .
