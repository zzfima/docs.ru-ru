# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a><span data-ttu-id="84f8b-102">Метод\<'методимя1>' должен быть объявлен «частным»\<для реализации частичного метода «метод2>»</span><span class="sxs-lookup"><span data-stu-id="84f8b-102">Method '\<methodname1>' must be declared 'Private' in order to implement partial method '\<methodname2>'</span></span>
<span data-ttu-id="84f8b-103">Реализация разделяемого метода должна быть объявлена как `Private`.</span><span class="sxs-lookup"><span data-stu-id="84f8b-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="84f8b-104">Например, следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="84f8b-104">For example, the following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub

```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,
    ' causes this error.
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 <span data-ttu-id="84f8b-105">**Идентификатор ошибки:** BC31441</span><span class="sxs-lookup"><span data-stu-id="84f8b-105">**Error ID:** BC31441</span></span>

```vb  
End Sub  
```
