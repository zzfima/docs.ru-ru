# <a name="ltspecifiergt-is-not-valid-on-an-enum-declaration"></a>"&lt;описатель&gt;" является недопустимым объявлением перечисления
Использовался спецификатор, недопустимый в объявлении `Enum` .  
  
 **Идентификатор ошибки:** BC30396  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Удалите спецификатор.
