# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39;&lt;математическая_функция1&gt; &#39; не объявлена
"\<математическая_функция1 >" не объявлен. Функция была перемещена в класс System.Math и теперь называется "\<математическая_функция2 >".  
  
 Несколько функций, которые были встроенными в Visual Basic в предыдущих версиях, были перемещены в <xref:System.Math?displayProperty=nameWithType> пространства имен. Это делает их функциональность более доступной для всех языков программирования.  
  
 **Идентификатор ошибки:** BC30819  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте методы, объявленные в <xref:System.Math?displayProperty=nameWithType>.
