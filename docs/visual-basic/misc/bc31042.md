# Sub New не может реализовывать члены интерфейса
`Sub New` представляет собой конструктор и не может реализовывать члены.  
  
 **Идентификатор ошибки:** BC31042  
  
### Исправление ошибки  
  
-   Удалите операторы `Implements` из процедур `Sub New`.
