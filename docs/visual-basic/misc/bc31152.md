# Префикс пространства имен XML &quot;префикс&quot; зарезервирован для использования в XML, поэтому не удается изменить URI пространства имен.
В XML\-литерале использовался зарезервированный префикс пространства имен XML, такой как `xmlns`.  
  
 **Идентификатор ошибки:** BC31152  
  
### Исправление ошибки  
  
-   Измените префикс пространства имен XML на не зарезервированное значение.
