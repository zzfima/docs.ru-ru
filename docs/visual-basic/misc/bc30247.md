# <a name="39ltmodifiergt39-is-not-valid-on-a-local-variable-declaration"></a>&#39; &lt;модификатор&gt;&#39; является недопустимым в объявлении локальной переменной
Оператор `Dim` в процедуре содержит недопустимое ключевое слово, например `Overloads`.  
  
 **Идентификатор ошибки:** BC30247  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите недопустимое ключевое слово из инструкции `Dim` .
