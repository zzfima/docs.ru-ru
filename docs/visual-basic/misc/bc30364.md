# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39;Sub New&#39; не может объявляться как &#39; &lt;модификатор&gt;&#39;
Вы попытались объявить `Sub New` с модификатором, таким как `Overrides`, который недопустим.  
  
 **Идентификатор ошибки:** BC30364  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите этот модификатор.
