# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39; EndIf &#39; операторы больше не поддерживаются; использовать &#39; End If &#39; Вместо этого
Оператор `EndIf` больше не поддерживается.  
  
 **Идентификатор ошибки:** BC30826  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте `End If` для завершения операторов `If` .
