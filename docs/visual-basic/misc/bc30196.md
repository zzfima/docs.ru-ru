# <a name="comma-expected"></a>Ожидалась запятая
В списке аргументов отсутствует запятая (,) между двумя аргументами.  
  
 **Идентификатор ошибки:** BC30196  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Убедитесь, что все аргументы разделены запятыми.
