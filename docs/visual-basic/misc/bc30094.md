# <a name="label-39ltlabelnamegt39-is-already-defined-in-the-current-methodmultiline-lambda"></a>Метка &#39; &lt;имя_метки&gt;&#39; уже определена в текущей метода/многострочного лямбда-выражения
Имя метки повторяется в рамках процедуры. Все имена в процедуре должны быть уникальными.  
  
 **Идентификатор ошибки:** BC30094  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Переименуйте повторяющуюся метку.
