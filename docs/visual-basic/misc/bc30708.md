# Операторы цикла недопустимы в окне интерпретации
Операторы `Loop` нельзя использовать в окне **интерпретации**.  
  
 **Идентификатор ошибки:** BC30708  
  
### Исправление ошибки  
  
-   Удалите операторы `Loop` из отлаживаемого кода.
