# <a name="ltassemblynamegt-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>"&lt;assemblyname&gt;" является сборкой и не может быть в ссылках как модуль
Предпринята попытка использовать сборку в качестве модуля.  
  
 **Идентификатор ошибки:** BC31077  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените ссылку на модуль.
