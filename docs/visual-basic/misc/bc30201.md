# Ожидалось выражение
Оператор, такой как `+` или `=`, требует выражения для правого операнда, однако распознаваемое выражение отсутствует.  
  
 **Идентификатор ошибки:** BC30201  
  
### Исправление ошибки  
  
-   Укажите допустимое выражение для правого операнда оператора.
