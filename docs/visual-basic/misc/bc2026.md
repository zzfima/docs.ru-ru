# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Номер предупреждения &#39; &lt;номер&gt; &#39; для параметра &#39; &lt;имя_параметра&gt; &#39; не настраивается или не является допустимым
Идентификатор предупреждения является недопустимым. Идентификатор должен быть целым числом больше 0.  
  
 **Идентификатор ошибки:** BC2026  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте, является ли указанный идентификатор предупреждения допустимым.
