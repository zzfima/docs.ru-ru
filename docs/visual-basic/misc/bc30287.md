# Требуется &quot;.&quot;
В коде есть предложение `Handles`, содержащее восклицательный знак \(`!`\).  
  
 **Идентификатор ошибки:** BC30287  
  
### Исправление ошибки  
  
1.  Если предложение `Handles` ссылается на событие в объекте, используйте точку \(`.`\) для отделения объекта от события.
