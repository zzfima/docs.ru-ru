# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-number-characters"></a>Полные имена для типов в пространствах имен не может быть длиннее \<число > символов
Полное имя типа не может быть длиннее, чем указанное количество символов.  
  
 **Идентификатор ошибки:** BC30031  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укоротите имя типа.
