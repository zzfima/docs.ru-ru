# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>Полные имена типов в пространства имен не может быть длиннее &lt;номер&gt; символов
Полное имя типа не может быть длиннее, чем указанное количество символов.  
  
 **Идентификатор ошибки:** BC30031  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укоротите имя типа.
