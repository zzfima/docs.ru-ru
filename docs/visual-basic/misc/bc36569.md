# Не задан аргумент для параметра &quot;&lt;имя_параметра&gt;&quot;
Аргумент для обязательного параметра не указан.  
  
 **Идентификатор ошибки:** BC36569  
  
### Исправление ошибки  
  
-   Предоставьте аргумент для указанного параметра.
