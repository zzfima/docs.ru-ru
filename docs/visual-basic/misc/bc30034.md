# У идентификатора в квадратных скобках отсутствует закрывающая скобка &quot;]&quot;.
Квадратные скобки в имени должны быть парными.  
  
 **Идентификатор ошибки:** BC30034  
  
### Исправление ошибки  
  
-   Поместите закрывающую скобку в конце имени.
