<span data-ttu-id="3a098-103">Глобальный атрибут задан без указания того, применяется ли он для всей сборки или только для текущего модуля.</span><span class="sxs-lookup"><span data-stu-id="3a098-103">A global attribute is specified without indicating whether it applies to the entire assembly or only to the current module.</span></span> <span data-ttu-id="3a098-104">Для глобальных атрибутов необходимо указать `Assembly` или `Module`.</span><span class="sxs-lookup"><span data-stu-id="3a098-104">Global attributes must specify either `Assembly` or `Module`.</span></span>
