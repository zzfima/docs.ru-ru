# Операторы Error недопустимы в окне интерпретации
Операторы `Error` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30724  
  
### Исправление ошибки  
  
-   Удалите операторы `Error` из отлаживаемого кода.
