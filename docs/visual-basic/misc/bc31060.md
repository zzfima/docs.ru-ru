# <a name="39ltpropertynamegt39-implicitly-defines-39ltmembernamegt39-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-39lttypenamegt39"></a>&#39; &lt;propertyname&gt;&#39; неявно определяет &#39;&lt; имя пользователя&gt;&#39; который конфликтует с членом с тем же именем в &lt;тип&gt; &#39;&lt; TypeName&gt;&#39;
Имя члена типа конфликтует с именем члена, созданного неявно для свойства. Свойства неявно создают несколько неявных переменных, которые могут конфликтовать с определенными членами.  
  
 **Идентификатор ошибки:** BC31060  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переименуйте явно объявленный член, чтобы устранить конфликт имен.
