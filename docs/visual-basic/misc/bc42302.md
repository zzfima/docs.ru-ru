# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>Комментарий XML должен быть первым оператором в строке
Комментарий XML должен быть первым оператором в строке. Комментарий XML будет проигнорирован.  
  
 Комментарий XML помещен после другого элемента языка, что не допускается.  
  
 **Идентификатор ошибки:** BC42302  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Переместите комментарий XML на новую строку.
