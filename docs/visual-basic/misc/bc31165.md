# Ожидался начальный элемент &quot;&lt;&quot; для тега XML
В XML\-теге XML\-литерала отсутствует требуемая открывающая скобка "\<".  
  
 **Идентификатор ошибки:** BC31165  
  
### Исправление ошибки  
  
-   Добавьте символ "\<" в начало XML\-тега для XML\-литерала.
