# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39;&lt;имя&gt; &#39; неоднозначен для объявлений в модулях &#39; &lt;имя_модуля1&gt; &#39; и &#39; &lt;имя_модуля2&gt;&#39;
Вы указали неоднозначное имя, которое конфликтует с другим именем. Компилятор Visual Basic не имеет правил разрешения конфликтов; Вы должны самостоятельно устранить неоднозначность имен.  
  
 **Идентификатор ошибки:** BC30562  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите полное имя.
