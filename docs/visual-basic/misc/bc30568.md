# Инициализатор массива имеет слишком много элементов &lt;число&gt; 
Инициализатор массива содержит слишком много элементов.  
  
 **Идентификатор ошибки:** BC30568  
  
### Исправление ошибки  
  
-   Используйте `ReDim` для изменения размера массива.
