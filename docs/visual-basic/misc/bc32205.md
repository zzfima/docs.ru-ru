# Длина имени переменной, объявленной с модификатором WithEvents, не может превышать 1019 знаков
Имена или идентификаторы переменных, объявленных с атрибутом `WithEvents`, ограничены 1019 знаками.  
  
 **Идентификатор ошибки:** BC32205  
  
### Исправление ошибки  
  
-   Уменьшите длину идентификатора.
