# <a name="paramarray-parameter-must-be-an-array"></a>Параметр ParamArray должен быть массивом
Аргумент `ParamArray` должен быть объявлен как тип одномерного массива. В объявлении отсутствуют круглые скобки после имени аргумента `ParamArray` .  
  
 **Идентификатор ошибки:** BC30050  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте пару скобок после имени аргумента `ParamArray` .
