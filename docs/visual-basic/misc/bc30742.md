# <a name="value-39ltvaluename1gt39-cannot-be-converted-to-39ltvaluename2gt39"></a>Значение &#39; &lt;имя_значения1&gt;&#39; невозможно преобразовать в &#39;&lt; имя_значения2&gt;&#39;
Предпринята попытка преобразования между двумя несовместимыми типами.  
  
 **Идентификатор ошибки:** BC30742  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте преобразуемые типы.
