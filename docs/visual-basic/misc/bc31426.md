# Имена типов, отличных от встроенных, не допустимы в выражениях условной компиляции
Выражение условной компиляции ссылается на невстроенный тип, а допустимы только встроенные типы.  
  
 Дополнительные сведения о встроенных типах в [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] см. в разделе [Сводка по типам данных](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **Идентификатор ошибки:** BC31426  
  
### Исправление ошибки  
  
-   Замените невстроенный тип на встроенный.
