# <a name="the-file-ltfilenamegt-is-not-a-text-file"></a>Файл "&lt;filename&gt;" не является файлом
Запись в указанный файл невозможна, так как это двоичный файл.  
  
 **Идентификатор ошибки:** BC2015  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Убедитесь в том, что вы указали правильное имя файла и путь к нему.
