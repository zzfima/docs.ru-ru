# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>Проект "\<имя_проекта >" не может ссылаться на проект "\<имя_проекта >" так как "\<имя_проекта >" напрямую или косвенно ссылается на "\<имя_проекта >"
Проект ссылается на второй проект. Второй проект, в свою очередь, содержит ссылку на проект, в котором на него есть ссылка.  
  
 **Идентификатор ошибки:** BC30761  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите одну из ссылок.
