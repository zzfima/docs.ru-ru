# <a name="39ltnamegt39-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-39lttypenamegt39-through-ltnamegt"></a>&#39; &lt;имя&gt;&#39; не может представлять тип &lt;тип1&gt; в &lt;описатель&gt; &lt;тип2&gt; &#39;&lt; TypeName&gt;&#39; с помощью &lt;имя&gt;
Предпринята попытка предоставить закрытый тип за пределами открытого класса `MyClass`.  
  
 **Идентификатор ошибки:** BC30508  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Объявите тип `Public`.
