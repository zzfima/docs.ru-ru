# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>"&lt;имя&gt;" не может представлять тип &lt;тип1&gt; в &lt;описатель&gt; &lt;тип2&gt; "&lt;typename&gt;" через &lt;имя&gt;
Предпринята попытка предоставить закрытый тип за пределами открытого класса `MyClass`.  
  
 **Идентификатор ошибки:** BC30508  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Объявите тип `Public`.
