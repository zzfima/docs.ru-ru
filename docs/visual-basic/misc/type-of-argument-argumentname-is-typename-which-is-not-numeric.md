# Тип аргумента &quot;&lt;имя_аргумента&gt;&quot;&#160;— &lt;имя_типа&gt;, который не является числовым
Вместо ожидаемого числового аргумента используется нечисловой.  
  
### Исправление ошибки  
  
-   Убедитесь в том, что аргумент является числовым.
