# Операторы Wend больше не поддерживаются; используйте операторы End While
Конструкция `End While` заменяет ключевое слово `Wend`.  
  
 **Идентификатор ошибки:** BC30809  
  
### Исправление ошибки  
  
-   Используйте `End While` для завершения блока `While`.
