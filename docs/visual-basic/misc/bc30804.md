# <a name="39variant39-is-no-longer-a-supported-type-use-the-39object39-type-instead"></a>&#39; Variant &#39; больше не является поддерживаемым типом; использовать &#39; объект &#39; Вместо этого тип
Тип данных `Object` заменяет `Variant`.  
  
 **Идентификатор ошибки:** BC30804  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Объявите элемент как `Object`.
