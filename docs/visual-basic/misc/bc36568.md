# <a name="39ltelementnamegt39-is-not-a-method-parameter"></a>&#39;&lt;ElementName&gt; &#39; не является параметром метода
В вызове процедуры указывается дополнительный аргумент, который не определен методом.  
  
 **Идентификатор ошибки:** BC36568  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите лишний аргумент из списка аргументов.
