# <a name="error-setting-assembly-manifest-option-lterror-messagegt"></a>Ошибка при задании параметра манифеста сборки: &lt;сообщение об ошибке&gt;
Компилятор Visual Basic вызывает компоновщик сборок (Al.exe, который также называется Alink) для создания сборки с манифестом. Компоновщик сообщил об ошибке поиска файла ключа или имени ключа, используемого для подписи сборки.  
  
 **Идентификатор ошибки:** BC30139  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Изучите приведенное сообщение об ошибке и обратитесь к ошибок, перечисленных на [Al.exe (компоновщик сборок)](../../framework/tools/al-exe-assembly-linker.md) для пояснения и рекомендации.  
  
2.  Если ошибка не устранена, соберите сведения об условиях ее возникновения и уведомите службу технической поддержки Майкрософт.
