# Целевая версия .NET Compact Framework не поддерживает разрешение перегрузки позднего связывания
Целевая версия .NET Compact Framework не поддерживает разрешение перегрузки позднего связывания.  
  
 **Идентификатор ошибки:** BC30764  
  
### Исправление ошибки  
  
-   Приведите параметры типа Object к соответствующему типу.
