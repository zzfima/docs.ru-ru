# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>&#39; &lt;variablename&gt;&#39; не является локальной переменной или параметра и поэтому не может быть использована как &#39; CATCH &#39; переменная
Переменные в операторах `Try...Catch...Finally` должны быть объявлены локально или представлять собой параметры текущей процедуры.  
  
 **Идентификатор ошибки:** BC31082  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Объявите локальные переменные или параметры для операторов `Try...Catch...Finally` .
