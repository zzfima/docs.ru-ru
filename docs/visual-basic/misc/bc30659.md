# <a name="property-or-field-name-does-not-have-a-valid-attribute-type"></a>Свойство или поле "\<имя >" не имеет атрибута допустимого типа
Возможно, вы используете конструктор атрибута, который не принимает является допустимым типом атрибута, несмотря на то, что у атрибута могут быть иные подходящие конструкторы.  
  
 **Идентификатор ошибки:** BC30659  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте использование данного атрибута.
