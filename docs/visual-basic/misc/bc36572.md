# Параметр типа &lt;имя_параметра_типа&gt; не может быть определен
Универсальная процедура вызывается без указания списка аргументов типов, и для одного из аргументов типа не удается определить тип.  
  
 Как правило, при вызове универсальной процедуры указывается аргумент типа для каждого параметра типа, определяемого этой процедурой. Тем не менее существует возможность полностью опустить список аргументов типов. При этом компилятор пытается определить тип каждого аргумента типа из контекста вызова. Дополнительные сведения см. в подразделе "Определение типа" в разделе [Универсальные процедуры в Visual Basic](../../visual-basic/programming-guide/language-features/data-types/generic-procedures.md).  
  
 **Идентификатор ошибки:** BC36572  
  
### Исправление ошибки  
  
-   Убедитесь, что типы нормальных аргументов таковы, что определение типа согласуется с параметрами типов, объявленными для универсальной процедуры.  
  
     \-или\-  
  
-   Вызовите универсальную процедуру с полным списком аргументов типов, чтобы определение типа не требовалось.
