# <a name="join-expected"></a><span data-ttu-id="df71b-102">Требуется "Join"</span><span class="sxs-lookup"><span data-stu-id="df71b-102">'Join' expected</span></span>
<span data-ttu-id="df71b-103">Предложение `Group Join` было указано без ключевого слова `Join` .</span><span class="sxs-lookup"><span data-stu-id="df71b-103">A `Group Join` clause has been specified without a `Join` keyword.</span></span>  
  
 <span data-ttu-id="df71b-104">**Идентификатор ошибки:** BC36631</span><span class="sxs-lookup"><span data-stu-id="df71b-104">**Error ID:** BC36631</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="df71b-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="df71b-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="df71b-106">Добавьте ключевое слово `Join` в предложение `Group Join` , как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="df71b-106">Add the `Join` keyword to the `Group Join` clause, as shown in the following example:</span></span>

```vb  
            Group Join var3 In collection3 _

            Select JoinID = var1, var2.Name, Matches  
```
