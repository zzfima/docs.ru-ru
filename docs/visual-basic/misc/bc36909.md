<span data-ttu-id="10025-103">Если размерность, используемая для инициализации массива, не соответствует размерности в объявлении, компилятор не может вывести тип данных для массива.</span><span class="sxs-lookup"><span data-stu-id="10025-103">If the dimensions used to initialize an array do not match the dimensions in the declaration, the compiler cannot infer a data type for the array.</span></span> <span data-ttu-id="10025-104">Например, следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="10025-104">For example, the following code causes this error.</span></span>

' Not valid.  
'Dim exampleArray2(,) = New Integer() {1, 2, 3}
