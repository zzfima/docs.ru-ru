# <a name="conflicts-with-39ltname1gt39-which-is-implicitly-created-for-39ltname2gt39-in-lttypegt-39ltname3gt39"></a>Конфликтует с &#39; &lt;name1&gt;&#39; который неявно создан для &#39;&lt; Name2&gt;&#39; в &lt;тип&gt; &#39;&lt; Name3&gt;&#39;
Имя члена типа конфликтует с другим неявно созданным членом.  
  
 **Идентификатор ошибки:** BC31058  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переименуйте явно объявленный член, чтобы устранить конфликт имен.
