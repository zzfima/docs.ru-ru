# <a name="xml-name-expected"></a>Требуется XML-имя
XML-литерал или свойство оси XML указано без имени.  
  
 **Идентификатор ошибки:** BC31146  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Добавьте имя для XML-литерала или свойства оси XML.
