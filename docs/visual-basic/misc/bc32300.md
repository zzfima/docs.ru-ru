# Объект больше не существует из-за ошибки компиляции или удаления
В компиляторе [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] произошла ошибка доступа к элементу кода.  
  
 **Идентификатор ошибки:** BC32300  
  
### Исправление ошибки  
  
1.  Скомпилируйте программу еще раз, чтобы узнать, повторится ли ошибка.  
  
2.  Если устранить ошибку не удается, переустановите компилятор [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
3.  Если ошибка не устранена после переустановки, соберите сведения об условиях ее возникновения и уведомите службу технической поддержки Майкрософт.
