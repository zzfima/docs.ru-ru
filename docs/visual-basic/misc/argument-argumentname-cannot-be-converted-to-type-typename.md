# Не удается преобразовать аргумент &quot;&lt;имя_аргумента&gt;&quot; в тип &quot;&lt;имя_типа&gt;&quot;.
Преобразование пытается изменить аргумент на тип, который является недопустимым.  
  
### Исправление ошибки  
  
-   Измените тип на допустимый.
