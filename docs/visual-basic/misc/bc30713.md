# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>Операторы «Exit» недопустимы в окне интерпретации
`Exit`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30713  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `Exit` из отлаживаемого кода.
