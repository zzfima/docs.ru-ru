# Операторы Exit недопустимы в окне интерпретации
Операторы `Exit` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30713  
  
### Исправление ошибки  
  
-   Удалите операторы `Exit` из отлаживаемого кода.
