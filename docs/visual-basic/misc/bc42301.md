# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Только один блок XML-комментариев допускается для каждого элемента языка
К элементу языка применено больше одного блока XML-комментариев.  
  
 **Идентификатор ошибки:** BC42301  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите лишний блок XML-комментариев.
