# &quot;&lt;член&gt;&quot; уже объявлен в этой структуре
Член перечисления объявлен в этой структуре более одного раза.  
  
 **Идентификатор ошибки:** BC31421  
  
### Исправление ошибки  
  
-   Удалите лишнее объявление.
