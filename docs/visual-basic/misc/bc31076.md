# <a name="39ltmodulenamegt39-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>&#39; &lt;modulename&gt;&#39; является модулем и невозможно обращаться как к сборке
Предпринята попытка использовать модуль в качестве сборки.  
  
 **Идентификатор ошибки:** BC31076  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените ссылку на сборку.
