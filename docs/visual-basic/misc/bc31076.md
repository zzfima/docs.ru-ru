# <a name="ltmodulenamegt-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>"&lt;modulename&gt;" является модулем и не может обращаться как к сборке
Предпринята попытка использовать модуль в качестве сборки.  
  
 **Идентификатор ошибки:** BC31076  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените ссылку на сборку.
