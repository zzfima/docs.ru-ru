# Требуется &quot;Next&quot;
Конструкция `On Error Resume` используется без завершающего ключевого слова `Next`.  
  
 **Идентификатор ошибки:** BC30003  
  
### Исправление ошибки  
  
-   Добавьте ключевое слово `Next` в конец оператора `On Error`.
