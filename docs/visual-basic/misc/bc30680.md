# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39; #End region &#39; должен предшествовать соответствующий &#39; #Region &#39;
С помощью `#Region` вы можете указать блок кода, который можно разворачивать и сворачивать при использовании функции структурирования редактора кода [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] . Начало и конец операторов `#Region` должны находиться в одном блоке кода.  
  
 **Идентификатор ошибки** : BC30680  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Вставьте `#Region` в нужное место до соответствующего оператора `#End``Region` .
