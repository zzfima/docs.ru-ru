# <a name="method-39ltmethodnamegt39-is-already-declared-in-interface-39ltinterfacenamegt39"></a>Метод &#39; &lt;имя_метода&gt;&#39; уже объявлен в интерфейсе &#39;&lt; Имя интерфейса&gt;&#39;
Вы объявили метод интерфейса второй раз, дублируя его.  
  
 **Идентификатор ошибки:** BC30377  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите повторяющееся объявление.
