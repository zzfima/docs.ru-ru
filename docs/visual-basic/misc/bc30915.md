# &quot;&lt;имя&gt;&quot; не может представлять базовый тип делегата &quot;&lt;тип_делегата&gt;&quot; события, которое он реализует вне проекта посредством &lt;спецификатор&gt; &quot;&lt;тип&gt;&quot;
Код представляет закрытый тип за пределами открытого класса.  
  
 **Идентификатор ошибки:** BC30915  
  
### Исправление ошибки  
  
-   Объявите тип `Public`.  
  
     \-или\-  
  
-   Используйте другой тип.
