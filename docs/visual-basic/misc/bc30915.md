# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type"></a>"\<имя >" не может представлять базовый тип делегата "\<тип_делегата >" события, он реализует вне проекта посредством \<описатель > "\<тип >"
Код представляет закрытый тип за пределами открытого класса.  
  
 **Идентификатор ошибки:** BC30915  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Объявите тип `Public`.  
  
     -или-  
  
- Используйте другой тип.
