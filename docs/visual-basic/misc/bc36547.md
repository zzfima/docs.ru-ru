# <a name="anonymous-type-member-or-property-39ltpropertynamegt39-is-already-declared"></a><span data-ttu-id="008a8-102">Анонимный член типа или свойство &#39; &lt;propertyname&gt;&#39; уже объявлен</span><span class="sxs-lookup"><span data-stu-id="008a8-102">Anonymous type member or property &#39;&lt;propertyname&gt;&#39; is already declared</span></span>
<span data-ttu-id="008a8-103">Имя свойства может быть установлено только один раз в объявлении анонимного типа.</span><span class="sxs-lookup"><span data-stu-id="008a8-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="008a8-104">Например, следующие объявления неправильны:</span><span class="sxs-lookup"><span data-stu-id="008a8-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```
