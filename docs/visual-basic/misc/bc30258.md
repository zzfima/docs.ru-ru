# Классы могут наследовать только от других классов
В операторе `Inherits` в качестве базового типа указан тип, отличный от класса.  
  
 **Идентификатор ошибки:** BC30258  
  
### Исправление ошибки  
  
-   Измените базовый тип на класс, который уже определен.
