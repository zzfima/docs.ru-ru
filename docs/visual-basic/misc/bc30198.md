# Требуется &quot;)&quot;
Список, например список аргументов в определении или вызове процедуры, не завершен знаком закрывающей скобки \(`)`\).  
  
 **Идентификатор ошибки:** BC30198  
  
### Исправление ошибки  
  
-   Добавьте знак закрывающей скобки в конец списка.
