-   <span data-ttu-id="bd5f8-115">По возможности используйте аргумент вызова того же типа, что и параметр процедуры, поэтому Visual Basic не нужно выполнять никаких преобразований.</span><span class="sxs-lookup"><span data-stu-id="bd5f8-115">If possible, use a calling argument of the same type as the procedure parameter, so Visual Basic does not need to do any conversion.</span></span>

-   <span data-ttu-id="bd5f8-117">Если требуется возвращать значение в аргумент вызова, определите оператор обратного преобразования, Visual Basic можно преобразовать обратно в тип аргумента вызова.</span><span class="sxs-lookup"><span data-stu-id="bd5f8-117">If you need to return a value into the calling argument, define the reverse conversion operator so Visual Basic can convert back to the calling argument type.</span></span>
