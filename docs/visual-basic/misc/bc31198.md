<span data-ttu-id="b1d77-103">Объявление XML-литерала используется в неоднозначном для компилятора месте [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] .</span><span class="sxs-lookup"><span data-stu-id="b1d77-103">An XML literal declaration is used in an expression in a location that is ambiguous to the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler.</span></span> <span data-ttu-id="b1d77-104">То есть [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] компилятор не может определить ли меньше-не символ (<) предназначен как оператор сравнения или запуска XML-литерала.</span><span class="sxs-lookup"><span data-stu-id="b1d77-104">That is, the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="b1d77-105">Примером является следующий код:</span><span class="sxs-lookup"><span data-stu-id="b1d77-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="b1d77-106">**Идентификатор ошибки:** BC31198</span><span class="sxs-lookup"><span data-stu-id="b1d77-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b1d77-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="b1d77-107">To correct this error</span></span>

                  Select element
