<span data-ttu-id="ab64f-103">Вызов процедуры предоставляет аргумент по имени после пропуска того же аргумента по позиции, например:</span><span class="sxs-lookup"><span data-stu-id="ab64f-103">A procedure call supplies an argument by name after omitting the same argument by position; for example:</span></span>

```vb  
                                Optional ByVal Z As Byte = 0)

```
