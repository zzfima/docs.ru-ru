# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39;MyBase&#39; не может использоваться с методом &#39; &lt;имя_метода&gt; &#39; , так как он объявлен как &#39;MustOverride&#39;
Вы попытались использовать `MyBase` с методом, который был объявлен `MustOverride`.  
  
 **Идентификатор ошибки:** BC30399  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите объявление `MustOverride` .
