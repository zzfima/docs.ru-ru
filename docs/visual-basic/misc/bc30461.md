# <a name="classes-cannot-be-declared-39ltspecifiergt39"></a>Классы не могут объявляться &#39; &lt;описатель&gt;&#39;
Предпринята попытка объявить `Class` или `Module` с недопустимым спецификатором.  
  
 **Идентификатор ошибки:** BC30461  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите спецификатор.
