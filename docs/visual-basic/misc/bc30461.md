# <a name="classes-cannot-be-declared-ltspecifiergt"></a>Классы не могут объявляться как "&lt;описатель&gt;"
Предпринята попытка объявить `Class` или `Module` с недопустимым спецификатором.  
  
 **Идентификатор ошибки:** BC30461  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите спецификатор.
